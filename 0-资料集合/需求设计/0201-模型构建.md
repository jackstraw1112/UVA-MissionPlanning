# 反辐射无人机仿真系统——模型构建模块开发实现方案

## 1. 总体架构

模型构建模块采用**数据驱动建模**架构，将试验数据通过数学建模方法转化为可执行的仿真模型。模块支持线性/非线性回归、多项式拟合、神经网络、系统辨识等多种算法，并与模型库管理系统集成，实现模型的版本化存储和复用。

└─────────────────┘      │  ┌──────────┐ ┌──────────┐        │      └─────────────────┘
                          │  │数据分割器│ │算法工厂 │        │              ↓
                          │  └──────────┘ └──────────┘        │        仿真引擎调用
                          │  ┌─────────────────────────────┐  │

                          │  └──────────┘ └──────────┘        │
                          └─────────────────────────────────────┘

### 1.1 核心类设计
- **ModelBuilder**：总控类，管理建模流程，协调数据分割、算法选择、训练验证、模型导出。
- **DataSplitter**：负责训练集/验证集划分，支持随机划分、时间序列划分（按时间顺序）、K折交叉验证。
- **AlgorithmFactory**：根据用户选择的算法类型创建相应的建模器实例。
- **BaseModeler**：抽象基类，定义训练、预测、保存接口，各具体算法继承实现。
- **ModelValidator**：计算验证指标（R²、RMSE、MAE等），生成验证报告。
- **ModelExporter**：将模型参数序列化为标准格式（JSON/ONNX/PMML），并保存元数据。

## 3. 功能实现细节

### 3.1 数据准备与分割

#### 3.1.1 数据矩阵构建
- 从数据管理模块读取指定数据集，将输入变量组合成设计矩阵 **X**（行样本 × 列特征），输出变量为向量 **y**。
- 支持自动添加偏置项（截距）。
- 数据类型检查：确保输入变量为数值型，分类变量需先进行独热编码。

```cpp
Eigen::MatrixXd buildDesignMatrix(const QVector<QVector<double>>& data,
                                  const QVector<int>& inputCols,
                                  bool includeIntercept = true) {
    int nSamples = data.size();
    int nFeatures = inputCols.size() + (includeIntercept ? 1 : 0);
    Eigen::MatrixXd X(nSamples, nFeatures);
    
    for (int i = 0; i < nSamples; ++i) {
        int col = 0;
        if (includeIntercept) {
            X(i, col++) = 1.0;  // 偏置项
        }
        for (int j = 0; j < inputCols.size(); ++j) {
            X(i, col++) = data[i][inputCols[j]];
        }
    }
    return X;
}
```

#### 3.1.2 数据集划分
- **随机划分**：打乱样本顺序，按比例分割（默认70%训练，30%验证）。
- **时间序列划分**：保持时间顺序，前N%为训练，后(100-N)%为验证。
- **K折交叉验证**：将数据分成K份，轮流用K-1份训练、1份验证，取平均性能。

```cpp
class DataSplitter {
public:
    void randomSplit(const Eigen::MatrixXd& X, const Eigen::VectorXd& y,
                     double trainRatio, 
                     Eigen::MatrixXd& X_train, Eigen::VectorXd& y_train,
                     Eigen::MatrixXd& X_val, Eigen::VectorXd& y_val) {
        int n = X.rows();
        int nTrain = static_cast<int>(n * trainRatio);
        
        // 生成随机索引
        std::vector<int> indices(n);
        std::iota(indices.begin(), indices.end(), 0);
        std::shuffle(indices.begin(), indices.end(), std::mt19937(std::random_device()()));
        
        // 分割
        for (int i = 0; i < nTrain; ++i) {
            X_train.row(i) = X.row(indices[i]);
            y_train(i) = y(indices[i]);
        }
        for (int i = nTrain; i < n; ++i) {
            X_val.row(i - nTrain) = X.row(indices[i]);
            y_val(i - nTrain) = y(indices[i]);
        }
    }
};
```

### 3.2 算法实现

#### 3.2.1 线性回归（最小二乘法）
- **原理**：求解正规方程 **β = (XᵀX)⁻¹Xᵀy**，或使用QR分解提高数值稳定性。
- **实现**：Eigen的 `householderQr().solve(y)`

```cpp
class LinearRegression : public BaseModeler {
public:
    bool train(const Eigen::MatrixXd& X, const Eigen::VectorXd& y) override {
        // 使用QR分解求解最小二乘，数值稳定性更好
        m_coefficients = X.householderQr().solve(y);
        return true;
    }
    
    Eigen::VectorXd predict(const Eigen::MatrixXd& X) override {
        return X * m_coefficients;
    }
    
    void save(QJsonObject& json) override {
        json["type"] = "linear";
        json["coeff"] = vectorToJsonArray(m_coefficients);
    }
    
private:
    Eigen::VectorXd m_coefficients;
};
```

#### 3.2.2 多项式拟合
- **原理**：将输入变量扩展为多项式基函数（如 `x, x², x³, ...`），然后应用线性回归。
- **实现**：构建范德蒙矩阵，调用线性回归。

```cpp
class PolynomialFitting : public BaseModeler {
public:
    void setDegree(int degree) { m_degree = degree; }
    
    bool train(const Eigen::MatrixXd& X, const Eigen::VectorXd& y) override {
        // 假设X是单变量（n×1），扩展为多项式特征
        int n = X.rows();
        Eigen::MatrixXd polyX(n, m_degree + 1);
        for (int i = 0; i < n; ++i) {
            double x = X(i, 0);
            for (int d = 0; d <= m_degree; ++d) {
                polyX(i, d) = std::pow(x, d);
            }
        }
        // 调用线性回归
        m_coeff = polyX.householderQr().solve(y);
        return true;
    }
    
private:
    int m_degree = 2;
    Eigen::VectorXd m_coeff;
};
```

### 3.3 模型验证与评估

#### 3.3.1 评价指标
- **R²（决定系数）**：`1 - SS_res / SS_tot`
- **RMSE（均方根误差）**：`sqrt(mean((y_pred - y_true)²))`
- **MAE（平均绝对误差）**：`mean(|y_pred - y_true|)`

```cpp
struct ValidationMetrics {
    double R2;
    double RMSE;
    double MAE;
    
    static ValidationMetrics compute(const Eigen::VectorXd& y_true,
                                      const Eigen::VectorXd& y_pred) {
        int n = y_true.size();
        double ss_res = (y_true - y_pred).squaredNorm();
        double mean_y = y_true.mean();
        double ss_tot = (y_true.array() - mean_y).square().sum();
        
        ValidationMetrics metrics;
        metrics.R2 = 1.0 - ss_res / ss_tot;
        metrics.RMSE = std::sqrt(ss_res / n);
        metrics.MAE = (y_true - y_pred).cwiseAbs().mean();
        return metrics;
    }
};
```

#### 3.3.2 交叉验证
- 实现K折交叉验证，返回平均指标和标准差。

```cpp
class CrossValidator {
public:
    ValidationMetrics validate(BaseModeler* model, const Eigen::MatrixXd& X,
                               const Eigen::VectorXd& y, int k = 5) {
        std::vector<double> r2_scores;
        // 划分K折，轮流训练验证
        for (int fold = 0; fold < k; ++fold) {
            // 获取训练集和验证集索引
            auto [train_idx, val_idx] = getFoldIndices(n, k, fold);
            
            // 构建训练数据和验证数据
            Eigen::MatrixXd X_train = extractRows(X, train_idx);
            Eigen::VectorXd y_train = extractRows(y, train_idx);
            Eigen::MatrixXd X_val = extractRows(X, val_idx);
            Eigen::VectorXd y_val = extractRows(y, val_idx);
            
            // 训练
            model->train(X_train, y_train);
            
            // 预测
            Eigen::VectorXd y_pred = model->predict(X_val);
            
            // 计算R²
            double r2 = computeR2(y_val, y_pred);
            r2_scores.push_back(r2);
        }
        
        // 计算均值和标准差
        return computeStats(r2_scores);
    }
};
```

## 7. 与反辐射无人机建模的特定应用

根据反辐射无人机建模需求，模型构建模块需要支持以下特定模型类型：

### 7.1 运动学模型
- **输入**：时间、控制指令（油门、舵面）
- **输出**：位置、速度、姿态角
- **方法**：基于飞行试验数据，使用系统辨识方法（ARX或状态空间）建模

### 7.2 导引头模型
- **输入**：目标相对角度、信号强度
- **输出**：导引头输出（误差信号）、探测概率
- **方法**：基于半实物仿真数据，使用神经网络或支持向量机回归

### 7.3 目标特性模型
- **输入**：目标类型、距离、方位
- **输出**：雷达散射截面（RCS）、红外辐射强度
- **方法**：基于外场测试数据，使用多项式拟合或高斯过程回归
