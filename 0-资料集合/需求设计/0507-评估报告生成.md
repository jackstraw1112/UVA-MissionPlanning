## 4. 分析流程

### 4.1 报告结构设计
报告应包含以下部分：
- **封面**：任务名称、生成时间、报告编号等。
- **摘要**：简要概述评估结果。
- **任务背景**：任务描述、评估目的。
- **指标体系**：以树形图或表格展示指标层次结构和权重。
- **指标得分**：列出每个底层指标的得分，可附加与基准值的对比。
- **综合得分**：显示顶层指标的综合得分，并给出评价等级（如优秀/良好/一般/较差）。
- **图表展示**：嵌入雷达图、柱状图等可视化图表。
- **结论与建议**：基于评估结果的分析结论和改进建议。

### 4.2 图表嵌入
- 对于PDF生成，可使用 `QPdfWriter` 绘制矢量图形，或将 `QChart` 渲染为图片嵌入。建议优先使用矢量方式。
- 对于Word生成，可通过 `QAxObject` 调用Word COM接口插入图表，或生成HTML后用Word打开。

### 4.3 模板机制
- 模板文件可以是带有特定占位符的HTML文件（如 `{{task_name}}`），报告生成引擎读取后替换占位符。
- 对于Word，模板可以是带有书签的docx文件，利用 `docx` 库操作。

### 4.4 性能考虑
- 生成大型报告时，应异步执行，避免阻塞UI。
- 图表生成若耗时，可预先缓存。

# 反辐射无人机仿真系统——评估报告生成模块开发实现方案

## 1. 总体架构

评估报告生成模块采用**模板+数据填充**的架构，与评估计算模块、结果展示模块解耦。支持多种输出格式（PDF、Word、HTML），并提供可扩展的模板机制。核心组件包括：

- **报告数据收集器**：从评估计算模块获取结构化结果。
- **模板引擎**：解析用户自定义或系统默认模板，替换占位符。
- **图表渲染器**：将Qt Charts图表转换为矢量或位图嵌入报告。
- **格式转换器**：根据用户选择调用相应生成器（PDF/Word/HTML）。
- **进度管理器**：异步生成时反馈进度，避免界面阻塞。

```plaintext

│ 评估计算模块    │ ──→ │        报告生成引擎                   │ ──→ │   PDF/Word/HTML │
└─────────────────┘      │  ┌──────────┐ ┌──────────┐        │      └─────────────────┘
                          │  │数据收集器│ │模板引擎  │        │
                          │  └──────────┘ └──────────┘        │
                          │  ┌──────────┐ ┌──────────┐        │
                          │  │图表渲染器│ │格式转换器│        │

                          └─────────────────────────────────────┘
```

## 3. 报告结构设计

报告统一包含以下章节，由模板定义具体布局：

1. **封面**：任务名称、生成时间、报告编号。
2. **摘要**：综合得分及简要结论。
3. **任务信息**：任务ID、仿真时间、参与兵力等。
4. **指标体系**：树形结构或表格展示指标及其权重。
5. **指标得分**：各指标得分表，可包含与基准值的对比。
6. **图表展示**：雷达图、柱状图等（根据模板位置嵌入）。
7. **结论与建议**：基于评估结果的文字结论。
8. **附录**：原始数据（可选）。

## 4. 模板机制实现

### 4.1 模板格式
- **HTML模板**：使用`{{placeholder}}`占位符，可包含CSS样式。
- **Word模板**：使用带书签的docx文件，通过`QAxObject`操作书签插入内容。
- **PDF模板**：不支持直接模板，可通过编程方式绘制，或先生成HTML再转PDF。

### 4.2 模板加载与填充
```cpp
class ReportTemplate {
public:
    bool load(const QString& templatePath);
    QString fill(const QVariantMap& data); // 返回填充后的HTML字符串

private:
    QString m_content;
};

// 使用正则表达式替换占位符
QString ReportTemplate::fill(const QVariantMap& data) {
    QString result = m_content;
    QRegularExpression re("\\{\\{(\\w+)\\}\\}");
    QRegularExpressionMatchIterator i = re.globalMatch(result);
    while (i.hasNext()) {
        QRegularExpressionMatch match = i.next();
        QString placeholder = match.captured(1);
        if (data.contains(placeholder)) {
            result.replace(match.captured(0), data[placeholder].toString());
        }
    }
    return result;
}
```

## 5. 图表渲染与嵌入

### 5.1 图表生成
使用评估结果展示模块已有的`QChart`对象，将其渲染为图像：

```cpp
QImage ChartRenderer::renderToImage(QChart* chart, const QSize& size) {
    chart->resize(size);
    QPixmap pixmap(size);
    QPainter painter(&pixmap);
    painter.setRenderHint(QPainter::Antialiasing);
    chart->render(&painter);
    return pixmap.toImage();
}
```

对于PDF，可直接使用`QPainter`绘制矢量图表，避免像素化。

### 5.2 嵌入方式
- **HTML**：将图表保存为Base64编码的图片，嵌入`<img>`标签。
- **Word**：通过`QAxObject`调用`InlineShapes.AddPicture()`插入图片。
- **PDF**：使用`QPdfWriter`的`drawImage()`绘制图片，或直接用`QPainter`绘制矢量。

## 6. 多格式导出实现

### 6.1 PDF生成（QPdfWriter）
```cpp
bool generatePDF(const QString& fileName, const QVariantMap& data, QChart* chart) {
    QPdfWriter writer(fileName);
    writer.setPageSize(QPageSize(QPageSize::A4));
    QPainter painter(&writer);
    
    // 绘制封面、标题等
    QFont titleFont("Arial", 16);
    painter.setFont(titleFont);
    painter.drawText(100, 100, data["task_name"].toString());
    
    // 绘制表格
    drawTable(painter, data);
    
    // 绘制图表（矢量）
    chart->paint(&painter, QRect(100, 300, 400, 300));
    
    painter.end();
    return true;
}
```

### 6.2 Word生成（QAxObject，Windows平台）
```cpp
bool generateWord(const QString& fileName, const QVariantMap& data, QChart* chart) {
    QAxObject* word = new QAxObject("Word.Application");
    word->setProperty("Visible", false);
    QAxObject* docs = word->querySubObject("Documents");
    QAxObject* doc = docs->querySubObject("Add()");
    
    // 获取Selection对象
    QAxObject* selection = word->querySubObject("Selection");
    
    // 插入文字
    selection->dynamicCall("TypeText(const QString&)", "任务名称：" + data["task_name"].toString());
    selection->dynamicCall("TypeParagraph");
    
    // 插入图片（需先保存临时文件）
    QString tempImage = QDir::tempPath() + "/chart_temp.png";
    chart->grab().save(tempImage);
    selection->querySubObject("InlineShapes")->dynamicCall("AddPicture(const QString&)", tempImage);
    
    // 保存文档
    doc->dynamicCall("SaveAs(const QString&)", fileName);
    doc->dynamicCall("Close()");
    word->dynamicCall("Quit()");
    delete word;
    return true;
}
```

### 6.3 HTML生成
```cpp
bool generateHTML(const QString& fileName, const QVariantMap& data, QChart* chart) {
    QFile file(fileName);
    if (!file.open(QIODevice::WriteOnly | QIODevice::Text))
        return false;
    
    QTextStream out(&file);
    out << "<html><head><title>评估报告</title></head><body>";
    out << "<h1>" << data["task_name"].toString() << "</h1>";
    // 插入表格
    out << "<table border='1'>";
    // ... 指标数据
    out << "</table>";
    
    // 插入图表（Base64）
    QByteArray ba;
    QBuffer buffer(&ba);
    buffer.open(QIODevice::WriteOnly);
    chart->grab().save(&buffer, "PNG");
    out << "<img src='data:image/png;base64," << ba.toBase64() << "'/>";
    
    out << "</body></html>";
    file.close();
    return true;
}
```
