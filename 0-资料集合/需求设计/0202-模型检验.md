## 2. 关键术语解释

| **术语** | **解释** |
|----------|----------|
| **模型系数** | 模型中各个输入变量的权重或参数，表示变量对输出的影响程度。对线性模型，系数即为回归系数。 |
| **评价指标** | 衡量模型预测性能的统计量，如R²（决定系数）、RMSE（均方根误差）、MAE（平均绝对误差）等。 |
| **残差** | 实际观测值与模型预测值之差，即 e = y - ŷ。残差反映了模型未能解释的部分。 |
| **残差分析** | 通过对残差的分布、相关性、异方差性等进行检验，评估模型假设（如独立性、正态性、同方差性）是否满足。 |
| **残差图** | 以残差为纵坐标，以拟合值或某个变量为横坐标绘制的散点图，用于直观检验模型的适用性。 |
| **正态性检验** | 检验残差是否符合正态分布，常用方法有Shapiro-Wilk检验、Q-Q图等。 |
| **自相关检验** | 检验残差是否存在序列相关性，常用Durbin-Watson检验。 |
| **异方差检验** | 检验残差方差是否随拟合值变化，常用Breusch-Pagan检验。 |

## 4. 分析流程

### 4.1 系数提取与显著性检验（线性模型）
- 对于线性回归模型，系数估计值 β̂ 及其协方差矩阵可从模型文件中获取。
- 计算标准误 SE(β̂) = sqrt(diag(Cov(β̂)))。
- 计算 t 统计量 t = β̂ / SE(β̂)，对应 p 值 = 2 * P(T > |t|)。
- 输出系数表，包含变量名、系数、标准误、t值、p值，并标记显著性（如 * p<0.05, ** p<0.01）。

### 4.2 评价指标计算
- **R²** = 1 - SS_res / SS_tot
- **调整R²** = 1 - (1-R²)*(n-1)/(n-p-1)
- **RMSE** = sqrt(mean((y - ŷ)²))
- **MAE** = mean(|y - ŷ|)
- **MAPE** = mean(|(y - ŷ)/y|) * 100%
- **AIC** = n * ln(SS_res/n) + 2p（适用于线性模型）

### 4.3 残差分析

#### 4.3.1 基本统计量
- 残差均值 ≈ 0（理想情况）
- 残差方差 σ²_e
- 残差偏度、峰度

#### 4.3.2 正态性检验
- **Shapiro-Wilk检验**：适用于小样本（<5000），计算W统计量及p值。
- **Kolmogorov-Smirnov检验**：比较残差分布与正态分布。
- **Q-Q图**：将残差分位数与正态分布理论分位数对比，点应大致在直线上。

#### 4.3.3 自相关检验（适用于时序数据）
- **Durbin-Watson检验**：DW ≈ 2(1-ρ)，值接近2表示无自相关。

#### 4.3.4 异方差检验
- **Breusch-Pagan检验**：将残差平方对自变量回归，检验回归系数是否显著。
- **残差vs拟合值图**：若点随机分布无趋势，表示同方差。

### 4.4 残差图绘制
- **残差 vs 拟合值**：横坐标为ŷ，纵坐标为e，用于检验异方差和非线性。
- **残差 vs 每个输入变量**：横坐标为x_i，纵坐标为e，检查变量是否应加入非线性项。
- **Q-Q图**：检验正态性。
- **残差时间序列图**：若有时间顺序，检查自相关性。

#### 3.2.1 系数提取

#### 3.2.2 标准误与t检验
```cpp
struct CoefficientStats {
    double estimate;
    double stdError;
    double tStat;
    double pValue;
    bool significant;  // p < 0.05
};

QVector<CoefficientStats> computeCoefficientStats(
    const Eigen::VectorXd& beta,
    const Eigen::MatrixXd& X,
    double sigma2) {
    
    Eigen::MatrixXd XtX_inv = (X.transpose() * X).inverse();
    Eigen::VectorXd se = (sigma2 * XtX_inv.diagonal()).array().sqrt();
    
    QVector<CoefficientStats> stats;
    for (int i = 0; i < beta.size(); ++i) {
        CoefficientStats s;
        s.estimate = beta(i);
        s.stdError = se(i);
        s.tStat = beta(i) / se(i);
        // 计算p值（双尾t检验）
        s.pValue = 2.0 * (1.0 - boost::math::cdf(boost::math::students_t(n - p), std::abs(s.tStat)));
        s.significant = s.pValue < 0.05;
        stats.append(s);
    }
    return stats;
}
```

### 3.3 评价指标计算

#### 3.3.1 基础指标
```cpp
struct Metrics {
    double R2;
    double adjR2;
    double RMSE;
    double MAE;
    double MAPE;  // 平均绝对百分比误差
    double AIC;   // 赤池信息准则
    double BIC;   // 贝叶斯信息准则
};

Metrics computeMetrics(const Eigen::VectorXd& y, const Eigen::VectorXd& y_pred, int p) {
    int n = y.size();
    double ss_res = (y - y_pred).squaredNorm();
    double ss_tot = (y.array() - y.mean()).square().sum();
    
    Metrics m;
    m.R2 = 1.0 - ss_res / ss_tot;
    m.adjR2 = 1.0 - (1.0 - m.R2) * (n - 1) / (n - p - 1);
    m.RMSE = std::sqrt(ss_res / n);
    m.MAE = (y - y_pred).cwiseAbs().mean();
    m.MAPE = ((y - y_pred).cwiseAbs().array() / y.array()).mean() * 100.0;
    m.AIC = n * std::log(ss_res / n) + 2 * p;
    m.BIC = n * std::log(ss_res / n) + p * std::log(n);
    
    return m;
}
```

struct ResidualStats {
    double mean;
    double variance;
    double skewness;
    double kurtosis;
    double min;
    double max;
};

ResidualStats computeResidualStats(const Eigen::VectorXd& residuals) {
    ResidualStats stats;
    stats.mean = residuals.mean();
    stats.variance = (residuals.array() - stats.mean).square().sum() / (residuals.size() - 1);

#### 3.4.3 自相关检验（Durbin-Watson）
```cpp
double durbinWatson(const Eigen::VectorXd& residuals) {
    int n = residuals.size();
    double numerator = 0.0;
    double denominator = residuals.squaredNorm();
    
    for (int i = 1; i < n; ++i) {
        numerator += std::pow(residuals(i) - residuals(i-1), 2);
    }
    
    double dw = numerator / denominator;
    return dw;  // 接近2表示无自相关
}
```

#### 3.4.4 异方差检验（Breusch-Pagan）
```cpp
struct HeteroscedasticityTest {
    double statistic;
    double pValue;
    bool hasHeteroscedasticity;  // p < 0.05
};

HeteroscedasticityTest breuschPagan(
    const Eigen::VectorXd& residuals,
    const Eigen::MatrixXd& X) {
    
    int n = X.rows();
    int k = X.cols();
    
    // 残差平方
    Eigen::VectorXd e2 = residuals.array().square();
    
    // 辅助回归：e2 ~ X
    Eigen::VectorXd gamma = (X.transpose() * X).inverse() * X.transpose() * e2;
    Eigen::VectorXd e2_pred = X * gamma;
    
    // 计算LM统计量
    double ssr_aux = (e2 - e2_pred).squaredNorm();
    double ssr_total = (e2.array() - e2.mean()).square().sum();
    double R2_aux = 1.0 - ssr_aux / ssr_total;
    double LM = n * R2_aux;
    
    // 卡方检验，自由度 = k-1
    double pValue = 1.0 - boost::math::cdf(boost::math::chi_squared(k-1), LM);
    
    return {LM, pValue, pValue < 0.05};
}
```

QChart* createQQPlot(const Eigen::VectorXd& residuals) {
    int n = residuals.size();
    
    // 计算理论分位数
    std::vector<double> sorted_residuals(residuals.data(), residuals.data() + n);
    std::sort(sorted_residuals.begin(), sorted_residuals.end());
    
    QScatterSeries* series = new QScatterSeries();
    series->setName("残差分位数");
    
    for (int i = 0; i < n; ++i) {
        double p = (i + 0.5) / n;  // 经验累积概率
        double theoretical = boost::math::quantile(boost::math::normal(), p);
        series->append(theoretical, sorted_residuals[i]);
    }
    
    // 添加45度参考线
    QLineSeries* refLine = new QLineSeries();
    double minVal = std::min(theoretical_min, sorted_min);
    double maxVal = std::max(theoretical_max, sorted_max);
    refLine->append(minVal, minVal);
    refLine->append(maxVal, maxVal);
    refLine->setPen(QPen(Qt::red, 1, Qt::DashLine));
    
    QChart* chart = new QChart();
    chart->addSeries(series);
    chart->addSeries(refLine);
    chart->createDefaultAxes();
    chart->setTitle("Q-Q图");
    return chart;
}
