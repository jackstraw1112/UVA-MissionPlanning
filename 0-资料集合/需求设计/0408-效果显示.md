# 反辐射无人机仿真系统——仿真推演模块效果显示功能需求描述

## 1. 功能需求描述

| **项目**         | **描述** |
|------------------|----------|
| **需求名称**     | 效果显示 |
| **需求标识**     | FR-SIM-08 |
| **需求概述**     | 能够在三维场景中动态显示实体设备的作战行动特性及行动效果，包括作战实体运行轨迹、雷达扫描波束、导引头搜索波束、火力毁伤效果、威力范围等，以增强对作战过程的理解和分析能力。 |
| **需求来源**     | 系统总体设计要求 / 仿真推演模块能力指标 |
| **前置条件**     | 1. 三维场景已初始化，地形、实体模型加载完成。<br>2. 仿真引擎正在运行或已完成数据记录，可获取相关行动数据。<br>3. 轨迹、波束、毁伤等数据已由仿真引擎生成并提供。<br>4. 相关的视觉特效资源（如粒子系统、光束模型）已加载。 |
| **触发条件**     | 1. 仿真运行过程中实时显示。<br>2. 用户通过控制面板选择开启/关闭特定效果显示。<br>3. 用户回放仿真数据时按时间进度显示。 |
| **退出条件**     | 1. 仿真结束或用户关闭效果显示功能。<br>2. 相关行动结束（如导弹命中后轨迹消失）。 |
| **功能参与者**   | 仿真引擎（提供行动数据）、三维渲染引擎（显示效果）、用户（观察、控制） |
| **输入**         | 实体轨迹数据（时间序列位置点）、雷达/导引头状态（开/关、扫描参数、方向）、武器发射事件（发射时刻、类型、目标）、毁伤事件（命中点、毁伤半径）、威力范围参数（类型、位置、半径/角度） |
| **输出**         | 三维场景中的动态视觉元素：轨迹线、扫描波束（动态扫描动画）、导引头视场锥、爆炸特效、火焰/烟雾、威力范围半透明体（球体、圆锥等） |
| **处理流程**     | 1. **数据接收**：从仿真引擎或回放数据源接收行动和效果数据。<br>2. **轨迹显示**：对每个实体，根据历史位置生成轨迹线（如折线或曲线），可设置颜色、粗细、持续时间。<br>3. **波束显示**：根据雷达/导引头状态，生成动态波束几何体（如扇形、圆锥），并使其随时间旋转或扫描，体现工作模式。<br>4. **毁伤效果**：当收到武器命中事件时，在命中点触发爆炸粒子效果，随后根据目标类型添加火焰/烟雾，并可能改变目标模型外观（如变黑、碎片）。<br>5. **威力范围**：根据武器或传感器的参数，生成半透明体积（球体表示杀伤半径、圆锥表示探测范围），并随实体移动而更新位置。<br>6. **实时更新**：所有效果按仿真时间推进或回放时间同步更新。 |
| **成功判据**     | 1. 轨迹线正确反映实体运动路径，无跳变或错位。<br>2. 波束显示方向、扫描动态与实际数据一致，易于辨识。<br>3. 毁伤效果在正确位置触发，视觉表现合理（爆炸、烟雾）。<br>4. 威力范围与实际参数相符，半透明渲染不影响场景辨识。 |
| **异常处理**     | 1. 数据缺失或格式错误：忽略该效果并记录日志。<br>2. 性能下降（帧率过低）：可自动降低特效质量或关闭部分效果，并提示用户。<br>3. 波束参数异常（如方向向量为零）：不显示或显示默认方向。 |
| **后置条件**     | 1. 效果显示持续至仿真结束或用户关闭。<br>2. 效果数据可保存用于事后回放。 |
| **显示约束**     | 1. 轨迹线应支持多种样式（实线、虚线、颜色渐变）以区分不同类型实体。<br>2. 波束显示应半透明，避免完全遮挡背景，可设置不同的颜色（如雷达波为红色，导引头为橙色）。<br>3. 毁伤效果应有层次感，爆炸后残留火焰/烟雾随时间消散。<br>4. 威力范围应使用半透明材质，允许用户开关显示。<br>5. 所有效果应支持LOD（细节层次），远距离简化或隐藏。 |
| **其他约束**     | 1. 效果显示不应过度消耗性能，需经过优化（如粒子数量限制、轨迹点简化）。<br>2. 支持用户自定义效果显示选项（如开关特定效果、调整透明度）。<br>3. 效果数据应与仿真时间严格同步。 |

## 2. 关键术语解释

| **术语** | **解释** |
|----------|----------|
| **作战实体运行轨迹** | 作战实体（如反辐射无人机、导弹、作战飞机）在三维空间中的运动路径，通常以连续线条表示，反映其位置随时间的变化。 |
| **雷达扫描波束** | 雷达天线发射的电磁波束形状，在仿真中常以动态扫描的扇形或圆锥体表示，指示雷达的探测范围和当前指向。 |
| **导引头搜索波束** | 反辐射无人机导引头（被动或主动）的视场范围，通常表现为一个圆锥体，表示导引头当前搜索的空域。 |
| **火力毁伤效果** | 武器命中目标后产生的爆炸、火焰、烟雾等视觉特效，以及目标受损后的外观变化（如变黑、破裂）。 |
| **威力范围** | 武器杀伤半径、雷达探测距离等作用范围的几何表示，通常以半透明的球体、圆盘或圆锥体在三维场景中展示。 |

## 4. 分析流程

### 4.1 效果显示的数据驱动模型
- **轨迹显示**：每帧根据实体历史位置更新轨迹线。可采用线带（ribbon）或点线，支持设置最大长度（如只显示最近N秒轨迹）。颜色可随速度变化（如速度高为红色）。
- **波束显示**：波束几何体与实体绑定，根据实体朝向动态更新旋转。雷达扫描波束可添加旋转动画；导引头视场锥可设置固定方向。
- **毁伤效果**：收到命中事件后，在命中点实例化爆炸粒子系统，并根据目标类型添加持续火焰/烟雾。同时，改变目标模型的材质（如增加烧灼纹理）或替换为受损模型。
- **威力范围**：静态范围（如雷达探测距离）可随实体移动更新位置；动态范围（如导弹杀伤半径）可在发射后显示，随时间变化（如助推段后变大）。

### 4.2 性能优化
- **轨迹简化**：对长轨迹进行抽稀，保留关键点，减少顶点数。
- **波束LOD**：远距离波束简化或隐藏，或使用低面片模型。
- **粒子系统优化**：使用GPU粒子，限制同屏粒子数；火焰/烟雾采用公告牌（billboard）技术。
- **威力范围**：使用低分辨率网格，或仅显示轮廓。

### 4.3 用户交互
- 提供“效果显示设置”面板，允许用户开关各类效果、调整透明度、设置轨迹持续时间等。
- 支持点击效果对象查看详细信息（如点击轨迹可显示对应实体）。

|---------------------|----------|----------|
| FR-SIM-PERF-54      | 同时显示轨迹的实体数量 | ≥500个 |

| FR-SIM-PERF-58      | 轨迹数据更新延迟 | ≤50ms |
| FR-SIM-PERF-59      | 威力范围几何体面数 | ≤500面/个 |
| FR-SIM-PERF-60      | 效果显示开关响应时间 | ≤0.2秒 |
