# 反辐射无人机仿真系统——试验数据处理模块可视化展示功能需求描述

## 1. 功能需求描述

| **项目**         | **描述** |
|------------------|----------|
| **需求名称**     | 可视化展示 |
| **需求标识**     | FR-DATA-06 |
| **需求概述**     | 具备数据可视化展示功能，能够利用多种图表方式，如柱状图、折线图、散点图、饼图等，展示数据的分布、趋势和关系，帮助用户直观理解试验数据特征和规律。 |
| **需求来源**     | 系统总体设计要求 / 试验数据处理模块能力指标 |
| **前置条件**     | 1. 试验数据已完成采集、清洗、转换等预处理，或至少已导入系统。<br>2. 用户已选择待可视化的数据集和变量。<br>3. 系统图表渲染引擎（Qt Charts / ECharts等）已初始化。 |
| **触发条件**     | 1. 用户在“可视化展示”界面点击“生成图表”按钮。<br>2. 用户选择不同的图表类型或变量后自动刷新图表。<br>3. 用户在数据分析流程中点击“可视化”入口。 |
| **退出条件**     | 1. 图表生成完成并显示在界面中。<br>2. 用户关闭可视化窗口或切换到其他功能模块。<br>3. 发生不可恢复错误，模块退出并记录错误。 |
| **功能参与者**   | 用户（试验分析人员）、可视化渲染引擎、数据访问接口 |
| **输入**         | 数据集（表）、用户选择的变量（X轴、Y轴、分组字段、颜色映射等）、图表类型（柱状图、折线图、散点图、饼图等）、图表定制参数（标题、坐标轴标签、颜色主题、图例位置等）。 |
| **输出**         | 生成的图表（支持交互：悬停显示数值、缩放、平移、保存为图片）、图表对应的数据表格（可选）。 |
| **处理流程**     | 1. **数据准备**：从数据库中读取用户指定的数据集和变量，根据图表类型对数据进行聚合或采样（如柱状图需统计频数，散点图需保留原始点）。<br>2. **图表类型映射**：<br>   - 柱状图：用于展示分类变量的频数分布或比较不同组的数值。X轴为类别，Y轴为频数或汇总统计量（均值、总和）。<br>   - 折线图：用于展示随时间或有序类别变化的趋势。X轴为时间或有序变量，Y轴为数值。<br>   - 散点图：用于展示两个连续变量之间的关系。X轴为第一变量，Y轴为第二变量，可添加颜色映射第三变量。<br>   - 饼图：用于展示各部分的占比。输入为类别及其数值（如频数或总和）。<br>3. **图表渲染**：调用图表库生成交互式图表，配置坐标轴、图例、标题等。<br>4. **交互增强**：添加悬停提示、缩放、平移、数据点高亮等功能。<br>5. **结果导出**：支持图表导出为PNG/SVG，数据导出为CSV。 |
| **成功判据**     | 1. 图表正确展示所选变量的分布特征，无渲染错误。<br>2. 交互功能（悬停、缩放、平移）正常响应。<br>3. 导出文件内容正确，格式符合要求。 |
| **异常处理**     | 1. 所选变量类型与图表不匹配（如饼图选了两个连续变量）：提示用户建议修改图表类型或变量。<br>2. 数据量过大导致图表渲染缓慢：自动进行数据采样（如散点图采样），并提示用户。<br>3. 图表库初始化失败：降级为静态图片，并记录错误日志。<br>4. 内存不足：建议用户减少数据量或分批显示。 |
| **后置条件**     | 1. 生成的图表保存在内存或临时文件中，可随时重新生成。<br>2. 操作记录写入系统日志。 |
| **显示约束**     | 1. 图表区域应支持缩放、平移，适应不同分辨率。<br>2. 提供图表类型切换按钮，允许用户动态改变可视化方式。<br>3. 悬停在数据点上应显示具体数值。<br>4. 支持图例点击隐藏/显示相应系列。<br>5. 提供工具栏用于导出图片、复制数据等。 |
| **其他约束**     | 1. 支持的数据集大小至少为100万行，20个数值列。<br>2. 图表渲染时间应在3秒内（百万级数据点采样后）。<br>3. 应提供常用图表类型：柱状图（含堆叠、分组）、折线图、散点图（含气泡图）、饼图（含环形图）。<br>4. 支持多图表布局（如并列显示多个图表）。 |

| **术语** | **解释** |

## 4. 分析流程

### 4.1 图表类型与数据映射
- **柱状图**：对X轴变量（分类）进行分组，计算每组的Y值（计数、总和、均值等）。支持分组柱状图（按第二个分类变量分颜色）。
- **折线图**：X轴为有序变量（时间），Y轴为数值，可按分组变量绘制多条折线。
- **散点图**：直接使用每行的X、Y值绘制点，可按分组变量设置不同颜色。
- **饼图**：按分类变量计算频数或总和，绘制扇形。

### 4.2 数据采样策略
- 对于散点图，当数据点超过5000时，随机采样5000点，保持整体分布特征。
- 对于柱状图，直接聚合计数，无需采样。
- 采样算法应可配置（随机采样、系统采样、分层采样）。

### 4.3 交互功能
- 悬停显示数值：显示当前点的具体值。
- 框选缩放：用鼠标框选区域放大查看细节。
- 平移：拖拽移动视图。
- 图例点击：隐藏/显示对应系列。
- 数据刷选：在图表上选择数据点，可在关联表格中高亮。

# 反辐射无人机仿真系统——可视化展示模块开发实现方案

## 1. 总体架构

可视化展示模块采用**数据适配层 + 图表引擎**分离架构，支持多种图表类型和交互功能。后端基于C++/Qt处理数据聚合和采样，前端使用Qt Charts（桌面）或ECharts（Web）实现交互式渲染。

```plaintext
┌─────────────────┐      ┌─────────────────────────────────────┐      ┌─────────────────┐
│ 数据管理模块    │ ──→ │        可视化控制器                  │ ──→ │   图表引擎     │
└─────────────────┘      │  ┌──────────┐ ┌──────────┐        │      ├────────┬────────┤
                          │  │数据适配器│ │采样器   │        │      │Qt Charts│ECharts │
                          │  └──────────┘ └──────────┘        │      └────────┴────────┘
                          │  ┌─────────────────────────────┐  │              ↓
                          │  │       交互管理器            │  │           图表显示
                          │  └─────────────────────────────┘  │
                          └─────────────────────────────────────┘
```

### 1.1 核心类设计
- **VisualizationController**：总控类，协调数据加载、图表生成和交互。
- **DataAdapter**：根据不同图表类型对原始数据进行聚合、分组、采样。
- **ChartFactory**：根据用户选择创建不同类型的图表（柱状图、折线图等）。
- **InteractionManager**：处理悬停、缩放、平移、点击等交互。
- **ExportManager**：负责图表导出为图片、数据导出为CSV。

## 2. 核心技术选型

| 功能                | 技术方案（桌面）                | 技术方案（Web）                | 说明                               |
|---------------------|--------------------------------|-------------------------------|------------------------------------|
| 图表引擎            | Qt Charts                      | ECharts                       | 与Qt原生集成 / 交互丰富            |
| 数据聚合            | Qt SQL + 自定义聚合             | 后端计算后JSON传递             | 百万级数据先聚合再传输             |
| 数据采样            | 随机采样 / 系统采样             | 后端采样后传递                 | 避免前端性能瓶颈                   |
| 交互处理            | Qt信号槽 + 事件过滤器            | ECharts内置交互               |                                    |
| 导出功能            | QPixmap::save() / QPrinter      | ECharts 内置导出              | 支持PNG/SVG/PDF                    |
| 并行处理            | QtConcurrent                    | 后端多线程                     | 数据聚合和采样并行化               |

## 3. 功能实现细节

### 3.1 数据适配与预处理

#### 3.1.1 数据聚合
- **柱状图**：对X轴分类变量进行`GROUP BY`，计算Y轴的聚合值（COUNT, SUM, AVG等）。
- **折线图**：若X轴为时间，可能需要按时间窗口聚合（如每分钟均值）。
- **饼图**：按分类变量分组计算总和或频数。

```cpp
// 柱状图数据聚合示例（使用SQL）
QSqlQuery query;
query.prepare("SELECT category, COUNT(*) FROM data GROUP BY category");
query.exec();
while (query.next()) {
    QString category = query.value(0).toString();
    int count = query.value(1).toInt();
    m_categories.append(category);
    m_counts.append(count);
}
```

#### 3.1.2 数据采样
- **随机采样**：对大数据集（>5000点）随机抽取N个样本。
- **系统采样**：等间隔抽取。
- **重要性采样**：保留离群点和关键点（可选）。

```cpp
QVector<QPointF> sampleData(const QVector<QPointF>& original, int maxPoints) {
    if (original.size() <= maxPoints) return original;
    QVector<QPointF> sampled;
    sampled.reserve(maxPoints);
    std::random_device rd;
    std::mt19937 gen(rd());
    std::uniform_int_distribution<> dis(0, original.size() - 1);
    for (int i = 0; i < maxPoints; ++i) {
        sampled.append(original[dis(gen)]);
    }
    return sampled;
}
```

### 3.2 图表渲染实现

#### 3.2.1 Qt Charts 方案
- **柱状图**：使用 `QBarSeries` + `QBarSet`。
- **折线图**：使用 `QLineSeries`。
- **散点图**：使用 `QScatterSeries`，可设置点大小和颜色。
- **饼图**：使用 `QPieSeries`。

```cpp
// 散点图创建示例
QScatterSeries* series = new QScatterSeries();
series->setName("高度-速度关系");
series->setMarkerSize(8.0);
for (const auto& point : m_scatterData) {
    series->append(point.x(), point.y());
}
m_chart->addSeries(series);
m_chart->createDefaultAxes();
```

#### 3.2.2 ECharts 方案
- 通过 `QWebEngineView` 加载HTML模板，使用 `QWebChannel` 传递数据。
- ECharts 配置灵活，支持丰富的交互。

```javascript
// ECharts 柱状图配置示例
option = {
    xAxis: { type: 'category', data: categories },
    yAxis: { type: 'value' },
    series: [{
        type: 'bar',
        data: counts,
        itemStyle: { color: '#4a9eff' }
    }]
};
```

### 3.3 交互功能实现

#### 3.3.1 悬停提示
- Qt Charts：连接 `series` 的 `hovered` 信号，显示 `QToolTip`。
- ECharts：内置 `tooltip` 配置，自动显示。

#### 3.3.2 缩放平移
- Qt Charts：`QChartView` 支持鼠标拖拽平移和滚轮缩放，需设置 `rubberBand`。
- ECharts：`dataZoom` 组件或鼠标滚轮缩放。

#### 3.3.3 图例交互
- 点击图例隐藏/显示系列，Qt Charts 需手动处理，ECharts 内置支持。

### 3.4 导出功能实现

#### 3.4.1 图表导出
- Qt Charts：`chartView->grab().save("chart.png")` 或 `QPdfWriter`。
- ECharts：`myChart.getDataURL({ type: 'png' })` 导出图片。

#### 3.4.2 数据导出
- 将图表背后的数据（如柱状图的类别和频数）导出为CSV。
```cpp
QFile file("chart_data.csv");
if (file.open(QIODevice::WriteOnly)) {
    QTextStream stream(&file);
    stream << "category,count\n";
    for (int i = 0; i < categories.size(); ++i) {
        stream << categories[i] << "," << counts[i] << "\n";
    }
}
```

## 4. 性能优化措施

| **优化项**         | **实现方法**                                                | **效果**                     |
|--------------------|-------------------------------------------------------------|------------------------------|
| 数据预聚合         | 在数据库层面完成分组统计，减少传输量                         | 数据量从百万行降到几百行      |
| 数据采样           | 对散点图等保留原始点的图表进行随机采样                       | 渲染时间从秒级降到毫秒级      |
| 异步渲染           | 图表生成在后台线程进行，主线程保持响应                       | 界面不卡顿                    |
| 缓存               | 对相同数据和图表类型的请求缓存渲染结果                       | 重复查看无需重新计算           |
| 增量更新           | 用户切换图表类型时，只重新渲染变化部分                       | 响应更快                      |
| 硬件加速           | 使用OpenGL渲染（Qt Charts支持）                              | 提升渲染性能                   |

## 6. 关键技术难点与解决方案

| **难点**                     | **解决方案**                                             |
|------------------------------|----------------------------------------------------------|
| 百万级散点图重叠             | 使用透明度 + 随机采样 + 六边形分箱（hexbin）             |
| 大数据量下前端渲染卡顿       | 后端预聚合 + 采样，前端只接收少量数据点                   |
| 多种图表类型动态切换         | 使用工厂模式创建不同图表，复用数据适配器                 |
| 跨平台图表一致性             | 分别使用Qt Charts和ECharts，各自优化，保持API统一        |
| 用户自定义颜色主题           | 提供配色方案配置文件，支持动态切换                       |
