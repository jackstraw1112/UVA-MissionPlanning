## 4. 分析流程

### 4.1 目标函数定义
- **单输出模型**：
  - MSE = mean((y_true - y_pred)²)
  - MAE = mean(|y_true - y_pred|)
  - MAXE = max(|y_true - y_pred|)
- **多输出模型**：
  - 加权和：J = Σ w_i * MSE_i
  - 归一化后求和

### 4.2 优化算法选择与实现

#### 4.2.1 梯度下降法
- 适用于可导模型，计算目标函数对参数的梯度，沿负梯度方向更新。
- 需要设置学习率、动量等参数。

#### 4.2.2 遗传算法
- 适用于参数空间复杂、不可导的情况。
- 种群初始化、选择、交叉、变异迭代。

#### 4.2.3 粒子群优化
- 粒子代表候选解，通过个体最优和全局最优更新速度。

#### 4.2.4 贝叶斯优化
- 适用于计算代价高的目标函数，建立代理模型指导搜索。

### 4.3 约束处理
- 参数上下界约束：优化过程中投影到可行域。
- 非线性约束：罚函数法。

### 4.4 校准验证
- 使用验证集计算校准后模型的误差。
- 绘制校准前后散点图（真实值 vs 预测值）。
- 计算误差指标提升百分比。

# 反辐射无人机仿真系统——模型校准模块开发实现方案

## 1. 总体架构

模型校准模块采用**优化求解器+误差分析引擎**架构，通过调整模型参数使模型输出与真实试验数据之间的误差最小化。模块支持多种优化算法、约束处理和校准验证，与模型库、数据管理模块紧密集成。

### 1.1 核心类设计
- **ModelCalibrator**：总控类，管理校准流程，协调数据加载、优化求解、验证评估。
- **ObjectiveFunction**：抽象基类，定义目标函数计算接口，支持MSE、MAE、加权和等。
- **Optimizer**：抽象优化算法基类，派生实现梯度下降、遗传算法、粒子群、贝叶斯优化等。
- **ConstraintHandler**：处理参数边界约束、非线性约束。
- **Validator**：验证校准后模型性能，计算误差指标、绘制对比图。
- **ReportGenerator**：生成校准报告（HTML/PDF）。

```cpp
class ObjectiveFunction {
public:
    virtual double evaluate(const Eigen::VectorXd& params) = 0;
    virtual int getNumParams() const = 0;
    virtual Eigen::VectorXd getGradient(const Eigen::VectorXd& params) {
        // 默认使用有限差分
        return finiteDifferenceGradient(params);
    }
};

class MSEObjective : public ObjectiveFunction {
public:
    MSEObjective(Model* model, const Eigen::MatrixXd& X, const Eigen::VectorXd& y)
        : m_model(model), m_X(X), m_y(y) {}
    
    double evaluate(const Eigen::VectorXd& params) override {
        m_model->setParameters(params);
        Eigen::VectorXd y_pred = m_model->predict(m_X);
        return (y_pred - m_y).squaredNorm() / m_y.size();
    }
    
private:
    Model* m_model;
    Eigen::MatrixXd m_X;
    Eigen::VectorXd m_y;
};
```

### 3.2 优化算法实现

#### 3.2.1 梯度下降法（带动量）
```cpp
class GradientDescent : public Optimizer {
public:
    OptimizationResult optimize(ObjectiveFunction* obj, 
                                const Eigen::VectorXd& initial,
                                const OptimizationParams& params) override {
        Eigen::VectorXd x = initial;
        Eigen::VectorXd v = Eigen::VectorXd::Zero(x.size());
        std::vector<double> history;
        
        for (int iter = 0; iter < params.maxIterations; ++iter) {
            double fx = obj->evaluate(x);
            history.push_back(fx);
            
            if (fx < params.tolerance) break;
            
            Eigen::VectorXd grad = obj->getGradient(x);
            
            // 动量更新
            v = params.momentum * v - params.learningRate * grad;
            x += v;
            
            // 边界约束处理
            x = x.cwiseMax(params.lowerBound).cwiseMin(params.upperBound);
            
            emit progressUpdated(iter, fx);
        }
        
        return {x, history};
    }
};
```

#### 3.2.2 遗传算法
```cpp
class GeneticAlgorithm : public Optimizer {
public:
    struct Individual {
        Eigen::VectorXd genes;
        double fitness;
    };
    
    OptimizationResult optimize(ObjectiveFunction* obj,
                                const Eigen::VectorXd& initial,
                                const OptimizationParams& params) override {
        int popSize = params.populationSize;
        int nParams = obj->getNumParams();
        
        // 初始化种群
        std::vector<Individual> population(popSize);
        for (int i = 0; i < popSize; ++i) {
            population[i].genes = initializeRandom(nParams, params);
            population[i].fitness = obj->evaluate(population[i].genes);
        }
        
        std::vector<double> history;
        
        for (int gen = 0; gen < params.maxGenerations; ++gen) {
            // 选择
            auto parents = selection(population, popSize/2);
            
            // 交叉
            auto offspring = crossover(parents, popSize - parents.size());
            
            // 变异
            mutate(offspring, params.mutationRate);
            
            // 评估新个体
            for (auto& ind : offspring) {
                ind.fitness = obj->evaluate(ind.genes);
            }
            
            // 合并新一代
            population = merge(population, offspring);
            
            // 记录最佳适应度
            double bestFitness = population[0].fitness;
            history.push_back(bestFitness);
            
            if (bestFitness < params.tolerance) break;
        }
        
        return {population[0].genes, history};
    }
};
```

#### 3.2.3 粒子群优化
```cpp
class ParticleSwarm : public Optimizer {
public:
    struct Particle {
        Eigen::VectorXd position;
        Eigen::VectorXd velocity;
        Eigen::VectorXd bestPosition;
        double bestFitness;
    };
    
    OptimizationResult optimize(ObjectiveFunction* obj,
                                const Eigen::VectorXd& initial,
                                const OptimizationParams& params) override {
        int nParticles = params.populationSize;
        int nParams = obj->getNumParams();
        
        std::vector<Particle> particles(nParticles);
        Eigen::VectorXd globalBestPosition;
        double globalBestFitness = std::numeric_limits<double>::max();
        
        // 初始化粒子
        for (auto& p : particles) {
            p.position = initializeRandom(nParams, params);
            p.velocity = Eigen::VectorXd::Zero(nParams);
            p.bestPosition = p.position;
            p.bestFitness = obj->evaluate(p.position);
            
            if (p.bestFitness < globalBestFitness) {
                globalBestFitness = p.bestFitness;
                globalBestPosition = p.position;
            }
        }
        
        std::vector<double> history;
        
        for (int iter = 0; iter < params.maxIterations; ++iter) {
            for (auto& p : particles) {
                // 更新速度
                Eigen::VectorXd r1 = randomVector(nParams);
                Eigen::VectorXd r2 = randomVector(nParams);
                
                p.velocity = params.inertia * p.velocity +
                             params.c1 * r1.cwiseProduct(p.bestPosition - p.position) +
                             params.c2 * r2.cwiseProduct(globalBestPosition - p.position);
                
                // 更新位置
                p.position += p.velocity;
                p.position = p.position.cwiseMax(params.lowerBound).cwiseMin(params.upperBound);
                
                // 评估
                double fitness = obj->evaluate(p.position);
                
                // 更新个体最优
                if (fitness < p.bestFitness) {
                    p.bestFitness = fitness;
                    p.bestPosition = p.position;
                }
                
                // 更新全局最优
                if (fitness < globalBestFitness) {
                    globalBestFitness = fitness;
                    globalBestPosition = p.position;
                }
            }
            
            history.push_back(globalBestFitness);
            
            if (globalBestFitness < params.tolerance) break;
        }
        
        return {globalBestPosition, history};
    }
};
```

#### 3.3.1 边界约束
```cpp
class BoundedConstraint : public Constraint {
public:
    BoundedConstraint(const Eigen::VectorXd& lower, const Eigen::VectorXd& upper)
        : m_lower(lower), m_upper(upper) {}
    
    Eigen::VectorXd project(const Eigen::VectorXd& x) override {
        return x.cwiseMax(m_lower).cwiseMin(m_upper);
    }
    
    double penalty(const Eigen::VectorXd& x) override {
        double penalty = 0.0;
        for (int i = 0; i < x.size(); ++i) {
            if (x(i) < m_lower(i)) penalty += std::pow(m_lower(i) - x(i), 2);
            if (x(i) > m_upper(i)) penalty += std::pow(x(i) - m_upper(i), 2);
        }
        return penalty;
    }
};
```

### 3.4 校准验证

#### 3.4.1 误差指标计算
```cpp
struct CalibrationMetrics {
    double beforeMSE;
    double afterMSE;
    double beforeMAE;
    double afterMAE;
    double improvement;
    
    static CalibrationMetrics compute(Model* model, 
                                      const Eigen::MatrixXd& X, 
                                      const Eigen::VectorXd& y,
                                      const Eigen::VectorXd& initialParams,
                                      const Eigen::VectorXd& calibratedParams) {
        CalibrationMetrics metrics;
        
        model->setParameters(initialParams);
        Eigen::VectorXd y_pred_before = model->predict(X);
        metrics.beforeMSE = (y_pred_before - y).squaredNorm() / y.size();
        metrics.beforeMAE = (y_pred_before - y).cwiseAbs().mean();
        
        model->setParameters(calibratedParams);
        Eigen::VectorXd y_pred_after = model->predict(X);
        metrics.afterMSE = (y_pred_after - y).squaredNorm() / y.size();
        metrics.afterMAE = (y_pred_after - y).cwiseAbs().mean();
        
        metrics.improvement = (metrics.beforeMSE - metrics.afterMSE) / metrics.beforeMSE * 100;
        
        return metrics;
    }
};
```

## 6. 关键技术难点与解决方案

| **难点**                     | **解决方案**                                             |
|------------------------------|----------------------------------------------------------|
| 多局部极小值问题             | 使用多种初始值尝试，或采用全局优化算法（遗传算法、粒子群） |
| 参数敏感性分析               | 校准前进行敏感性分析，固定不敏感参数，降低维度            |
| 过拟合校准数据               | 使用验证集监控，早停策略，正则化目标函数                  |
| 非线性约束处理               | 罚函数法 + 可行域投影                                     |
| 与仿真引擎集成               | 提供统一模型接口，校准模块可调用任意实现该接口的模型      |

## 7. 针对反辐射无人机的特定校准

### 7.1 气动参数校准
- **可调参数**：升力系数、阻力系数、力矩系数
- **校准数据**：风洞试验数据、飞行试验数据
- **约束条件**：参数物理合理范围（如升力系数0.1-2.0）

### 7.2 导引头参数校准
- **可调参数**：探测概率模型参数、测量噪声方差
- **校准数据**：半实物仿真数据、外场测试数据
- **目标函数**：探测概率匹配度、测量误差最小化

### 7.3 发动机模型校准
- **可调参数**：推力曲线系数、油耗系数
- **校准数据**：台架试验数据
- **约束条件**：推力随油门单调递增
