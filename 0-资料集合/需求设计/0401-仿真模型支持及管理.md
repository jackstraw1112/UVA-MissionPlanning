## 3. 分析流程

### 3.1 模型库组织结构
模型库采用分层目录结构存储，按类型分类：
```
/models
  ├── uav/                # 反辐射无人机
  │    ├── JWS-01.xml
  │    └── custom_uav.xml
  ├── aircraft/           # 飞机（敌方）
  ├── missile/            # 导弹
  ├── ship/               # 舰船
  ├── threat/             # 威胁力量（地面防空系统）
  └── radar/              # 雷达系统
```
每个模型文件包含模型的完整定义，包括属性标签和行为脚本引用（或内嵌）。

### 3.2 模型加载流程
1. 仿真引擎启动时，扫描模型目录，解析所有XML文件，建立内存中的模型原型对象。
2. 对每个模型，校验属性完整性，预编译行为脚本（若启用）。
3. 生成模型索引，供仿真任务快速查询。

### 3.3 自定义模型流程
1. 用户选择“新建模型”，指定类型。
2. 系统根据类型加载默认属性模板，用户编辑属性。
3. 用户编写行为逻辑脚本（可基于模板修改）。
4. 系统进行：
   - 属性校验（数值范围、必填项）
   - 脚本语法检查
   - 模型名称唯一性检查
5. 保存为XML文件到对应目录。

### 3.4 模型实例化
在仿真推演中，用户通过拖拽模型到场景中创建实例。实例继承模型的原型属性，但可单独调整（如初始位置、状态）。系统需支持原型与实例的关联关系。

### 3.5 行为逻辑执行
行为脚本在仿真循环中被调用，通过API获取自身状态和战场态势，输出控制指令（如机动、武器发射）。脚本运行在沙箱中，防止死循环和资源滥用。

# 反辐射无人机仿真系统——仿真模型支持及管理模块开发实现方案

## 1. 总体技术架构

针对仿真模型支持及管理需求，采用**分层解耦、插件式扩展、元数据驱动**的架构，实现对反辐射无人机、飞机、导弹、舰船、威胁力量、雷达系统等仿真模型的统一管理，并支持用户自定义模型属性和行为逻辑。

### 1.1 系统分层架构

| **层级** | **功能描述** | **技术选型** |
|----------|--------------|--------------|
| **应用层** | 模型管理界面、属性编辑器、行为脚本编辑器 | Qt Widgets, QScintilla（语法高亮） |
| **模型管理层** | 模型库索引、版本控制、依赖管理 | SQLite, Git-LFS |
| **模型定义层** | 模型数据结构、属性定义、参数校验 | XML Schema, JSON Schema |
| **行为引擎层** | 脚本执行沙箱、事件驱动、行为调度 | LuaJIT / Python 嵌入 |
| **仿真接口层** | 模型实例化、仿真数据交换 | HLA/RTI接口、共享内存、FMI/FMU标准 |

### 1.2 开发路线

1. **模型格式定义**：设计基于XML的模型描述语言，定义各类模型的属性模板
2. **模型库管理**：实现模型文件的导入、导出、版本控制
3. **属性编辑器**：根据模型类型动态生成属性编辑界面
4. **行为脚本系统**：集成脚本引擎，支持用户编写行为逻辑
5. **模型实例化**：实现从模型原型到仿真实例的映射
6. **性能优化**：针对2000+模型实例的懒加载和内存优化

## 3. 模型管理核心实现

### 3.1 模型加载与索引

```cpp
class ModelLibrary : public QObject {
    Q_OBJECT
public:
    bool initialize(const QString& libraryPath) {
        // 扫描模型目录
        QDirIterator it(libraryPath, QStringList() << "*.xml", 
                        QDir::Files, QDirIterator::Subdirectories);
        
        while (it.hasNext()) {
            QString filePath = it.next();
            ModelMetaData meta = parseModelFile(filePath);
            
            // 建立类型索引
            m_modelsByType[meta.type].append(meta);
            
            // 建立名称索引
            m_modelsByName[meta.name] = meta;
            
            // 记录文件路径
            m_modelPaths[meta.id] = filePath;
        }
        
        // 加载行为脚本索引
        loadBehaviorIndex(libraryPath + "/behavior");
        
        return true;
    }
    
    // 获取模型原型（懒加载）
    ModelPrototypePtr getPrototype(const QString& modelId) {
        if (m_prototypes.contains(modelId)) {
            return m_prototypes[modelId];
        }
        
        // 从XML文件加载完整模型
        ModelPrototypePtr proto = loadModelFromFile(m_modelPaths[modelId]);
        m_prototypes[modelId] = proto;
        return proto;
    }
    
private:
    QMap<QString, QList<ModelMetaData>> m_modelsByType;  // 类型索引
    QMap<QString, ModelMetaData> m_modelsByName;         // 名称索引
    QMap<QString, QString> m_modelPaths;                 // 文件路径
    QMap<QString, ModelPrototypePtr> m_prototypes;       // 原型缓存
    QScopedPointer<BehaviorEngine> m_behaviorEngine;     // 行为引擎
};
```

### 3.2 动态属性编辑器

根据模型类型动态生成编辑界面，参考NVIDIA Isaac Sim中通过USD属性暴露变量的方法：

```cpp
class DynamicPropertyEditor : public QWidget {
    Q_OBJECT
public:
    void setModel(const ModelPrototypePtr& model) {
        clearLayout(m_layout);
        
        // 按类别分组显示属性
        for (const auto& category : model->getCategories()) {
            QGroupBox* groupBox = new QGroupBox(category);
            QFormLayout* formLayout = new QFormLayout;
            
            for (const auto& param : model->getParametersInCategory(category)) {
                QWidget* editor = createEditorForParameter(param);
                formLayout->addRow(param.displayName + ":", editor);
                
                // 连接信号以便实时更新
                connect(editor, &QWidget::editingFinished, [=]() {
                    emit parameterChanged(param.name, getEditorValue(editor));
                });
            }
            
            groupBox->setLayout(formLayout);
            m_layout->addWidget(groupBox);
        }
    }
    
private:
    QWidget* createEditorForParameter(const ModelParameter& param) {
        switch (param.type) {
        case ParamType::Double:
            return createDoubleSpinBox(param);
        case ParamType::Integer:
            return createSpinBox(param);
        case ParamType::Enum:
            return createComboBox(param);
        case ParamType::Boolean:
            return createCheckBox(param);
        case ParamType::DoubleRange:
            return createRangeEditor(param);
        default:
            return createLineEdit(param);
        }
    }
```

```cpp
class BehaviorEngine {
public:
    bool loadScript(const QString& scriptPath, const QString& modelId) {
        // 读取脚本文件
        QFile file(scriptPath);
        if (!file.open(QIODevice::ReadOnly)) return false;
        
        QString scriptContent = file.readAll();
        
        // 创建Lua状态机
        lua_State* L = luaL_newstate();
        luaL_openlibs(L);
        
        // 注册仿真API
        registerSimulationAPI(L);
        
        // 编译脚本
        if (luaL_loadstring(L, scriptContent.toUtf8().constData()) != LUA_OK) {
            QString error = lua_tostring(L, -1);
            emit scriptError(modelId, error);
            return false;
        }
        
        // 存储编译后的脚本
        m_compiledScripts[modelId] = L;
        
        return true;
    }
    
    // 执行脚本的do_step函数（每帧调用）
    void executeStep(const QString& modelId, double time, double dt) {
        lua_State* L = m_compiledScripts[modelId];
        
        // 获取do_step函数
        lua_getglobal(L, "do_step");
        
        // 压入参数
        lua_pushnumber(L, time);
        lua_pushnumber(L, dt);
        
        // 调用函数
        if (lua_pcall(L, 2, 1, 0) != LUA_OK) {
            QString error = lua_tostring(L, -1);
            emit scriptRuntimeError(modelId, error);
        }
        
        // 获取返回值（控制指令）
        ControlCommand cmd = parseControlCommand(L, -1);
        
        // 将控制指令传递给仿真引擎
        emit controlCommandGenerated(modelId, cmd);
    }
    
private:
    void registerSimulationAPI(lua_State* L) {
        // 注册获取自身状态的函数
        lua_register(L, "get_position", lua_get_position);
        lua_register(L, "get_velocity", lua_get_velocity);
        lua_register(L, "get_sensor_data", lua_get_sensor_data);
        
        // 注册控制指令函数
        lua_register(L, "set_target", lua_set_target);
        lua_register(L, "launch_missile", lua_launch_missile);
        lua_register(L, "change_heading", lua_change_heading);
    }
    
    QMap<QString, lua_State*> m_compiledScripts;
};
```

```lua
-- 定义模型初始化
function on_init()
    -- 初始化状态变量
    state = {
        mode = "SEARCH",  -- SEARCH, TRACK, ATTACK, EVADE
        target_id = nil,
        search_center = {x=0, y=0, z=0},
        search_radius = 5000
    }

    max_speed = get_parameter("maxSpeed")
    
    return true
end

-- 每帧执行（时间步进）
function do_step(time, dt)
    -- 获取当前状态
    local pos = get_position()
    local sensors = get_sensor_data()
    
    -- 状态机
    if state.mode == "SEARCH" then
        -- 执行搜索行为
        execute_search(pos, sensors, dt)
        
        -- 检测到目标则切换状态
        if sensors.target_detected then
            state.mode = "TRACK"
            state.target_id = sensors.target_id
            log("Target detected, switching to TRACK mode")
        end
        
    elseif state.mode == "TRACK" then
        -- 执行跟踪行为
        execute_track(pos, sensors, dt)
        
        -- 进入攻击范围则攻击
        if sensors.target_distance < 20 then  -- 20km
            state.mode = "ATTACK"
            launch_missile(state.target_id)
        end
        
    elseif state.mode == "ATTACK" then
        -- 攻击后脱离
        if sensors.missile_expended then
            state.mode = "EVADE"
        end
        
    elseif state.mode == "EVADE" then
        -- 规避威胁
        execute_evade(pos, sensors, dt)
        
        -- 规避结束返回搜索
        if sensors.threat_level < 0.3 then
            state.mode = "SEARCH"
        end
    end
    
    return true
end

function execute_search(pos, sensors, dt)
    -- 螺旋搜索算法
    local angle = time * 0.1  -- 随时间变化的角度
    local radius = 1000 + time * 50  -- 逐渐扩大搜索半径
    
    local target_x = state.search_center.x + radius * math.cos(angle)
    local target_y = state.search_center.y + radius * math.sin(angle)
    
    -- 设置目标航向
    change_heading(target_x, target_y, 3000)  -- 高度3000m
end

function execute_track(pos, sensors, dt)
    if state.target_id then
        local target_pos = get_target_position(state.target_id)
        -- 跟踪目标
        change_heading(target_pos.x, target_pos.y, target_pos.z + 500)
    end
end

function execute_evade(pos, sensors, dt)
    -- 急转爬升脱离
    local threat_dir = sensors.threat_direction
    local evade_dir = (threat_dir + 180) % 360  -- 反方向逃离
    change_heading_by_angle(evade_dir, pos.z + 1000)
end

-- 模型销毁时的清理
function on_finish()
    log("Mission completed, returning to base")
    return true
end
```

```cpp
class ScriptCache {
public:
    // 编译并缓存脚本
    CompiledScriptPtr compile(const QString& scriptContent) {
        // 计算哈希
        QByteArray hash = QCryptographicHash::hash(
            scriptContent.toUtf8(), QCryptographicHash::Sha256);
        
        // 检查缓存
        if (m_cache.contains(hash)) {
            return m_cache[hash];
        }
        
        // 编译脚本
        CompiledScriptPtr compiled = doCompile(scriptContent);
        
        // 缓存
        m_cache[hash] = compiled;
        
        return compiled;
    }
    
    // 批量执行相同脚本的实例（向量化执行）
    void executeBatch(const QList<ModelInstancePtr>& instances, 
                      double time, double dt) {
        if (instances.isEmpty()) return;
        
        // 获取编译后的脚本（所有实例共享）
        CompiledScriptPtr script = instances.first()->getCompiledScript();
        
        // 向量化执行：将实例数据打包传递给脚本
        script->executeBatch(instances, time, dt);
    }
    
private:
    QCache<QByteArray, CompiledScript> m_cache;
};
```

### 5.3 实例池与内存优化

借鉴OpenUSD的图层合成技术，通过实例池复用内存：

```cpp
class ModelInstancePool {
public:
    ModelInstancePool(int maxInstances) : m_maxInstances(maxInstances) {
        // 预分配内存池
        m_pool.resize(maxInstances * sizeof(ModelInstance));
        m_freeList.reserve(maxInstances);
        
        // 初始化空闲列表
        for (int i = 0; i < maxInstances; ++i) {
            m_freeList.append(i);
        }
    }
    
    ModelInstancePtr allocate(const QString& modelId) {
        QMutexLocker locker(&m_mutex);
        
        if (m_freeList.isEmpty()) {
            return nullptr;  // 池满
        }
        
        int index = m_freeList.takeFirst();
        
        // 从内存池中构造实例
        void* memory = m_pool.data() + index * sizeof(ModelInstance);
        ModelInstance* instance = new (memory) ModelInstance(modelId);
        instance->setPoolIndex(index);
        
        return ModelInstancePtr(instance, [this](ModelInstance* inst) {
            deallocate(inst);
        });
    }
    
    void deallocate(ModelInstance* instance) {
        QMutexLocker locker(&m_mutex);
        
        // 调用析构函数
        instance->~ModelInstance();
        
        // 归还索引
        m_freeList.append(instance->poolIndex());
    }
    
private:
    int m_maxInstances;
    QByteArray m_pool;              // 内存池
    QList<int> m_freeList;          // 空闲索引
    QMutex m_mutex;
};
```
