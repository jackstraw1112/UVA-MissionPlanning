## 2. 关键术语解释

| **术语** | **解释** |
|----------|----------|
| **描述性统计分析** | 通过计算集中趋势（均值、中位数）、离散程度（方差、标准差）、分布形状（偏度、峰度）以及分位数等统计量，概括和描述数据的基本特征。 |
| **相关性分析** | 研究两个或多个变量之间关联程度的统计方法，常用指标为相关系数，如皮尔逊相关系数（衡量线性相关）和斯皮尔曼秩相关系数（衡量单调相关）。 |
| **可视化呈现** | 使用图形（如直方图、箱线图、散点图、热力图等）直观展示数据分布、趋势和关系，帮助用户快速理解数据特征。 |
| **直方图** | 将数据划分为若干连续的区间（组），用矩形条的高度表示落在每个区间内的数据频数或频率，用于展示单变量的分布形态。 |
| **箱线图** | 通过最小值、第一四分位数、中位数、第三四分位数和最大值（可能含异常值）展示数据分布，常用于识别离群点和比较多个组。 |
| **散点图** | 在二维坐标系中用点表示两个变量的观测值，用于探索两变量间的关系。 |
| **热力图** | 用颜色深浅表示矩阵中数值的大小，常用于显示相关系数矩阵，便于快速识别强相关变量。 |

## 4. 分析流程

### 4.1 描述性统计分析
- 对每个数值型变量，计算：
  - 样本量（N）
  - 最小值（Min）
  - 最大值（Max）
  - 均值（Mean）
  - 中位数（Median）
  - 众数（Mode，可选）
  - 方差（Variance）
  - 标准差（Standard Deviation）
  - 偏度（Skewness）
  - 峰度（Kurtosis）
  - 第一四分位数（Q1）、第三四分位数（Q3）
  - 四分位距（IQR）
- 结果以表格形式展示，可点击列头排序。

### 4.2 相关性分析
- 计算所选变量两两之间的相关系数：
  - 皮尔逊相关系数：适用于连续、正态分布数据。
  - 斯皮尔曼秩相关系数：适用于不满足正态分布或有序数据。
- 生成相关系数矩阵，并在热力图中用颜色映射相关系数大小。

### 4.3 可视化
- **直方图**：用户可选择分组数（bin）或自动优化。
- **箱线图**：可分组（如按不同任务阶段）。
- **散点图**：可添加回归线，支持按第三变量着色。
- **热力图**：显示相关系数矩阵。

## 3. 功能实现细节

### 3.1 描述性统计分析

#### 3.1.1 统计量计算
- **一次性扫描**：对每列数据只扫描一次，同时计算多个统计量（使用递推公式），减少I/O。
- **递推公式**：
  - 均值：`mean = (mean * (n-1) + x) / n`
  - 方差：使用 Welford 算法（数值稳定性好）
  - 偏度和峰度：使用高阶矩递推公式
- **分位数计算**：使用 `std::nth_element` 快速选择算法，复杂度 O(n)。

```cpp
class DescriptiveStats {
public:
    void update(double x) {
        n++;
        double delta = x - mean;
        mean += delta / n;
        double delta2 = x - mean;
        m2 += delta * delta2;
        m3 += delta * delta2 * (n-2) - 3 * delta2 * m2;
        m4 += delta * delta2 * delta2 * (n-3) + 6 * delta2 * delta2 * m2 - 4 * delta2 * m3;
    }
    
    double variance() const { return m2 / (n - 1); }
    double skewness() const { return std::sqrt(double(n)) * m3 / std::pow(m2, 1.5); }
    // ...
private:
    long n = 0;
    double mean = 0.0, m2 = 0.0, m3 = 0.0, m4 = 0.0;
};
```

#### 3.1.2 并行化
- 多列并行计算：OpenMP 循环遍历各列。
- 每列内使用上述递推公式，线程安全。

```cpp
#pragma omp parallel for
for (int col = 0; col < numCols; ++col) {
    DescriptiveStats stats;
    for (const auto& row : data) {
        stats.update(row[col].toDouble());
    }
    results[col] = stats.finalize();
}
```

### 3.2 相关性分析

#### 3.2.1 皮尔逊相关系数
- 公式：`r = Σ((x - μx)(y - μy)) / sqrt(Σ(x-μx)² * Σ(y-μy)²)`
- 一次性扫描计算协方差和方差，避免多次遍历。
- 使用 Eigen 矩阵运算加速多变量同时计算：

```cpp
Eigen::MatrixXd computePearson(const Eigen::MatrixXd& data) {
    // data 为 n行 × m列 矩阵
    Eigen::MatrixXd centered = data.rowwise() - data.colwise().mean();
    Eigen::MatrixXd cov = (centered.adjoint() * centered) / double(data.rows() - 1);
    Eigen::VectorXd invStd = cov.diagonal().cwiseSqrt().cwiseInverse();
    Eigen::MatrixXd corr = cov;
    corr = invStd.asDiagonal() * corr * invStd.asDiagonal();
    return corr;
}
```

#### 3.2.2 斯皮尔曼秩相关系数
- 先对数据排序获得秩，然后计算皮尔逊相关系数。
- 使用并行排序算法（如 `std::sort` + OpenMP）加速。

#### 3.2.3 缺失值处理
- 支持 pairwise complete：只使用两个变量均有值的观测对。

## 4. 性能优化措施

| **优化项**         | **实现方法**                                                | **效果**                     |
|--------------------|-------------------------------------------------------------|------------------------------|
| 增量计算           | 单次扫描计算多个统计量，避免重复遍历                         | 时间减少50%                  |
| 向量化计算         | 使用 Eigen 的矩阵运算代替循环                               | 相关性计算加速10倍以上        |
| 并行列处理         | OpenMP 并行遍历各列                                         | 充分利用多核                 |
| 快速分位数         | `std::nth_element` 代替全排序                               | 分位数计算 O(n)  vs O(nlogn) |
| 内存池             | 使用 Eigen 的 Matrix 直接映射数据，避免拷贝                  | 内存占用降低                 |
| 图表懒加载         | 只有当用户切换到图表标签时才渲染                             | 界面响应更快                 |
| 数据采样           | 对百万级数据，可视化前可进行随机采样（保留分布特征）         | 图表渲染时间从秒级降到毫秒级  |

### 4.1 数据采样策略
- 直方图：直接聚合计数，无需采样。
- 散点图：当数据点超过5000时，随机采样5000点，仍能反映趋势。

## 6. 关键技术难点与解决方案

| **难点**                     | **解决方案**                                             |
|------------------------------|----------------------------------------------------------|
| 大数据量下内存溢出           | 使用列式存储，按需加载；分块处理                         |
| 缺失值处理                   | 相关性计算时 pairwise complete，统计量计算时忽略缺失值   |
| 偏度、峰度数值不稳定         | 使用高阶矩递推公式，避免大数相减                         |
| 散点图重叠                   | 使用透明度或六边形分箱（hexbin）                         |
| 用户交互响应慢               | 图表渲染在后台线程进行，主线程保持响应                    |
