## 2. 关键术语解释

| **术语** | **解释** |
|----------|----------|
| **层次分析法（AHP）** | 一种多准则决策方法，通过构建两两比较判断矩阵，计算特征向量得到各指标的权重，并检验一致性。 |
| **熵权法** | 一种客观赋权法，根据各指标数据的变异程度（信息熵）确定权重，数据离散程度越大，权重越高。 |
| **归一化** | 将一组权重值转换为总和为1（或100%）的过程，通常用各权重除以总和实现。 |
| **一致性检验** | 在AHP中，用于验证判断矩阵是否逻辑一致，指标为一致性比例CR，通常要求CR < 0.1。 |
| **判断矩阵** | AHP中用于比较两两指标相对重要性的矩阵，满足正互反性（a_ij = 1/a_ji）。 |
| **正互反矩阵** | 矩阵中元素满足 a_ij > 0，且 a_ji = 1/a_ij。 |

## 4. 分析流程

### 4.1 手工输入校验流程
1. 监听所有权重输入框的`valueChanged`事件。
2. 对于每个父节点，收集其所有子节点的权重值，计算总和 `sum`。
3. 若 `abs(sum - 1.0) > 1e-6`，则显示警告“该组权重和应为1”，并高亮相关输入框。
4. 若某个权重值 `< 0`，则显示“权重不能为负数”。
5. 校验通过前，禁用保存按钮。

### 4.2 AHP计算流程
1. 读取判断矩阵 `A`（n×n）。
2. 验证矩阵是否为正互反矩阵：所有元素>0，且 `A[i][j] * A[j][i] ≈ 1`。
3. 计算最大特征值 `λmax` 和对应的特征向量（可使用幂法或和积法）。
4. 归一化特征向量得到权重 `w`。
5. 计算一致性指标 `CI = (λmax - n)/(n-1)`。
6. 根据矩阵阶数查表得随机一致性指标 `RI`，计算 `CR = CI/RI`。
7. 返回权重和CR值，若CR≥0.1则提示警告。

### 4.3 熵权法计算流程
1. 读取原始数据矩阵 `X`（m×n），按指标方向处理：
   - 正向指标：保持不变。
   - 负向指标：取倒数或取负（常用方法为取倒数或 `max - x`）。
2. 数据标准化：计算第j项指标下第i个样本的比重 `p_ij = x_ij / Σ_i x_ij`。
3. 计算第j项指标的信息熵 `e_j = -k * Σ_i p_ij * ln(p_ij)`，其中 `k = 1/ln(m)`。
4. 计算信息冗余度 `d_j = 1 - e_j`。
5. 归一化得到权重 `w_j = d_j / Σ_j d_j`。
6. 返回权重。

## 2. 手工输入权重实现

### 2.1 界面交互
- 在指标体系树或表格中，每个指标旁边显示一个`QDoubleSpinBox`用于输入权重。
- 当选中某个父节点时，其下所有子节点的输入框会高亮，并显示“子节点权重和”提示。

### 2.2 实时校验逻辑
- 监听所有权重输入框的`valueChanged`信号。
- 对每个父节点，收集其子节点权重，计算和。
- 使用`abs(sum - 1.0) < 1e-6`判断归一化。
- 若校验失败，禁用保存按钮，并显示红色警告文字。
- 提供“归一化”按钮，自动将当前组所有权重按比例调整为和为1。

### 2.3 代码示例
```cpp
void WeightSettingWidget::onWeightChanged() {
    m_validationErrors.clear();
    // 按父节点分组验证
    for (auto parentId : m_hierarchy.keys()) {
        double sum = 0.0;
        for (auto childId : m_hierarchy[parentId]) {
            sum += m_weightSpinBoxes[childId]->value();
        }
        if (abs(sum - 1.0) > 1e-6) {
            m_validationErrors[parentId] = tr("子节点权重和应为1，当前为 %1").arg(sum, 0, 'f', 3);
        }
    }
    updateValidationUI();
}

void WeightSettingWidget::normalizeGroup(int parentId) {
    double sum = 0.0;
    for (auto childId : m_hierarchy[parentId]) {
        sum += m_weightSpinBoxes[childId]->value();
    }
    if (sum > 0) {
        for (auto childId : m_hierarchy[parentId]) {
            double newVal = m_weightSpinBoxes[childId]->value() / sum;
            m_weightSpinBoxes[childId]->setValue(newVal);
        }
    }
}
```

## 3. 自动计算权重实现

### 3.1 层次分析法（AHP）

#### 3.1.1 数据输入
- 提供`QTableWidget`用于输入n×n判断矩阵，默认填充1。
- 支持从Excel/CSV导入矩阵。
- 自动检查正互反性：`if (matrix(i,j) * matrix(j,i) - 1.0 > 1e-6) error`

#### 3.1.2 计算流程
```cpp
QVector<double> AHPStrategy::calculate(const QVariantMap& params) {
    QVector<QVector<double>> matrix = params["matrix"].value<QVector<QVector<double>>>();
    int n = matrix.size();
    
    // 1. 计算每行几何平均数
    QVector<double> weights(n, 0.0);
    for (int i = 0; i < n; ++i) {
        double product = 1.0;
        for (int j = 0; j < n; ++j) {
            product *= matrix[i][j];
        }
        weights[i] = pow(product, 1.0 / n);
    }
    
    // 2. 归一化
    double sum = std::accumulate(weights.begin(), weights.end(), 0.0);
    for (auto& w : weights) w /= sum;
    
    // 3. 计算最大特征值
    QVector<double> AW(n, 0.0);
    for (int i = 0; i < n; ++i) {
        for (int j = 0; j < n; ++j) {
            AW[i] += matrix[i][j] * weights[j];
        }
    }
    double lambdaMax = 0.0;
    for (int i = 0; i < n; ++i) {
        lambdaMax += AW[i] / weights[i];
    }
    lambdaMax /= n;
    
    // 4. 一致性检验
    double CI = (lambdaMax - n) / (n - 1);
    double RI = getRandomIndex(n); // 查表
    m_consistencyRatio = CI / RI;
    
    return weights;
}
```

#### 3.1.3 界面反馈
- 显示权重结果和一致性比例CR。
- 若CR≥0.1，显示警告但允许用户接受或调整。

### 3.2 熵权法

#### 3.2.1 数据输入
- 提供`QTableWidget`用于输入m×n原始数据，或导入文件。
- 增加一列复选框让用户选择指标方向（正向/负向）。

#### 3.2.2 计算流程
```cpp
QVector<double> EntropyWeightStrategy::calculate(const QVariantMap& params) {
    QVector<QVector<double>> data = params["data"].value<QVector<QVector<double>>>();
    QVector<int> directions = params["directions"].value<QVector<int>>(); // 1正向，-1负向
    int m = data.size();    // 样本数
    int n = data[0].size(); // 指标数
    
    // 1. 指标方向处理（负向指标取倒数）
    QVector<QVector<double>> processed(m, QVector<double>(n));
    for (int i = 0; i < m; ++i) {
        for (int j = 0; j < n; ++j) {
            if (directions[j] == 1) processed[i][j] = data[i][j];
            else processed[i][j] = 1.0 / data[i][j]; // 或采用其他转换
        }
    }
    
    // 2. 标准化：计算比重 p_ij
    QVector<QVector<double>> p(m, QVector<double>(n));
    for (int j = 0; j < n; ++j) {
        double sum = 0.0;
        for (int i = 0; i < m; ++i) sum += processed[i][j];
        for (int i = 0; i < m; ++i) p[i][j] = processed[i][j] / sum;
    }
    
    // 3. 计算信息熵 e_j
    QVector<double> e(n, 0.0);
    double k = 1.0 / log(m);
    for (int j = 0; j < n; ++j) {
        for (int i = 0; i < m; ++i) {
            if (p[i][j] > 0) e[j] -= p[i][j] * log(p[i][j]) * k;
        }
    }
    
    // 4. 计算权重
    QVector<double> weights(n);
    double total = 0.0;
    for (int j = 0; j < n; ++j) {
        weights[j] = 1 - e[j];
        total += weights[j];
    }
    for (auto& w : weights) w /= total;
    
    return weights;
}
```

#### 3.2.3 验证
- 检查数据是否包含负值或零（对某些转换可能不适用），若存在则提示。

## 5. 校验与异常处理

### 5.1 手工输入校验
- 负数：立即红色边框提示。
- 和不为1：底部状态栏警告，禁用保存。

### 5.2 AHP校验
- 矩阵元素必须为正。
- 满足正互反性：`|a_ij * a_ji - 1| < 1e-6`。
- 一致性比例CR ≥0.1时弹窗警告，但允许强制保存。

### 5.3 熵权法校验
- 数据不能有负值（取决于处理方式）。
- 若某指标所有值相等，熵为1，权重为0，可接受。

## 6. 性能优化

- AHP计算复杂度O(n³)，但n通常≤10，无需优化。
- 熵权法涉及大量数据，可使用并行算法（OpenMP）加速：
```cpp
#pragma omp parallel for
for (int j = 0; j < n; ++j) {
    // 计算p_ij和e_j
}
```
- 矩阵输入使用`QTableWidget`时，限制更新频率，避免卡顿。
