# 反辐射无人机任务规划（完整版）

## 1. 任务目标与约束

### 1.1 任务类型
- **摧毁**：对目标雷达实施物理摧毁，使其永久丧失作战能力
  - 硬杀伤：直接命中雷达天线、方舱等关键部位
  - 软杀伤：破坏雷达供电、通信等辅助系统
- **压制**：迫使目标雷达暂时关机或转移，丧失作战功能
  - 威慑压制：通过无人机存在威胁迫使雷达关机
  - 干扰压制：释放干扰信号降低雷达探测效能
- **诱偏**：吸引敌方防空火力或雷达注意力，为其他作战单元创造机会
  - 诱饵无人机：模拟攻击机特征，消耗敌方防空弹药
  - 佯动无人机：制造虚假攻击方向，分散敌防御注意力
- **侦察**：探测和识别敌方雷达信号特征，获取情报
  - 信号侦察：截获分析雷达信号参数
  - 位置侦察：精确测定雷达坐标
- **评估**：对打击效果进行实时评估
  - 毁伤效果评估：确认目标是否被摧毁
  - 辐射源状态评估：判断雷达是否恢复工作

### 1.2 目标优先级
- **一级目标（最高优先）**：
  - 火控雷达（对我方空中平台构成直接威胁）
  - 制导雷达（引导防空导弹）
  - 预警雷达（提供早期预警）
- **二级目标（中等优先）**：
  - 跟踪雷达
  - 搜索雷达
  - 多功能相控阵雷达
- **三级目标（一般优先）**：
  - 气象雷达
  - 导航雷达
  - 民用雷达
- **优先级影响因素**：
  - 目标类型与威胁等级
  - 目标对当前作战任务的影响程度
  - 目标的时效性（如即将关机的目标优先级降低）
  - 目标的移动性（机动雷达需优先处理）

### 1.3 时间约束
- **到达时间约束**：
  - 各无人机需在规定时间窗内到达指定区域
  - 多机协同到达的时间同步误差≤3秒
  - 考虑风速、地形等对速度的影响
- **攻击窗口约束**：
  - 雷达开机时间窗口（需在雷达工作时攻击）
  - 气象窗口（能见度、风速等满足攻击条件）
  - 火力窗口（避开敌方防空火力活跃期）
  - 电子战窗口（电磁环境相对干净的时段）
- **续航限制**：
  - 最大航程限制（一般不超过理论航程的70%）
  - 最大留空时间（含巡航、搜索、待机、攻击全程）
  - 燃油/电量余量要求（返航或备降所需）
  - 发动机/电池寿命限制

### 1.4 攻击策略
- **单机攻击**：
  - 适用于单个高价值目标，且无协同需求
  - 可采用直接俯冲攻击或盘旋后攻击
  - 优势：指挥简单，资源消耗少
  - 劣势：突防概率低，易被拦截
- **多机协同攻击**：
  - **饱和攻击**：多机同时从不同方向攻击，使防空系统过载
  - **时序攻击**：分批次连续攻击，压制雷达持续关机
  - **分工攻击**：部分无人机负责压制/干扰，部分负责摧毁
  - **诱骗攻击**：诱饵机先暴露，吸引火力，主攻机随后突防

### 1.5 兵力需求计算

#### 1.5.1 辐射源数量
- 单目标单辐射源：一般需要2-3架确保摧毁概率
- 单目标多辐射源：按辐射源数量×1.5倍计算
- 多目标场景：总需求为各目标需求之和，考虑协同系数0.8-0.9

#### 1.5.2 目标机动性
- 固定雷达：标准配置
- 机动雷达：增加20-30%兵力覆盖可能移动范围
- 车载/舰载雷达：需考虑平台移动速度和范围

#### 1.5.3 辐射源威胁等级
- **高威胁**（火控/制导雷达）：需要优先摧毁，建议3-4架/目标
- **中威胁**（预警/跟踪雷达）：建议2-3架/目标
- **低威胁**（搜索/气象雷达）：建议1-2架/目标

#### 1.5.4 防护强度
- **无防护**：标准配置
- **软防护**（电子干扰、诱饵）：增加30%兵力
- **硬防护**（装甲、主动防御系统）：增加50%兵力，考虑使用大口径战斗部
- **综合防护**：增加80-100%兵力，采用多方向饱和攻击

#### 1.5.5 无人机性能
- 航程/续航：短航程无人机需更多架次或中途接力
- 速度：低速无人机突防概率低，需更多数量补偿
- 隐身性能：隐身无人机可减少30-40%需求
- 抗干扰能力：抗干扰差的无人机需增加冗余

#### 1.5.6 战场环境
- **地形遮蔽**：丘陵/山地地形可减少20%需求（利用地形掩护）
- **气象条件**：恶劣天气需增加30%需求（能见度低、风速大）
- **电磁环境**：复杂电磁环境需增加50%需求（干扰严重）
- **夜间条件**：夜战需增加20%需求（搜索难度大）

#### 1.5.7 战术冗余
- **基础冗余**：按计算需求的10-20%增加备用机
- **任务关键性**：关键任务按30-50%增加冗余
- **单机可靠性**：可靠性低于90%时，按故障率线性增加
- **链路可靠性**：通信不稳定时，每架配备中继机或增加备份

#### 1.5.8 计算公式
```
N_total = Σ(N_base × k_threat × k_mobile × k_protect × k_env) × k_redun

其中：
N_base：基础需求（按辐射源数量×2）
k_threat：威胁系数（高1.5-2.0，中1.0-1.2，低0.8-1.0）
k_mobile：机动系数（固定1.0，机动1.2-1.3）
k_protect：防护系数（无1.0，软1.3，硬1.5，综合1.8-2.0）
k_env：环境系数（良好1.0，复杂1.2-1.5）
k_redun：冗余系数（1.1-1.5）
```

## 2. 作战任务分配

### 2.1 时空约束

#### 2.1.1 同时到达
- **时间同步精度**：攻击集群到达时间差≤3秒
- **协同方法**：
  - 速度调整法：通过调整巡航速度同步
  - 路径长度法：设计不同长度路径同步
  - 待机等待法：先到者在待机区盘旋等待
- **影响因素**：风速、航向、无人机性能差异

#### 2.1.2 多角度进入
- **角度间隔**：主攻方向间隔30-60度
- **扇区划分**：根据雷达探测盲区划分攻击扇区
- **最小角度**：避免无人机相互干扰，≥15度
- **最大角度**：覆盖雷达所有薄弱方向，≤120度

#### 2.1.3 高度分层
- **低空层**：100-500m，利用地形掩护突防
- **中空层**：500-2000m，标准攻击高度
- **高空层**：2000-5000m，俯瞰攻击或待机
- **分层原则**：各层垂直间隔≥300m，避免相互干扰

#### 2.1.4 安全间隔
- **水平间隔**：同高度≥3km
- **垂直间隔**：不同层≥300m
- **时间间隔**：同方向连续攻击间隔≥10秒
- **冲突预警**：实时监控位置，提前30秒预警

### 2.2 功能角色

#### 2.2.1 搜索机
- **任务**：在指定区域搜索雷达信号
- **载荷**：宽带侦察接收机、测向设备
- **行为**：保持电磁静默，仅接收不发射
- **协同**：发现目标后引导定位机/攻击机

#### 2.2.2 定位机
- **任务**：精确测定目标雷达位置
- **载荷**：高精度测向设备、干涉仪
- **行为**：多机交叉定位或单机运动定位
- **精度要求**：定位误差≤50m

#### 2.2.3 攻击机
- **任务**：对目标实施物理摧毁
- **载荷**：反辐射导弹、战斗部
- **行为**：锁定目标后俯冲攻击
- **攻击方式**：直接命中或近炸

#### 2.2.4 诱偏机
- **任务**：吸引敌方注意力和火力
- **载荷**：角反射器、有源干扰机
- **行为**：模拟攻击机信号特征
- **协同**：为攻击机创造突防窗口

#### 2.2.5 评估机
- **任务**：打击效果评估
- **载荷**：光电/红外传感器
- **行为**：攻击后低空掠过目标区域
- **评估内容**：雷达是否被摧毁、是否恢复工作

#### 2.2.6 中继机
- **任务**：通信中继，延伸控制距离
- **载荷**：通信中继设备
- **行为**：在安全空域盘旋
- **协同**：保障超视距通信

### 2.3 电磁协同

#### 2.3.1 频域协同
- **频段分配**：不同无人机工作在不同频段避免互扰
- **跳频同步**：跳频模式需协同，避免同时跳频
- **频段匹配**：根据目标雷达频段分配侦收/干扰资源

#### 2.3.2 空域协同
- **侦收空域**：不同无人机负责不同空域的信号侦收
- **干扰空域**：干扰机覆盖指定方向，避免影响友机
- **协同静默**：攻击阶段部分无人机保持静默

#### 2.3.3 时域协同
- **侦收时序**：交替侦收，避免同时接收
- **干扰时序**：分时干扰，避免信号重叠
- **静默窗口**：设定统一静默时段

#### 2.3.4 能量协同
- **功率分配**：协调干扰功率，避免过饱和
- **方向控制**：干扰波束指向协同
- **能量管理**：干扰机轮换工作，延长续航

### 2.4 动态任务重分配

#### 2.4.1 触发条件
- **目标消失**：雷达关机/移动
- **新目标出现**：探测到新辐射源
- **无人机损失**：部分无人机被击落/故障
- **任务变更**：上级指令调整
- **环境变化**：气象/电磁环境突变

#### 2.4.2 重分配策略
- **就近分配**：距离最近的无人机接替任务
- **能力匹配**：按载荷能力重新分配
- **优先级重排**：按目标优先级重新排序
- **负载均衡**：均匀分配任务，避免过载

#### 2.4.3 重分配流程
1. 检测到触发条件
2. 收集当前所有无人机状态（位置、油量、载荷）
3. 运行重分配算法（匈牙利算法/遗传算法/合同网协议）
4. 生成新分配方案
5. 下发指令，无人机确认
6. 监控执行情况

### 2.5 频段匹配与威胁优先

#### 2.5.1 频段匹配原则
- **同频段优先**：相同频段的无人机优先分配同频段目标
- **频段覆盖**：确保所有目标频段都有对应载荷覆盖
- **宽频优先**：宽频带载荷优先处理多频段目标
- **窄频专用**：窄频载荷专门处理特定频段高价值目标

#### 2.5.2 威胁优先排序
1. 火控雷达（威胁最高）
2. 制导雷达
3. 预警雷达
4. 跟踪雷达
5. 搜索雷达
6. 其他

#### 2.5.3 分配算法
```
输入：无人机集合U，目标集合T
输出：分配矩阵A

步骤：
1. 按威胁等级对T排序
2. 对每个目标t in T:
   - 筛选频段匹配的无人机U_match
   - 按距离排序U_match
   - 分配最近且油量充足的无人机
3. 若有未分配目标，触发动态重分配
4. 输出分配矩阵A
```

## 3. 情报输入

### 3.1 目标空间信息

#### 3.1.1 雷达位置
- **精确坐标**：经纬度/UTM坐标，误差≤50m
- **高程信息**：海拔高度，相对高度
- **阵地类型**：固定阵地/机动阵地/预设阵地
- **周边环境**：道路、居民区、植被覆盖

#### 3.1.2 部署范围
- **固定范围**：永久阵地面积、建筑分布
- **机动范围**：车辆可到达的区域、移动速度
- **发射范围**：导弹发射架位置、覆盖扇区
- **雷达作用范围**：最大探测距离、盲区分布

#### 3.1.3 活动区
- **日常活动区**：雷达日常工作的区域
- **战备活动区**：战时可能转移的区域
- **活动规律**：时间规律、路线规律
- **隐蔽区**：可能隐蔽的区域

### 3.2 目标信号特征

#### 3.2.1 频率
- **工作频率**：主工作频率，单位MHz/GHz
- **频率范围**：捷变频范围、跳频范围
- **频率特性**：固定频/捷变频/跳频/自适应

#### 3.2.2 脉宽
- **脉冲宽度**：τ，单位μs
- **脉宽变化**：固定/参差/自适应
- **特殊波形**：脉冲压缩波形（线性调频/相位编码）

#### 3.2.3 重复间隔
- **脉冲重复间隔**：PRI，单位μs/ms
- **PRI类型**：固定/参差/抖动/滑变
- **PRI模式**：常规/高重频/中重频/低重频

#### 3.2.4 天线扫描
- **扫描方式**：圆周/扇形/螺旋/电子扫描
- **扫描周期**：T_scan，单位秒
- **波束宽度**：水平/垂直波束宽度，单位度
- **扫描模式**：搜索/跟踪/边搜索边跟踪

#### 3.2.5 其他特征
- **极化方式**：水平/垂直/圆极化
- **信号调制**：常规/线性调频/非线性调频
- **副瓣电平**：主瓣与副瓣比值，单位dB
- **辐射功率**：峰值功率/平均功率

### 3.3 威胁排序

#### 3.3.1 按功能威胁排序
1. **火控雷达**：直接控制武器系统，威胁最高
2. **制导雷达**：引导导弹拦截，威胁次高
3. **跟踪雷达**：持续跟踪目标，威胁中高
4. **搜索雷达**：发现目标，威胁中等
5. **预警雷达**：早期预警，威胁中低
6. **气象/导航雷达**：非直接威胁，威胁低

#### 3.3.2 按作战阶段排序
- **突防阶段**：搜索/预警雷达威胁最高
- **攻击阶段**：火控/制导雷达威胁最高
- **撤离阶段**：所有雷达均构成威胁

#### 3.3.3 按平台类型排序
- **地面固定雷达**：威胁持续，但位置已知
- **地面机动雷达**：威胁高，位置不确定
- **舰载雷达**：威胁高，移动平台
- **机载雷达**：威胁高，三维机动
- **弹载雷达**：末端威胁极大

### 3.4 雷达类型

#### 3.4.1 按功能分类
- **预警雷达**：远程探测，低频段
- **搜索雷达**：中远程探测，中频段
- **跟踪雷达**：精确跟踪目标，高频段
- **火控雷达**：武器制导，极高精度
- **制导雷达**：导弹指令/半主动制导
- **多功能雷达**：兼具多种功能

#### 3.4.2 按体制分类
- **脉冲雷达**：常规脉冲体制
- **脉冲多普勒雷达**：利用多普勒效应
- **相控阵雷达**：电扫阵列，快速波束指向
- **连续波雷达**：连续发射，测速精度高
- **合成孔径雷达**：高分辨率成像

#### 3.4.3 按平台分类
- **地面雷达**：固定/机动
- **舰载雷达**：水面舰艇/潜艇
- **机载雷达**：固定翼/直升机/无人机
- **星载雷达**：卫星平台
- **弹载雷达**：导弹末制导

### 3.5 开关机规律

#### 3.5.1 时间规律
- **定时开机**：固定时间间隔开机
- **任务驱动**：根据任务需要开机
- **随机开机**：无规律可循
- **休眠模式**：长时间关机，间歇性唤醒

#### 3.5.2 应对策略
- **发现即攻击**：开机后立即攻击
- **等待时机**：等待下次开机
- **逼迫开机**：通过佯动逼迫雷达开机

### 3.6 其他因素

#### 3.6.1 防空威胁
- **防空导弹**：射程、射高、制导方式
- **高炮**：有效射程、射速
- **电子干扰**：干扰类型、功率
- **综合防空系统**：多层防御体系

#### 3.6.2 地形遮蔽
- **山区**：可利用地形掩护
- **城市**：复杂多径效应
- **植被**：信号衰减
- **水域**：多路径反射

#### 3.6.3 电磁环境
- **己方电磁辐射**：通信、雷达、干扰
- **敌方电磁辐射**：雷达、干扰、通信
- **民用电磁辐射**：广播、通信、工业干扰
- **自然电磁辐射**：雷电、太阳活动

#### 3.6.4 气象因素
- **能见度**：影响光电侦察/攻击
- **风速风向**：影响无人机飞行/定位
- **降水**：信号衰减，飞行安全
- **云层**：影响光电制导
- **温度**：影响电池/发动机性能

## 4. 路径规划

### 4.1 巡航路径规划

#### 4.1.1 威胁规避

- **禁飞区规避**：
  - 识别各类禁飞区（政治边界、民航航线、己方阵地）
  - 规划绕行路径，保证安全裕度≥5km
  - 动态更新禁飞区信息
  
- **利用地形降低发现概率**：
  - 地形掩蔽：选择山谷、山脊背面的低空路线
  - 地杂波掩护：利用地面反射形成杂波掩护
  - 植被覆盖：利用树林、建筑等遮蔽
  
- **避开敌防空火力覆盖区**：
  - 识别火力范围（防空导弹射程、高炮射程）
  - 规划路径避开火力杀伤区
  - 无法避开时，采用高速突防、低空突防策略

#### 4.1.2 地形与高度规划

- **低空/超低空突防**：
  - 平原地区：100-300m高度
  - 丘陵地区：200-500m高度（相对地面）
  - 山地地区：300-800m高度（视地形起伏）
  
- **避免过高或过低**：
  - 过低风险：撞地概率增加，燃油消耗增加
  - 过高风险：暴露概率增加，防空威胁增大
  
- **地形跟随策略**：
  - 平缓起伏：平滑跟随地形
  - 陡峭变化：提前爬升/俯冲，避免大过载
  - 地形回避：绕开过高障碍物

#### 4.1.3 航程与续航

- **航程预留**：
  - 总航程 ≤ 最大航程 × 70%（安全系数）
  - 返航航程 ≥ 当前点至安全区距离 × 1.5
  
- **燃油管理**：
  - 巡航段：经济速度，油耗最小
  - 待机段：预留燃油≥总燃油20%
  - 攻击段：预留燃油≥总燃油15%
  - 返航段：预留燃油≥总燃油20%
  
- **航线优化**：
  - 尽量直线飞行，减少不必要的转弯
  - 转弯半径 ≥ 最小转弯半径 × 1.5
  - 合理分配大机动动作（集中在低威胁区）

#### 4.1.4 进入方向

- **雷达盲区进入**：
  - 分析雷达副瓣盲区、俯仰盲区、地形遮蔽区
  - 选择雷达探测概率最低的方向
  
- **侧后方进入**：
  - 多数雷达后方探测能力较弱
  - 侧方可利用多普勒盲区
  
- **多机协同方向**：
  - 分散进入方向，间隔≥30度
  - 避免所有无人机从同一方向进入
  
- **主瓣规避**：
  - 避免正对雷达主瓣方向
  - 主瓣暴露距离≥雷达最大探测距离

#### 4.1.5 时间与协同约束

- **准时到达**：
  - 计算精确到达时间，考虑风速、航向影响
  - 时间同步误差 ≤ 3秒
  
- **多机协同时序**：
  - 主攻机与诱饵机时序错开（诱饵先10-30秒）
  - 攻击波次间隔 ≥ 30秒（避免同时暴露）
  
- **动态调整**：
  - 实时监控各机位置，调整速度保持同步
  - 先到者在待机区盘旋，后到者直接进入

#### 4.1.6 气象条件

- **风的影响**：
  - 逆风/顺风对航速的影响计算
  - 侧风对航向的修正
  
- **能见度**：
  - 低能见度时降低高度，利用地形掩护
  - 保持安全高度，避免撞地
  
- **降水**：
  - 大雨/雪时考虑信号衰减
  - 避开积雨云等危险天气

### 4.2 搜索路径规划

#### 4.2.1 目标区域信息

- **区域特征**：
  - 区域半径：R_search，通常5-20km
  - 区域面积：S_total = π × R_search²
  - 区域形状：圆形/多边形/不规则
  
- **地形地貌**：
  - 高程变化：影响信号传播
  - 遮蔽区：信号盲区
  
- **雷达部署范围**：
  - 可能位置概率分布
  - 优先搜索高概率区域

#### 4.2.2 雷达信号特征

- **频段信息**：
  - 频率范围：决定侦察接收机设置
  - 天线方向图：主瓣/副瓣方向
  
- **扫描周期**：
  - 搜索路径需覆盖≥2个扫描周期
  - 保证截获概率
  
- **开关机规律**：
  - 开机时间段内重点搜索
  - 关机时可采用被动搜索模式

#### 4.2.3 无人机平台能力

- **巡航速度**：V_cruise，影响搜索效率
- **续航时间**：T_endurance，搜索时间上限
- **高度限制**：
  - 最低安全高度
  - 最优侦察高度
- **转弯半径**：R_turn，影响路径曲率

#### 4.2.4 搜索模式

- **螺旋搜索**：
  - 从外向内/从内向外螺旋
  - 间距 = 侦察宽度 × 0.8
  - 覆盖均匀，转弯次数中等
  
- **栅格搜索**：
  - 平行线扫描，类似耕地
  - 间距 = 侦察宽度 × 0.7-0.9
  - 覆盖率高，转弯次数多
  
- **随机搜索**：
  - 随机徘徊，适用于无先验信息
  - 马尔可夫路径，覆盖率低
  
- **8字搜索**：
  - 8字形路径，适用于固定点周边
  - 持续覆盖中心区域
  
- **智能搜索**：
  - 贝叶斯更新概率图
  - 动态调整搜索方向
  - 优先搜索高概率区域

#### 4.2.5 续航与能耗

- **搜索时间约束**：
  - T_search ≤ T_endurance × 30%-40%
  - 预留返航和攻击时间
  
- **转弯优化**：
  - 尽量减少转弯次数
  - 大半径转弯省油
  
- **机动动作控制**：
  - 避免频繁加减速
  - 匀速搜索最优

#### 4.2.6 威胁规避

- **防空火力**：
  - 识别火力覆盖区，尽量避开
  - 必须进入时，高速通过
  
- **雷达主瓣**：
  - 分析雷达主瓣方向
  - 避免长时间暴露在主瓣内
  
- **重复进入**：
  - 避免反复进入同一高威胁区
  - 建立威胁记忆地图
  
- **电磁静默**：
  - 搜索阶段尽量保持电磁静默
  - 必要时采用突发发射

#### 4.2.7 多机协同搜索

- **分区搜索**：
  - 区域分割法（等面积/概率加权）
  - 各机负责一个子区域
  
- **分扇区搜索**：
  - 以目标点为中心划分扇区
  - 各机负责一个扇区
  
- **分时搜索**：
  - 交替搜索，避免信号冲突
  - 时间窗口分配
  
- **覆盖优化**：
  - 目标函数：max(覆盖率) min(重复率)
  - 重复覆盖率 ≤ 10%
  
- **空域冲突避免**：
  - 建立时空坐标系
  - 预测轨迹，避免碰撞
  
- **信息共享**：
  - 实时共享探测结果
  - 更新协同概率地图

#### 4.2.8 覆盖率计算

- **区域覆盖率**：
  - C_area = S_covered / S_total
  - 要求 ≥ 95%
  
- **重复覆盖率**：
  - C_repeat = S_repeat / S_total
  - 要求 ≤ 10%
  
- **截获概率**：
  - P_detect = f(搜索时间、信号特征、接收机性能)
  - 要求 ≥ 90%

### 4.3 待机路径（丢失目标）

#### 4.3.1 待机位置选择

- **距离目标**：
  - 最佳距离：5-15km（取决于雷达探测范围）
  - 安全距离：≥敌方防空火力射程 × 1.2
  
- **高度选择**：
  - 标准待机高度：1500-3000m
  - 低空待机：利用地形掩护，500-1000m
  
- **方位选择**：
  - 雷达侧后方（避开主瓣）
  - 地形遮蔽方向
  
- **防空火力规避**：
  - 完全避开已知火力覆盖区
  - 火力边缘需快速机动

#### 4.3.2 待机圈

- **圆形盘旋**：
  - 半径：R_hold = V_turn / (g × tan(φ))
  - 坡度：φ = 15-25度
  
- **速度选择**：
  - 经济速度：油耗最小
  - 待机速度：V_hold = 0.7-0.8 × V_cruise
  
- **圈数限制**：
  - 单次盘旋：1-3分钟
  - 最长待机圈数 ≤ T_hold / T_cycle

#### 4.3.3 待机时间

- **单次盘旋时间**：
  - T_cycle = 2πR_hold / V_hold
  - 通常1-3分钟
  
- **最长待机**：
  - T_hold ≤ T_endurance × 20%
  - 同时考虑返航油量
  
- **超时策略**：
  - 超时未发现目标：按计划返航
  - 持续搜索机会：延长待机需评估风险

#### 4.3.4 目标状态应对

- **重新开机**：
  - 立即转入攻击模式
  - 计算攻击可行性（距离、角度）
  
- **持续关机**：
  - 等待预设时间（2-5分钟）
  - 超时后按计划返航或转为搜索
  
- **信号变弱**：
  - 可能目标关机或移动
  - 向信号源方向机动确认

#### 4.3.5 多机协同待机

- **分散待机**：
  - 多机分散在不同方位
  - 覆盖多个可能开机方向
  
- **空域冲突避免**：
  - 不同高度层待机
  - 同一高度层保持安全间隔
  
- **轮流待机**：
  - 部分无人机待机，部分返航补给
  - 实现连续覆盖

#### 4.3.6 能耗与续航管理

- **待机油耗**：
  - 盘旋油耗 ≈ 巡航油耗 × 1.2
  - 待机油耗 ≤ 总油耗 × 20%
  
- **保留油量**：
  - 攻击油量 ≥ 总油耗 × 15%
  - 返航油量 ≥ 总油耗 × 20%
  
- **机动控制**：
  - 保持稳定盘旋，不做大机动
  - 避免频繁加减速

### 4.4 打击路径

#### 4.4.1 攻击几何条件

- **攻击角度**：
  - 水平进入角：根据雷达盲区选择
  - 俯冲角：30-70度（视战斗部类型）
    - 近炸引信：30-50度
    - 触发引信：50-70度
  
- **进入方向**：
  - 主瓣盲区方向优先
  - 多机分散方向
  
- **攻击距离**：
  - D_attack ≤ 导引头最大作用距离 × 0.8
  - D_attack ≥ 最小安全距离
  
- **高度差**：
  - ΔH ≥ 保证动能所需高度
  - ΔH ≥ 引信解除保险高度

#### 4.4.2 无人机自身条件

- **剩余航程**：
  - 攻击后返航航程 ≤ 剩余航程
  - 必要时可单程攻击（自杀式）
  
- **速度要求**：
  - 俯冲速度 ≥ 最小攻击速度
  - 速度 ≤ 最大结构速度
  
- **过载限制**：
  - 拉起过载 ≤ 最大可用过载 × 0.8
  - 俯冲改出需考虑过载

#### 4.4.3 威胁规避

- **火力规避**：
  - 识别敌方火力响应时间
  - 快速通过暴露区
  
- **雷达规避**：
  - 俯冲段尽量短（≤30秒）
  - 利用地形遮蔽
  
- **机动策略**：
  - 俯冲段直线（不做机动，保持速度）
  - 突然大角度改变方向（最后时刻）

#### 4.4.4 俯冲段规划

- **俯冲起点**：
  - 距离目标 D_pullup = 5-10km
  - 高度 H_pullup = 2000-4000m
  
- **俯冲剖面**：
  - 直线俯冲：角度固定，计算命中点
  - 机动俯冲：先水平接近，再突然俯冲
  
- **时间计算**：
  - T_dive = ΔH / (V_dive × sinθ)
  - 俯冲时间 ≤ 敌方反应时间

## 5. 打击

### 5.1 目标锁定条件

#### 5.1.1 信号稳定度
- **幅度稳定**：信号强度波动 ≤ 3dB
- **频率稳定**：频率漂移 ≤ 带宽 × 10%
- **PRI稳定**：PRI抖动 ≤ ±5%
- **连续接收时间**：≥ 2个扫描周期

#### 5.1.2 信号强度
- **最小强度**：≥ 接收机灵敏度 + 6dB
- **最佳强度**：-30dBm 至 -50dBm
- **强度变化率**：符合距离变化规律

#### 5.1.3 目标可信度
- **识别匹配度**：与情报库匹配 ≥ 85%
- **假目标识别**：
  - 诱饵特征：信号过于理想，无扫描调制
  - 欺骗信号：与运动规律矛盾
- **多源确认**：多个传感器确认

#### 5.1.4 位置精度
- **角度精度**：测向误差 ≤ 1° RMS
- **距离精度**（被动测距）：误差 ≤ 10%
- **定位一致性**：多机交叉定位一致

#### 5.1.5 敌我识别
- **敌我属性**：非己方信号
- **应答识别**：无正确敌我应答
- **行为识别**：符合敌方战术特征

### 5.2 攻击窗口

#### 5.2.1 雷达工作状态
- **开机窗口**：雷达正在发射信号
- **跟踪状态**：雷达处于跟踪模式（威胁高）
- **搜索状态**：雷达处于搜索模式

#### 5.2.2 攻击时机
- **最佳时机**：
  - 雷达刚开机（未反应）
  - 雷达跟踪其他目标时
  - 雷达切换目标间隙
  
- **不宜攻击**：
  - 雷达关机
  - 已知诱饵
  - 敌方火力准备就绪

### 5.3 协同攻击

#### 5.3.1 攻击方向协同
- **方向分散**：
  - 攻击方向间隔 30-60度
  - 覆盖雷达所有薄弱方向
  
- **方向规划**：
  - 主攻方向：1-2个
  - 佯攻方向：2-3个

#### 5.3.2 攻击时序协同
- **时序错开**：
  - 诱饵机先攻击（提前10-30秒）
  - 主攻机紧随其后
  
- **波次安排**：
  - 第一波：压制/干扰
  - 第二波：摧毁
  - 第三波：补射/评估

#### 5.3.3 协同突防
- **饱和攻击**：
  - 同时多方向进入
  - 使防空系统过载
  
- **分层突防**：
  - 不同高度层同时进入
  - 垂直方向上饱和

### 5.4 攻击方式

#### 5.4.1 碰炸
- **适用目标**：
  - 硬防护目标
  - 大型固定目标
  
- **引信设定**：
  - 触发延时：0-5ms（穿透延时）
  - 触发灵敏度：根据目标硬度调整

#### 5.4.2 近炸
- **适用目标**：
  - 软防护目标
  - 机动目标
  - 天线等脆弱部件
  
- **引信设定**：
  - 炸高：5-15m
  - 方位：预置破片飞散方向

#### 5.4.3 复合引信
- **近炸+碰炸**：
  - 近炸为主，碰炸为备
  - 根据目标类型自动选择

### 5.5
