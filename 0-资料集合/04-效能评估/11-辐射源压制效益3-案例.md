## 一、 仿真案例背景设定

### 1.1 作战任务
红方出动 **4架反辐射无人机**，对蓝方部署在关键隘口的 **2部火控制导雷达（Radar A 和 Radar B）** 实施压制，目标是为后续 **2架F-16攻击机编队** 开辟一条持续时间 **300秒** 的安全突防走廊。

### 1.2 初始态势与参数

| 参数项 | 数值 | 说明 |
| :--- | :--- | :--- |
| **攻击窗口** \( T_{attack} \) | 520 秒 | 从首架无人机进入雷达截获线到末架无人机脱离/自毁的总时长。 |
| **突防窗口** \( T_{escort} \) | 300 秒 | 后续F-16编队穿越危险空域的计划时长。 |
| **突防窗口在攻击窗口内的位置** | \( T+120 \)s 至 \( T+420 \)s | F-16编队利用压制效果“趁乱突防”。 |
| **Radar A 位置** | 坐标 (0, 0) | 部署于隘口北侧。 |
| **Radar B 位置** | 坐标 (15km, 5km) | 部署于隘口南侧，与A形成交叉覆盖。 |
| **两雷达对F-16的有效探测距离** | 45 km | 若雷达开机，突防航线全程被覆盖。 |
| **无人机单价** \( C_{uav} \) | 30万美元 | 假设某型反辐射无人机成本。 |
| **雷达单价** \( C_{radar} \) | 1200万美元 | 火控制导雷达单价。 |

### 1.3 蓝方雷达战术设定
- **关机规避策略**：一旦截获反辐射无人机信号，立即关机，等待 **20秒** 后尝试重启；若重启后5秒内再次截获信号，则再次关机，如此循环。
- **Radar A 操作员**：保守型，严格遵循20秒关机原则。
- **Radar B 操作员**：激进型，尝试缩短关机间隔以维持态势感知（15秒后即尝试重启）。

## 三、 评估指标计算

### 3.1 P1：雷达关机覆盖率

**公式**：
\[
P1 = \frac{1}{N} \sum_{i=1}^{N} \frac{T_{off\_i}}{T_{attack}}
\]

**参数取值**：
- \( N = 2 \)（Radar A 和 B）
- \( T_{attack} = 520 \) 秒
- **Radar A 累计关机时长** \( T_{off\_A} \)：
  - T+0 至 T+20 = 20s
  - T+25 至 T+150 = 125s
  - T+150 后被摧毁，视为永久关机直至窗口结束：150s 至 520s = 370s
  - 合计：\( 20 + 125 + 370 = 515 \) 秒
- **Radar B 累计关机时长** \( T_{off\_B} \)：
  - T+60 至 T+75 = 15s
  - T+80 至 T+180 = 100s
  - T+180 至 T+195 = 15s
  - T+200 至 T+420 = 220s
  - 合计：\( 15 + 100 + 15 + 220 = 350 \) 秒

**计算**：
\[
P1 = \frac{515 + 350}{2 \times 520} = \frac{865}{1040} \approx 0.8317 = \mathbf{83.2\%}
\]

**解读**：在520秒的攻击窗口内，两部雷达平均有 **83.2%** 的时间处于关机致盲状态，压制效果显著。

---

### 3.2 P2：区域防空压制覆盖率

**前提**：
- 保护空域：F-16突防航线，该航线全程处于Radar A和B的交叉覆盖范围内（只要任一部雷达开机，航线即被覆盖）。
- 突防窗口：\( T+120 \)s 至 \( T+420 \)s，共 **300秒**。
- **“安全”判据**：两部雷达 **同时关机** 的时段，航线才完全安全。

**根据时序数据判断同时关机时段**：

| 时间段 | Radar A 状态 | Radar B 状态 | 同时关机？ | 安全时长 |
| :--- | :--- | :--- | :--- | :--- |
| T+120 ~ T+150s | 0 | 0 | ✅ 是 | 30s |
| T+150 ~ T+180s | 被毁(0) | 0 | ✅ 是 | 30s |
| T+180 ~ T+195s | 被毁(0) | 0 | ✅ 是 | 15s |
| T+195 ~ T+200s | 被毁(0) | **1** (重启) | ❌ 否 | 0s |
| T+200 ~ T+320s | 被毁(0) | 0 | ✅ 是 | 120s |
| T+320 ~ T+420s | 被毁(0) | 0 (被命中后关机) | ✅ 是 | 100s |

**计算**：
- 总安全时长 \( T_{safe} = 30 + 30 + 15 + 120 + 100 = 295 \) 秒
- 突防窗口 \( T_{escort} = 300 \) 秒
\[
P2 = \frac{295}{300} \approx 0.9833 = \mathbf{98.3\%}
\]

**解读**：在F-16编队突防的300秒窗口内，航线 **98.3%** 的时间处于完全安全的雷达静默区。仅有5秒钟（T+195至T+200）因Radar B激进重启而短暂暴露，几乎可以忽略不计。

---

### 3.3 P3：等效战损比 (LER)

**公式**：
\[
P3 = \frac{N_{lost}}{N_{K\_Kill} \times 1.0 + N_{F\_Kill} \times 0.8}
\]

**参数取值**：
- \( N_{lost} = 3 \)
- \( N_{K\_Kill} = 1 \) (Radar A)
- \( N_{F\_Kill} = 1 \) (Radar B，因关机超过30分钟且在突防窗口内失效)

**计算**：
\[
P3 = \frac{3}{1 \times 1.0 + 1 \times 0.8} = \frac{3}{1.8} \approx \mathbf{1.67}
\]

**解读**：战损比为1.67，意味着每压制1部等效雷达需要消耗1.67架无人机。虽略高于理想值1.0，但考虑到低成本无人机（90万美元总损失）换取了高价值雷达体系功能的瘫痪，此交换比在战术上 **可接受**。

## 四、 综合评估结论

### 4.1 指标汇总与达标判定

| 评估指标 | 实测值 | 期望阈值 | 达标判定 |
| :--- | :--- | :--- | :--- |
| **P1 雷达关机覆盖率** | 83.2% | ≥ 50% | ✅ **超标达成** |
| **P2 区域防空压制覆盖率** | 98.3% | ≥ 60% | ✅ **优秀达成** |
| **P3 等效战损比 (LER)** | 1.67 | ≤ 1.5 | ⚠️ **接近达标** |

### 4.2 综合结论

> **压制有效，战术目的圆满达成。**
>
> 本次推演中，4架反辐射无人机以损失3架的代价，成功迫使蓝方2部制导雷达在83%的攻击窗口内失能，并在F-16突防的300秒内创造了 **98.3% 的安全窗口**。尽管战损比略高于理想阈值（1.67 vs 1.5），但从 **效费比** 角度分析：
> - 红方消耗成本：3架无人机 × 30万美元 = **90万美元**。
> - 蓝方直接损失：1部雷达全毁 + 1部雷达中度受损 = 约 **1500万美元** 资产损失 + 体系瘫痪造成的隐性损失。
>
> **结论：** 该方案可作为实战预案的基线参考，后续可优化无人机航路规划以减少非战斗损失（如UAV-2的故障），进一步提升战损比。
