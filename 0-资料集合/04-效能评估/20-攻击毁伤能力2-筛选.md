好的，我们对攻击毁伤能力评估大项进行最大程度的精简，保留 **2个核心指标 + 1个可选指标**，确保在减少数据采集和计算负担的同时，仍能完整回答“能不能打上？打上能不能毁？”这两个根本问题。

## 一、 精简后的评估指标体系

| 指标代号 | 指标名称 | 类别 | 回答的核心问题 |
| :--- | :--- | :--- | :--- |
| **A1** | **单发摧毁概率 ( \( P_k \) )** | **最终综合指标（必选）** | 一架无人机从发射到命中，最终让目标雷达丧失战斗力的概率是多少？ |
| **A2** | **末端制导精度 (CEP)** | **过程诊断指标（必选）** | 如果 \( P_k \) 不达标，问题出在打不准还是威力不够？ |
| **A3** | **攻击发起率 ( \( P_{launch} \) )** | **前置条件指标（可选）** | 有多少无人机根本没机会进入攻击？用于区分“平台可靠性”与“武器精度”问题。 |

**精简逻辑**：
- **A1 (\( P_k \))** 是 **最终答案**，它内聚了命中概率、引战配合、战斗部威力等所有下游因素，是指挥员和论证人员最关心的单一数字。
- **A2 (CEP)** 是 **核心变量**，\( P_k \) 的绝大部分方差由 CEP 决定。保留 CEP 可以快速诊断是“导引头不行”还是“战斗部太小”。
- **A3 (\( P_{launch} \))** 是 **前置过滤条件**。如果无人机因为链路中断根本没进入末制导，那么 CEP 和 \( P_k \) 都无从谈起。在复杂电磁环境评估中此项尤为关键，建议保留。

## 二、 各指标详细定义与计算模型

### A1. 单发摧毁概率 (\( P_k \))

**定义**：单架反辐射无人机在典型作战环境下，对典型目标雷达造成 **K级（物理摧毁）或 F级（功能瘫痪超过30分钟）** 毁伤的全任务链路概率。

**全链路计算公式**：
\[
P_k = P_{launch} \times P_{hit} \times P_{kill|hit}
\]

其中：
- **\( P_{launch} \)**：攻击发起率。若只关注武器末端效能，可设 \( P_{launch} = 1.0 \)（即仅评估成功转入末制导后的 \( P_k \)）。
- **\( P_{hit} \)**：命中概率。可根据 CEP 和战斗部有效杀伤半径 \( R_{lethal} \) 计算：
  \[
  P_{hit} = 1 - 0.5^{\left(\frac{R_{lethal}}{CEP}\right)^2}
  \]
- **\( P_{kill|hit} \)**：命中条件下的毁伤概率。该值综合了引战配合效率与战斗部威力，通常通过 **静爆试验数据** 或 **毁伤仿真** 预先测定为常数（例如 0.85）。

**简化实用公式（直接统计法）**：
若仿真次数足够多，可直接统计：
\[
P_k = \frac{N_{\text{目标被K/F级毁伤}}}{N_{\text{总仿真次数}}}
\]

---

### A2. 末端制导精度 (CEP)

**定义**：以目标雷达天线阵面中心为瞄准点，包含 **50%** 命中弹着点（或爆炸投影点）的圆半径。

**单位**：米 (m)。

**计算方法**：
1. 在仿真或打靶试验中，记录 \( N \) 次有效末制导的落点偏差 \( d_i \)（三维空间距离或平面投影距离）。
2. 将 \( d_i \) 从小到大排序。
3. 取第 \( 50\% \) 分位值即为 CEP（若 \( N \) 为偶数，取中间两位均值）。

**物理意义**：
- CEP 越小，\( P_{hit} \) 越高。
- 若 CEP > 战斗部有效杀伤半径 \( R_{lethal} \)，则命中概率急剧下降，说明 **导引头或飞控是短板**。
- 若 CEP 很小但 \( P_k \) 仍低，说明 **战斗部威力或引战配合是短板**。

---

### A3. 攻击发起率 (\( P_{launch} \))（可选）

**定义**：无人机在截获目标信号后，能够成功建立攻击航路并转入末制导俯冲阶段的比例。

**计算公式**：
\[
P_{launch} = \frac{N_{\text{转入末制导}}}{N_{\text{截获目标信号}}}
\]

**失败原因分类**：
- 数据链中断 / 指令超时。
- 目标信号丢失（雷达关机）导致导引头未锁定。
- 载机平台故障或投放异常。

**使用建议**：
- 若评估重点是 **无人机武器系统本身** 的毁伤能力，可将 \( P_{launch} \) 设为 1，单独看末段效能。
- 若评估重点是 **实战体系下的综合效能**，则必须计入此项。

## 三、 精简版仿真数据采集清单

为计算上述 2+1 项指标，仿真系统仅需记录以下 **最小数据集**：

| 数据项 | 数据类型 | 用途 |
| :--- | :--- | :--- |
| 无人机ID + 任务结果 | 标称型 | 区分：未发起攻击 / 脱靶 / 命中但未毁 / 成功毁伤 |
| 落点偏差（或命中标志） | 数值型 | 计算 CEP 和 \( P_{hit} \) |
| 目标最终状态 | 标称型 | 判定 K/F 级毁伤（计算 \( P_k \)） |
| （可选）攻击发起失败原因 | 文本型 | 辅助分析 \( P_{launch} \) 短板 |

*注：若采用直接统计法计算 \( P_k \)，则无需记录中间过程的炸高、破片分布等详细数据。*

## 四、 仿真案例演示（精简版）

### 4.1 背景与参数设定

- **仿真总次数**：\( N = 500 \) 架次（蒙特卡洛打靶）。
- **目标**：某型中程制导雷达天线阵面。
- **战斗部有效杀伤半径**：\( R_{lethal} = 7.0 \) 米（根据静爆试验，破片密度 > 1枚/㎡的半径）。
- **命中条件下毁伤概率**：\( P_{kill|hit} = 0.88 \)（预估值，由易损性模型给出）。

### 4.2 仿真输出统计

| 统计项 | 数值 | 说明 |
| :--- | :--- | :--- |
| 总仿真次数 | 500 | |
| 成功转入末制导次数 | 460 | 40次因信号丢失/链路中断未发起攻击 |
| 命中次数（落点偏差 ≤ 7.0m） | 312 | |
| 目标被成功毁伤次数（K/F级） | 275 | 含命中后毁伤成功 + 极少数近炸毁伤 |

### 4.3 指标计算

#### A3（可选）：攻击发起率
\[
P_{launch} = \frac{460}{500} = 0.92 \quad (92\%)
\]

#### A2：末端制导精度 (CEP)

将 460 次成功转入末制导的落点偏差排序：
- 第 230 位的偏差值 = **5.8 米**。
- 即 **CEP = 5.8 米**。

#### A1：单发摧毁概率 (\( P_k \))

**方法一：直接统计法（推荐）**
\[
P_k = \frac{275}{500} = 0.55 \quad (55\%)
\]

**方法二：链路计算法（用于验证模型一致性）**
- 理论命中概率：\( P_{hit} = 1 - 0.5^{(7.0 / 5.8)^2} = 1 - 0.5^{1.46} \approx 1 - 0.364 = 0.636 \)
- 实际命中次数 312 / 460 ≈ 0.678（与理论值接近，略高因部分近炸仍计入有效）。
- 综合 \( P_k = P_{launch} \times P_{hit\_actual} \times P_{kill|hit} = 0.92 \times (312/460) \times 0.88 = 0.92 \times 0.678 \times 0.88 \approx 0.549 \)（与直接统计法一致）。

### 4.4 评估结论输出

| 指标 | 实测值 | 参考阈值 | 评价 |
| :--- | :--- | :--- | :--- |
| **A1 单发摧毁概率 \( P_k \)** | **55%** | ≥ 60% 为优秀<br>40%-60% 为合格 | **合格**，单发具备中等毁伤能力。 |
| **A2 末端制导精度 CEP** | **5.8 米** | ≤ 5.0 米为优秀<br>5-8 米为良好 | **良好**，导引头精度满足基本要求。 |
| **A3 攻击发起率** | **92%** | ≥ 95% 为优秀 | **需改进**，8%的发起失败率略高，建议加强链路抗干扰。 |

**综合诊断与建议**：
- **\( P_k = 55\% \) 的瓶颈在哪里？**
  - CEP=5.8m，略大于理想值，导致理论命中概率仅约64%。若能将 CEP 提升至 4.0m，\( P_{hit} \) 可提升至约 85%，\( P_k \) 将跃升至约 **69%**。
  - 攻击发起率 92% 损失了约 4.4 个百分点的最终 \( P_k \)（\( 0.08 \times 0.55 \)）。
- **战术建议**：若需确保对关键目标 90% 以上的摧毁置信度，应采用 **双机齐射**（\( 1 - (1-0.55)^2 = 79.8\% \)）或 **三机齐射**（约 90.9%）。

## 五、 精简方案对比总结

| 对比维度 | 原方案（9项指标） | 精简方案（2+1项） |
| :--- | :--- | :--- |
| **评估结论** | 全面但冗余，需专家解读 | **直观清晰**，\( P_k \) 一锤定音 |
| **数据需求** | 需详细记录炸高、破片分布、部件级毁伤 | **仅需最终落点和目标状态** |
| **诊断能力** | 可精准定位到引信延时问题 | 通过 CEP 可区分“精度问题”与“威力问题” |
| **计算复杂度** | 需专用后处理脚本 | **Excel 即可完成** |
| **适用场景** | 装备详细设计、定型试验 | **作战方案推演、快速效能对比** |

此精简方案可直接嵌入之前的 **辐射源压制效益评估** 中，用精确的 \( P_k \) 值替代粗略的等效系数，使战损比（LER）的计算更具物理依据。
