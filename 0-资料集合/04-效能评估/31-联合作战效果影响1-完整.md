## 一、 指标体系梳理与整合

### 1.1 指标重叠分析

| 原始指标 | 与之重叠的指标 | 整合建议 |
| :--- | :--- | :--- |
| 体系破击贡献率 | 友军战损率降低幅度 | **合并**。两者均衡量“战损减少”，体系破击贡献率是战损降低幅度的另一种表述。保留 **友军战损率降低幅度** 更直观。 |
| 安全走廊开辟效率 | 友军突防成功提升率 | **合并**。突防成功提升率是走廊开辟效率的直接结果。保留 **安全走廊开辟效率** 作为过程指标，**友军突防成功提升率** 作为结果指标。 |
| 电磁频谱控制时长 | — | **保留**。该指标独立性强，侧重电磁领域的主动权。 |

| 联合打击任务耗时缩短率 | — | **保留**。从时间效率角度衡量联合作战增益。 |
| 敌方防空拦截效能下降率 | — | **保留**。直接度量压制后敌方防御能力的弱化程度。 |

### 1.2 精简后的评估指标体系（6项）

| 代号 | 指标名称 | 类型 | 核心评估问题 |
| :--- | :--- | :--- | :--- |
| **J1** | **安全走廊开辟效率** | 空间-时间型 | 为友军开辟了多大宽度、多长时间的“无防空覆盖”通道？ |
| **J2** | **友军突防成功率提升幅度** | 结果型 | 有压制相比无压制，友军成功突防的比例提高了多少？ |
| **J3** | **友军战损率降低幅度** | 结果型 | 友军在突防过程中的战损比例下降了多少？ |
| **J4** | **敌方防空拦截效能下降率** | 对抗型 | 敌方防空系统对我方编队的拦截成功率被削弱了多少？ |
| **J5** | **OODA 循环干扰增益** | 认知型 | 敌方防空指挥链的决策周期被延长了多少？ |
| **J6** | **电磁频谱控制时长占比** | 电磁型 | 在关键任务时段内，我方掌握电磁主动权的时长比例。 |

## 二、 仿真推演所需记录的原始数据清单

为支撑上述6项指标的计算，仿真系统需在推演过程中记录以下基础数据。建议采用 **“有无压制对比”** 的实验设计（即运行两次仿真：基线场景 vs 加入反辐射无人机压制场景）。

### 2.1 全局参数

| 数据项 | 符号 | 说明 |
| :--- | :--- | :--- |
| 突防任务总时长 | \( T_{mission} \) | 友军编队从进入危险空域到脱离的总时间。 |
| 友军参战平台数量 | \( N_{friend} \) | 突防编队中的战机/导弹数量。 |
| 敌方防空系统配置 | — | 雷达数量、导弹发射架数量、拦截弹数量。 |

### 2.2 时序状态数据（需逐秒或逐事件记录）

| 记录项 | 数据格式 | 用途 |
| :--- | :--- | :--- |
| **安全走廊状态** | 时间序列：每秒钟，预定义的突防航线上 **是否被至少一部敌方雷达有效覆盖**（是/否） | 计算 J1 |
| **友军平台状态** | 每个平台 ID、每秒钟位置、是否被敌方雷达跟踪、是否被导弹锁定、是否被命中 | 计算 J2, J3 |
| **敌方拦截事件** | 每次拦截导弹发射时刻、目标ID、拦截结果（命中/脱靶） | 计算 J4 |
| **敌方雷达开关机状态** | 同前文压制效益评估方案，每部雷达每秒钟的开机(1)/关机(0)状态 | 计算 J6 |
| **敌方指挥决策事件** | 从“首次发现目标”到“下达射击指令”的时间差 | 计算 J5 |

### 2.3 对比基线数据（无压制场景）

| 记录项 | 符号 | 用途 |
| :--- | :--- | :--- |
| 无压制时友军突防成功率 | \( P_{success\_base} \) | 作为 J2 的基准 |
| 无压制时友军战损率 | \( L_{base} \) | 作为 J3 的基准 |
| 无压制时敌方拦截成功率 | \( P_{intercept\_base} \) | 作为 J4 的基准 |
| 无压制时敌方 OODA 平均周期 | \( T_{OODA\_base} \) | 作为 J5 的基准 |

## 三、 各指标详细计算方法

### J1. 安全走廊开辟效率

**定义**：在友军突防窗口 \( T_{escort} \) 内，预定义突防航线上 **连续无雷达覆盖** 的最大时间窗口长度 \( T_{safe\_max} \)，以及该窗口对应的 **有效掩护宽度** \( W_{safe} \)。

**计算**：
1. 从时序数据中提取所有“安全时段”（航线上无雷达覆盖的连续时间段）。
2. 取其中最长的连续时长作为 \( T_{safe\_max} \)。
3. 有效宽度 \( W_{safe} \)：若安全时段内，整个预设走廊宽度（如20km）均未被覆盖，则 \( W_{safe} = \) 走廊全宽；若仅部分宽度安全，按实际未被覆盖的横向距离计算。

**综合效率指数**：
\[
J1 = \frac{T_{safe\_max}}{T_{escort}} \times \frac{W_{safe}}{W_{corridor}}
\]
其中 \( W_{corridor} \) 为计划开辟的走廊总宽度。该指数取值范围 [0,1]，越接近1效果越好。

---

### J2. 友军突防成功率提升幅度

**定义**：有反辐射无人机压制时，友军平台成功突防（未被击落且完成投弹/侦察任务）的比例，相对于无压制基准的提升量。

**计算**：
- 有压制时突防成功率：
  \[
  P_{success\_with} = \frac{N_{\text{成功突防平台}}}{N_{friend}}
  \]
- 提升幅度（绝对值）：
  \[
  J2 = P_{success\_with} - P_{success\_base}
  \]
- 也可用相对提升率表示：
  \[
  J2_{rel} = \frac{P_{success\_with} - P_{success\_base}}{P_{success\_base}} \times 100\%
  \]

---

### J3. 友军战损率降低幅度

**定义**：有压制时友军战损率相对于无压制基准的下降量。

**计算**：
- 有压制时战损率：
  \[
  L_{with} = \frac{N_{\text{被击落平台}}}{N_{friend}}
  \]
- 降低幅度（绝对值）：
  \[
  J3 = L_{base} - L_{with}
  \]
- 相对降低率：
  \[
  J3_{rel} = \frac{L_{base} - L_{with}}{L_{base}} \times 100\%
  \]

---

### J4. 敌方防空拦截效能下降率

**定义**：敌方防空系统对我方编队的拦截成功率，在有压制时相对于无压制基准的下降程度。

**计算**：
- 有压制时敌方拦截成功率：
  \[
  P_{intercept\_with} = \frac{N_{\text{命中友军}}}{\text{敌方发射拦截弹总数}}
  \]
- 下降率：
  \[
  J4 = \frac{P_{intercept\_base} - P_{intercept\_with}}{P_{intercept\_base}} \times 100\%
  \]
  若基准拦截成功率为0，则定义 J4 = 100%（只要压制后仍为0，也视为有效压制）。

---

### J5. OODA 循环干扰增益

**定义**：有压制时，敌方防空指挥链的平均 OODA 周期相对于无压制基准的延长比例。

**OODA 周期测量方法**：
- 在仿真中，记录从“敌方传感器首次发现我方目标”到“武器系统下达射击指令”的时间差，作为单次 OODA 周期。
- 取所有有效拦截事件的平均值。

**计算**：
- 无压制基准：\( \bar{T}_{OODA\_base} \)
- 有压制时：\( \bar{T}_{OODA\_with} \)
- 干扰增益：
  \[
  J5 = \frac{\bar{T}_{OODA\_with} - \bar{T}_{OODA\_base}}{\bar{T}_{OODA\_base}} \times 100\%
  \]

---

### J6. 电磁频谱控制时长占比

**定义**：在突防任务窗口 \( T_{mission} \) 内，我方在关键频段（如通信、火控雷达频段）上 **不受敌方干扰且敌方雷达处于关机/致盲状态** 的累计时长占比。

**判据**：
- 条件1：我方通信链路误码率低于阈值。
- 条件2：我方火控雷达/侦察设备能正常工作。
- 条件3：敌方同频段雷达处于关机状态（简化判据可直接使用之前压制效益中的“雷达关机覆盖率”）。

**计算**：
\[
J6 = \frac{\sum T_{\text{电磁优势}}}{T_{mission}} \times 100\%
\]
其中 \( T_{\text{电磁优势}} \) 为同时满足上述条件的累计时长。

## 四、 综合评估模型与结果输出

由于6项指标量纲和性质不同，建议采用 **达标判定 + 加权综合** 两种方式并行，以兼顾直观性和量化精度。

### 4.1 达标判定（红绿灯法）

为每项指标设定期望阈值，快速判定压制行动是否达到预期联合作战效果。

| 指标 | 期望阈值 | 达标判据 |
| :--- | :--- | :--- |
| J1 走廊开辟效率 | ≥ 0.6 | 开辟宽度和时长满足突防基本需求 |
| J2 突防成功率提升 | ≥ 20个百分点 | 成功率从50%提升至70%以上 |
| J3 战损率降低 | ≥ 15个百分点 | 战损率从20%降至5%以下 |
| J4 敌方拦截效能下降 | ≥ 50% | 敌方拦截成功率减半 |
| J5 OODA 干扰增益 | ≥ 100% | 敌方决策周期延长一倍 |
| J6 频谱控制时长占比 | ≥ 70% | 大部分任务时段掌握电磁权 |

**综合结论逻辑**：
- 6项全部达标 → **压制行动对联合作战贡献显著**。
- 4-5项达标 → **贡献良好，局部需优化**。
- 2-3项达标 → **贡献有限，需大幅调整方案**。
- ≤1项达标 → **压制行动对联合作战未产生实质效果**。

### 4.2 加权综合指数（可选，用于精细化比较）

**步骤1：指标归一化**

| 指标 | 归一化方法 |
| :--- | :--- |
| J1 | 直接使用指数值（0~1） |
| J2 | \( \min(1, J2 / 0.4) \) （假设提升40个百分点为满分） |
| J3 | \( \min(1, J3 / 0.3) \) （假设降低30个百分点为满分） |
| J4 | 除以100%（本身就是百分比） |
| J5 | \( \min(1, J5 / 200\%) \) |
| J6 | 除以100% |

**步骤2：权重分配（参考）**

| 指标 | 建议权重 | 理由 |
| :--- | :--- | :--- |
| J2 突防成功率提升 | 0.25 | 最直接体现对友军的保护 |
| J3 战损率降低 | 0.25 | 衡量友军生存性改善 |
| J4 敌方拦截效能下降 | 0.20 | 反映压制对敌方的削弱程度 |
| J1 走廊开辟效率 | 0.15 | 空间控制能力 |
| J6 频谱控制时长 | 0.10 | 电磁优势 |
| J5 OODA 干扰增益 | 0.05 | 认知域效果 |

**步骤3：计算联合作战贡献指数 (JCI)**

\[
JCI = \sum_{i=1}^{6} w_i \cdot \text{Score}_i \times 100
\]

## 五、 仿真案例演示（简化数据）

### 5.1 背景设定

- **任务**：4架F-35突防敌方一体化防空网，执行纵深打击。
- **压制方案**：6架反辐射无人机预先压制2部关键制导雷达。
- **对比基线**：无压制时，F-35编队突防成功率60%，战损率25%，敌方拦截成功率35%。

### 5.2 仿真结果数据

| 指标 | 有压制时测量值 |
| :--- | :--- |
| 走廊最大安全窗口 \( T_{safe\_max} \) | 180秒（总突防窗口300秒） |
| 有效掩护宽度 \( W_{safe} \) | 18km（计划走廊宽20km） |
| 友军成功突防数量 | 3架（共4架） |
| 友军被击落数量 | 1架 |
| 敌方发射拦截弹总数 | 6枚 |
| 敌方命中友军次数 | 1次 |
| 敌方平均 OODA 周期 | 85秒（基线40秒） |
| 电磁优势累计时长 | 220秒 |

### 5.3 指标计算

- **J1 走廊开辟效率**：
  \[
  J1 = \frac{180}{300} \times \frac{18}{20} = 0.6 \times 0.9 = 0.54
  \]

- **J2 突防成功率提升**：
  \[
  P_{success\_with} = 3/4 = 75\%
  \]
  \[
  J2 = 75\% - 60\% = 15\%
  \]

- **J3 战损率降低**：
  \[
  L_{with} = 1/4 = 25\%
  \]
  \[
  J3 = 25\% - 25\% = 0\%
  \]
  *（本例中战损率未降低，因幸存者偏差：虽然成功率提升，但突防成功的3架中仍有1架被击落，战损率计算基数不同）*

- **J4 敌方拦截效能下降**：
  \[
  P_{intercept\_with} = 1/6 \approx 16.7\%
  \]
  \[
  J4 = \frac{35\% - 16.7\%}{35\%} \approx 52.3\%
  \]

- **J5 OODA 干扰增益**：
  \[
  J5 = \frac{85 - 40}{40} = 112.5\%
  \]

- **J6 频谱控制时长占比**：
  \[
  J6 = \frac{220}{300} \approx 73.3\%
  \]

### 5.4 综合评估

**达标判定**：
| 指标 | 值 | 阈值 | 达标 |
| :--- | :--- | :--- | :--- |
| J1 | 0.54 | ≥0.6 | ❌ |
| J2 | 15% | ≥20% | ❌ |
| J3 | 0% | ≥15% | ❌ |
| J4 | 52.3% | ≥50% | ✅ |
| J5 | 112.5% | ≥100% | ✅ |
| J6 | 73.3% | ≥70% | ✅ |

**结论**：6项达标3项，**贡献有限**。主要短板在于安全走廊开辟不够充分（持续时间不足），导致友军突防成功率提升幅度未达预期。建议增加反辐射无人机数量或优化攻击时序，以延长雷达关机窗口。
