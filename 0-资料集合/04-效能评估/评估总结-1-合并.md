# 反辐射无人机作战效能评估方案

## 一、文档概述

本文档为反辐射无人机作战效能评估的完整流程指南，整合了辐射源压制效益、攻击毁伤能力、联合作战效果影响三大评估模块。文档旨在指导仿真系统完成所需数据的采集，并为评估计算提供明确方法。

---

## 二、效能评估总体框架

### 2.1 三大评估模块

| 评估模块         | 核心问题            | 指标数量              |
| :----------- | :-------------- | :---------------- |

## 三、辐射源压制效益评估

### 3.1 评估指标体系

| 指标代号   | 指标名称        | 定义                               | 单位  |
| :----- | :---------- | :------------------------------- | :-- |
| **P1** | 雷达关机覆盖率     | 在攻击窗口内，目标雷达处于关机状态的累计时长占比         | %   |
| **P2** | 区域防空压制覆盖率   | 在指定保护空域内，完全不受敌方雷达有效覆盖的时长占突防窗口的比例 | %   |
| **P3** | 等效战损比 (LER) | 我方损失无人机数量与敌方等效压制雷达数量的比值          | 比值  |

### 3.2 仿真数据采集清单

#### 3.2.1 全局时间参数

| 数据项 | 符号 | 说明 |
| :--- | :--- | :--- |
| 攻击窗口起始时刻 | \( T_{start} \) | 第一架无人机进入敌方雷达截获距离的时刻 |
| 攻击窗口结束时刻 | \( T_{end} \) | 最后一架无人机命中目标/脱离/自毁的时刻 |
| 突防窗口起始时刻 | \( T_{esc\_start} \) | 后续突防编队进入危险空域的时刻 |
| 突防窗口结束时刻 | \( T_{esc\_end} \) | 后续突防编队离开危险空域的时刻 |
| 攻击窗口总时长 | \( T_{attack} = T_{end} - T_{start} \) | |
| 突防窗口总时长 | \( T_{escort} = T_{esc\_end} - T_{esc\_start} \) | |

#### 3.2.2 雷达状态时序数据（每部雷达）

| 记录项 | 格式要求 | 用途 |
| :--- | :--- | :--- |
| 雷达开关机状态时序 | 时间序列（每秒）：1=开机，0=关机 | 计算P1 |
| 雷达物理摧毁时刻 | 事件时间戳 | 计算P3 |
| 雷达有效探测范围 | 多边形坐标（预设参数） | 计算P2 |

#### 3.2.3 无人机损耗与战果数据

| 记录项           | 格式要求              | 用途             |
| :------------ | :---------------- | :------------- |
| 我方无人机总损失数     | 整数 \( N_{lost} \) | 计算P3           |
| 无人机命中目标雷达记录   | (时刻, 目标雷达ID)      | 判定物理摧毁(K-Kill) |
| 雷达持续关机超30分钟记录 | (雷达ID, 关机起止时刻)    | 判定功能压制(F-Kill) |

### 3.3 指标计算方法

#### P1. 雷达关机覆盖率

$$
P1 = \frac{1}{N} \sum_{i=1}^{N} \frac{T_{off\_i}}{T_{attack}} \times 100\%
$$

- \( N \)：参与评估的敌方雷达总数
- \( T_{off\_i} \)：雷达 \( i \) 在攻击窗口内的累计关机时长

**达标阈值**：≥ 50%

#### P2. 区域防空压制覆盖率

$$
P2 = \frac{N_{safe}}{T_{escort}} \times 100\%
$$

- \( N_{safe} \)：突防窗口内保护空域完全未被雷达覆盖的时间步数量

**达标阈值**：≥ 60%

#### P3. 等效战损比 (LER)

$$
P3 = \frac{N_{lost}}{N_{K\_Kill} \times 1.0 + N_{F\_Kill} \times 0.8}
$$

- \( N_{K\_Kill} \)：被物理摧毁的雷达数量
- \( N_{F\_Kill} \)：功能压制的雷达数量（连续关机超30分钟）

**达标阈值**：≤ 1.5

### 3.4 综合评估结论

| 达标情况组合         | 综合结论             | 建议行动                |
| :------------- | :--------------- | :------------------ |
| P1、P2均达标       | **压制有效**         | 方案可行，可投入实战或转入下一阶段推演 |
| 仅P1达标，P2未达标    | **点压制有效，面压制不足**  | 需增加无人机数量或优化航路规划     |
| P1、P2均未达标，P3达标 | **效费比合格但战术效果不佳** | 需重新设计协同策略           |
| 三项均未达标         | **压制失败**         | 方案需全面重新设计           |

### 3.5 仿真案例

**案例背景**：红方使用4架反辐射无人机压制蓝方2部制导雷达，掩护2架F-16突防。

**评估结果示例**：

| 评估指标           | 实测值   | 期望阈值 | 是否达标   |
| :------------- | :---- | :--- | :----- |
| P1 雷达关机覆盖率     | 83.2% | ≥50% | ✅达标    |
| P2 区域防空压制覆盖率   | 98.3% | ≥60% | ✅达标    |
| P3 等效战损比 (LER) | 1.67  | ≤1.5 | ⚠️接近达标 |

## 四、攻击毁伤能力评估

### 4.1 评估指标体系

| 指标代号   | 指标名称                   | 类别         | 回答的核心问题                         |
| :----- | :--------------------- | :--------- | :------------------------------ |
| **A1** | 单发摧毁概率 \( P_k \)       | 最终综合指标（必选） | 一架无人机最终让目标雷达丧失战斗力的概率是多少？        |
| **A2** | 末端制导精度 (CEP)           | 过程诊断指标（必选） | 如果 \( P_k \) 不达标，问题出在打不准还是威力不够？ |
| **A3** | 攻击发起率 \( P_{launch} \) | 前置条件指标（可选） | 有多少无人机根本没机会进入攻击？                |

### 4.2 仿真数据采集清单

| 数据项 | 数据类型 | 用途 |
| :--- | :--- | :--- |
| 无人机ID + 任务结果 | 标称型 | 区分：未发起攻击/脱靶/命中但未毁/成功毁伤 |
| 落点偏差（或命中标志） | 数值型 | 计算CEP和\( P_{hit} \) |
| 目标最终状态 | 标称型 | 判定K/F级毁伤（计算\( P_k \)） |
| 目标是否关机 | 0/1 | 辅助分析对抗效果 |
| （可选）攻击发起失败原因 | 文本型 | 辅助分析\( P_{launch} \)短板 |

### 4.3 指标计算方法

#### A1. 单发摧毁概率 \( P_k \)

**全链路公式**：
$$
P_k = P_{launch} \times P_{hit} \times P_{kill|hit}
$$

**直接统计法**（仿真次数足够时推荐）：
$$
P_k = \frac{N_{\text{目标被K/F级毁伤}}}{N_{\text{总仿真次数}}}
$$

**达标阈值**：≥60%为优秀，40%-60%为合格

#### A2. 末端制导精度 (CEP)

**定义**：包含50%命中弹着点的圆半径

**计算方法**：将落点偏差从小到大排序，取第50%分位值

**达标阈值**：≤5.0m为优秀，5-7m为良好

#### A3. 攻击发起率 \( P_{launch} \)

$$
P_{launch} = \frac{N_{\text{转入末制导}}}{N_{\text{截获目标信号}}}
$$

**达标阈值**：≥95%为优秀，85%-95%为合格

### 4.4 综合评估与诊断

| 短板因素 | 诊断方法 | 改进建议 |
| :--- | :--- | :--- |
| CEP过高 | 若CEP > 战斗部杀伤半径 | 改进导引头精度 |
| 攻击发起率低 | 若大量架次未转入末制导 | 加强记忆跟踪算法或增加人在回路 |
| 命中但未毁 | 若\( P_{hit} \)高但\( P_k \)低 | 增强战斗部威力或改进引战配合 |

**战术建议**：若单发\( P_k \)不足，可采用**双机齐射**（摧毁概率提升至\( 1-(1-P_k)^2 \)）或**三机齐射**。

### 4.5 仿真案例

**案例背景**：200架次蒙特卡洛打靶仿真，评估反辐射无人机对中程制导雷达的毁伤能力。

**评估结果示例**：

| 指标 | 实测值 | 参考阈值/目标值 | 评价 |
| :--- | :--- | :--- | :--- |
| A1 单发摧毁概率 \( P_k \) | 54.5% | ≥60%优秀，40%-60%合格 | **合格** |
| A2 末端制导精度 CEP | 5.4米 | ≤5.0m优秀，5-7m良好 | **良好** |
| A3 攻击发起率 | 89% | ≥95%优秀，85%-95%合格 | **合格偏弱** |

## 五、联合作战效果影响评估

### 5.1 评估指标体系

| 指标代号 | 指标名称 | 类别 | 回答的核心问题 |
| :--- | :--- | :--- | :--- |
| **J1** | 友军突防成功率提升幅度 | 最终效果指标（必选） | 反辐射无人机压制后，友军完成任务的比例实际提高了多少？ |
| **J2** | 敌方防空拦截效能下降率 | 对抗效果指标（必选） | 敌方的防御能力被削弱了多少？ |
| **J3** | 安全走廊连续有效时长 | 过程保障指标（可选） | 为友军创造的无威胁窗口有多长？ |

### 5.2 仿真数据采集清单

| 数据项 | 数据格式 | 用途 |
| :--- | :--- | :--- |
| 友军平台总数 | 整数 \( N_{friend} \) | 计算J1 |
| 友军平台最终状态 | 标称型（成功突防/被击落/中途返航） | 计算J1 |
| 敌方拦截弹发射总数 | 整数 | 计算J2 |
| 敌方拦截弹命中次数 | 整数 | 计算J2 |
| 突防航线雷达覆盖状态时序 | 时间序列（每秒，是/否被雷达覆盖） | 计算J3 |
| 无压制基准数据 | \( P_{success\_base} \)、\( P_{intercept\_base} \) | 作为基准线 |

### 5.3 指标计算方法

#### J1. 友军突防成功率提升幅度

$$
J1 = P_{success\_with} - P_{success\_base}
$$
其中：
- \( P_{success\_with} = \frac{N_{\text{成功突防平台（有压制）}}}{N_{\text{参战平台总数}}} \)

**达标阈值**：≥15%为有效

#### J2. 敌方防空拦截效能下降率

$$
J2 = \frac{P_{intercept\_base} - P_{intercept\_with}}{P_{intercept\_base}} \times 100\%
$$

**达标阈值**：≥50%为显著

#### J3. 安全走廊连续有效时长（可选）

$$
J3 = \max(\text{连续安全时段长度})
$$

**诊断逻辑**：若J3 ≥ 友军通过所需最小时间，但J1仍不达标，说明问题不在压制而在友军自身。

### 5.4 综合评估结论

| 指标 | 达标状态 |
| :--- | :--- |
| J1 成功率提升 | ✅ 显著（≥15%） |
| J2 拦截效能下降 | ✅ 显著（≥50%） |
| J3 走廊时长 | ✅/⚠️ 满足/临界满足需求 |

### 5.5 仿真案例

**案例背景**：6架反辐射无人机压制2部火控雷达，掩护4架F-16突防。

**评估结果示例**：

| 指标 | 实测值 | 参考阈值 | 评价 |
| :--- | :--- | :--- | :--- |
| J1 友军突防成功率提升 | +25% | ≥15%为有效 | ✅效果显著 |
| J2 敌方拦截效能下降 | 50% | ≥50%为显著 | ✅显著削弱 |
| J3 安全走廊连续时长 | 180秒 | ≥180秒（友军需时） | ⚠️临界满足 |

## 七、评估流程总结

### 7.1 从仿真到评估的完整流程

```
步骤1: 仿真初始化
   ├── 设置作战场景参数
   ├── 输入敌方雷达部署位置
   └── 输入我方无人机/友军作战方案

步骤2: 仿真执行与数据记录
   ├── 实时记录雷达开关机状态（每秒）
   ├── 记录无人机飞行轨迹和任务结果
   ├── 记录友军突防过程和最终状态
   └── 记录敌方拦截弹发射与命中情况

步骤3: 仿真结束与数据导出
   ├── 导出原始仿真数据
   └── 格式转换为评估模块可读取格式

步骤4: 指标计算
   ├── 计算P1/P2/P3（压制效益）
   ├── 计算A1/A2/A3（毁伤能力）
   └── 计算J1/J2/J3（联合作战效果）

步骤5: 评估结论输出
   ├── 生成指标达标判定表
   ├── 生成雷达图可视化
   ├── 输出综合评估结论
   └── 提出改进建议
```

## 八、结论与建议

### 8.1 三大模块联动关系

```
攻击毁伤能力(A1/A2/A3)
       ↓
  单发摧毁概率Pk
       ↓
等效战损比(P3) ←→ 辐射源压制效益(P1/P2)
       ↓
友军突防成功率(J1) ←→ 联合作战效果影响(J1/J2/J3)
```
