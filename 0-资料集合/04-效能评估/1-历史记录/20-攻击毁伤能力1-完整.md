好的，我们针对 **攻击毁伤能力** 这一评估大项进行系统性重构。您新增的指标将评估视角从“单发命中概率”扩展到了 **“任务链路全过程”**，覆盖了从攻击发起条件、飞行过程跟踪到最终毁伤等级的完整链条。

以下为您呈现一个逻辑闭环、可量化操作的 **反辐射无人机打击毁伤评估方案（独立模块）**。

## 一、 指标体系重构：三级层次模型

将您给出的 9 项子指标，按照 **攻击输入条件 → 飞行命中过程 → 毁伤输出结果** 的逻辑，划分为三个层级。

| 层级            | 代号  | 指标名称               | 指标性质  | 核心评估问题                 |
| :------------ | :-- | :----------------- | :---- | :--------------------- |
| **L1: 攻击输入层** | I1  | **攻击发起率**          | 条件性概率 | 无人机能否成功进入攻击链路？         |
| **L2: 命中过程层** | I2  | **末制导稳定跟踪数**       | 性能指标  | 导引头能不能“咬住”目标不丢？        |
|               | I3  | **平均脱靶量**          | 精度指标  | 没命中时，偏了多少？（分析系统误差）     |
|               | I4  | **末端制导精度 (CEP)**   | 精度指标  | 命中概率的统计基准线。            |
|               | I5  | **目标命中概率**         | 综合概率  | 考虑可靠性后，**单发弹到底打不打得上**？ |
| **L3: 毁伤输出层** | I6  | **引战配合效率**         | 效率指标  | 打上了，爆炸的时机对不对？          |
|               | I7  | **战斗部威力**          | 物理指标  | 炸了，杀伤范围够不够覆盖目标弱点？      |
|               | I8  | **雷达完全毁伤概率**       | 结果指标  | 针对天线/方舱的**重度摧毁**概率。    |
|               | I9  | **单车/单机摧毁概率 (Pk)** | 综合指标  | **最终结论：一发无人机搞定目标的概率。** |

## 二、 各子项详细定义与计算模型

### 2.1 L1层：攻击输入层

#### I1. 攻击发起率 (\( P_{launch} \))

- **定义**：反辐射无人机在接收到目标信号后，能够成功建立攻击航路、转入末制导俯冲阶段的概率。
- **失败原因**：载机投放失败、数据链中断、目标信号突然消失导致导引头未锁定、初段导航故障。
- **计算公式**：
  \[
  P_{launch} = \frac{N_{\text{转入末制导}}}{N_{\text{接收到目标信号}}} \times 100\%
  \]
- **仿真数据需求**：记录每架次无人机从“发现信号”到“转入俯冲”的状态迁移。

---

### 2.2 L2层：命中过程层

#### I2. 末制导稳定跟踪数 (\( N_{track} \))

- **定义**：在末制导段（通常为俯冲最后10-20秒），导引头 **未丢失目标且跟踪角误差 < 阈值** 的无人机架次数。
- **评估意义**：反映导引头抗目标雷达关机、抗诱饵闪烁的能力。
- **记录方式**：仿真中记录末制导段每一帧的跟踪有效性，若连续丢失超过1秒，则判定为“不稳定”。

#### I3. 平均脱靶量 (\( \bar{d}_{miss} \))

- **定义**：所有 **未直接命中** 或 **命中偏差** 的无人机，其爆炸点相对于天线阵面中心的平均空间距离。
- **单位**：米 (m)。
- **计算公式**：
  \[
  \bar{d}_{miss} = \frac{1}{M} \sum_{j=1}^{M} d_j
  \]
  *其中 \( M \) 为未命中架次数，\( d_j \) 为第 \( j \) 架的脱靶距离。*

#### I4. 末端制导精度 (\( CEP \))

- **定义**：以目标点为圆心，包含 **50%** 命中弹着点的圆半径。
- **计算方法**：在仿真中重复打靶 \( N \) 次，将所有命中点（或爆炸投影点）的偏差排序，取第 \( 50\% \) 位的偏差距离。
- **注**：CEP 和 平均脱靶量 的区别——CEP 是概率统计量（剔除野值），平均脱靶量是均值（受野值影响大）。

#### I5. 目标命中概率 (\( P_{hit} \))

- **定义**：在攻击发起成功的前提下，无人机战斗部能够 **直接撞击雷达天线阵面或有效杀伤半径覆盖阵面** 的概率。
- **计算公式**（基于 CEP 估算）：
  \[
  P_{hit} = 1 - 0.5^{\left(\frac{R_{lethal}}{CEP}\right)^2}
  \]
  *其中 \( R_{lethal} \) 为战斗部对天线阵面的有效杀伤半径。*
- **注**：这是不考虑雷达关机对抗的理想概率，实际命中概率需乘以 **I2 稳定跟踪成功率**。

---

### 2.3 L3层：毁伤输出层

#### I6. 引战配合效率 (\( \eta_{fuze} \))

- **定义**：战斗部实际起爆时机与理论最佳炸高/炸点的匹配程度，直接影响破片覆盖密度。
- **量化方法**：
  - **最佳炸高范围**：针对雷达天线，通常要求炸高 3m-8m（空炸模式）。
  - **配合效率**：
    \[
    \eta_{fuze} = \frac{\text{实际落入最佳炸高区间的次数}}{\text{总命中次数}} \times 100\%
    \]

#### I7. 战斗部威力 (\( P_{damage\_area} \))

- **定义**：在引战配合正常的前提下，战斗部破片场对特定目标（天线阵面、指挥方舱）的 **条件毁伤概率**。
- **仿真评估方法**：
  - 建立目标易损性模型（雷达天线等效为 3mm 铝板 + 波导管）。
  - 使用战斗部静爆数据（破片数、飞散角、比动能）进行 **射线追踪毁伤仿真**。
  - 输出 **毁伤概率矩阵**。

#### I8. 雷达完全毁伤概率 (\( P_{Kill\_hard} \))

- **定义**：目标雷达被击中后，**丧失修复价值（需返厂/报废）** 的概率。
- **判据**：天线阵面穿孔 > 30%、发射机被贯穿、伺服机构卡死。
- **计算公式**：
  \[
  P_{Kill\_hard} = P_{hit} \times \eta_{fuze} \times P_{damage\_area\_hard}
  \]

#### I9. 单车/单机摧毁概率 (\( P_k \))

- **定义**：单架反辐射无人机 **从进入战场到完成任务**，最终导致目标雷达 **功能丧失（含K级和F级）** 的全链路综合概率。
- **全链路串联公式**：
  \[
  P_k = P_{launch} \times P_{hit} \times P_{kill\_given\_hit}
  \]
  *其中 \( P_{kill\_given\_hit} \) 是命中条件下的毁伤概率（结合 I6, I7, I8）。*

## 三、 整合后的“打击评估方案”操作流程

### 3.1 仿真输入数据采集清单

为计算上述 9 项指标，仿真系统需输出的数据字段如下：

| 数据分类 | 具体字段 | 用途 |
| :--- | :--- | :--- |
| **任务日志** | 无人机ID、攻击发起状态（成功/失败原因） | 计算 **I1** |
| **末制导遥测** | 时间戳、斜距、视线角速度、锁定标志位 | 计算 **I2** 稳定跟踪数 |
| **弹道数据** | 命中点坐标 (X, Y, Z)、目标中心坐标 | 计算 **I3** 脱靶量、**I4** CEP |
| **引信数据** | 起爆时刻、炸高 | 计算 **I6** 配合效率 |
| **目标状态** | 受损部件列表（天线、发射机、方舱） | 计算 **I7**、**I8** |

### 3.2 综合计算流程（蒙特卡洛仿真框架）

1. **Step 1：初始化**
   - 设定仿真次数 \( N = 1000 \) 次。
   - 输入目标雷达易损性模型。

2. **Step 2：单次仿真循环**
   - 判定攻击发起：生成随机数，若 < \( P_{launch} \) 则继续。
   - 模拟末制导：根据 CEP 和系统误差生成落点偏差。
   - 判定命中：偏差 < 有效杀伤半径？
   - 判定炸高：引信是否在正确区间工作？
   - 判定毁伤：根据破片命中部位判定损伤等级（完好/轻伤/重伤/报废）。

3. **Step 3：统计输出**
   - 汇总计算 **I1 至 I9**。

## 四、 仿真案例演示（简化数据）

假设对某型反辐射无人机进行 1000 次蒙特卡洛仿真打靶，目标为某型中程制导雷达。

### 4.1 过程数据记录

| 指标代号 | 仿真记录结果 | 数值 |
| :--- | :--- | :--- |
| **I1** | 1000次试验中，成功转入末制导次数 | 920 次 |
| **I2** | 末制导过程中，连续稳定跟踪未丢失次数 | 850 次 |
| **I3** | 所有未命中弹的平均空间偏差 | 8.5 米 |
| **I4** | 命中点 CEP 统计值 | 5.2 米 |
| **I5** | 基于 CEP 和杀伤半径计算的理论命中概率 | 78% |
| **I6** | 命中且炸高在 3-8m 区间的比例 | 82% |
| **I7** | 命中最佳炸高下，对天线阵面的条件毁伤概率 | 90% |
| **I8** | 仿真中导致雷达“完全报废”的次数 | 541 次 |

### 4.2 核心指标计算

#### I1. 攻击发起率
\[
P_{launch} = 920 / 1000 = 92\%
\]

#### I5. 目标命中概率（实际过程统计）
\[
P_{hit\_actual} = \frac{\text{落点在杀伤半径内的次数}}{\text{成功转入末制导次数}} = \frac{650}{920} \approx 70.7\%
\]
*(注：理论值78%与实际70.7%的差异，源于 I2 不稳定跟踪导致的末段脱靶)*

#### I8. 雷达完全毁伤概率（单次命中条件）
\[
P_{Kill\_hard} = (541 / 1000) \times 100\% = 54.1\%
\]

#### I9. 单车/单机摧毁概率 (\( P_k \))
使用全链路串联计算：
\[
P_k = P_{launch} \times P_{hit\_actual} \times P_{kill\_given\_hit}
\]
其中 \( P_{kill\_given\_hit} = \text{I8的命中条件下概率} = 541 / 650 \approx 83.2\% \)

\[
P_k = 0.92 \times 0.707 \times 0.832 \approx \mathbf{54.1\%}
\]
*(注：此处巧合等于 I8 全样本概率，因为 I8 实际是 \( P_k \) 的最终体现。)*

### 4.3 评估结论

| 层级 | 评估维度 | 结论 |
| :--- | :--- | :--- |
| **L1** | 攻击发起 | 92% 发起率良好，但8%的失败率提示需优化中继链路或导引头灵敏度。 |
| **L2** | 命中过程 | 实际命中概率 70.7%，受不稳定跟踪影响（约8%的末段脱靶），建议提升 **I2 稳定跟踪算法**。 |
| **L3** | 毁伤结果 | 单发摧毁概率 **54.1%**。对于低成本无人机而言，这是一个 **中上等** 的毁伤效能。若要达到 90% 的摧毁置信度，需采用 **双发齐射**（\( 1 - (1-0.541)^2 = 79\% \)）或 **三发齐射**（约 90.3%）。 |

## 五、 方案小结

此重构方案将分散的 9 项指标编织成了一个 **“输入-过程-输出”** 的因果链条。

- **核心指标** 依然是 **I9 (Pk)**，它是所有子项的共同归宿。
- **诊断指标** 是 **I2 (稳定跟踪)** 和 **I6 (引战配合)**，当 Pk 不达标时，应首先排查这两项。

您可以将此独立模块的计算结果（特别是 **Pk** 值）作为输入，代入到我们之前讨论的 **辐射源压制效益综合方案** 中，用于精确计算 **战损比 (LER)** 和 **直接效费比**。例如，精确的 \( P_k \) 可以直接修正对敌方雷达 **等效压制数量** 的估算（K-Kill 和 F-Kill 的比例）。
