这是一个非常专业且具有实战意义的技术细节问题。**雷达关机覆盖率**是衡量反辐射无人机在“非直接命中”情况下，通过心理威慑和战术逼迫实现软杀伤效能的核心指标。

以下我将从定义、数学模型、计算步骤，结合一个**具体的连级防空阵地攻防案例**进行详细讲解。

### 一、 核心定义与数学表达

**定义：**
**雷达关机覆盖率（\( C_{off} \)）** 指在反辐射无人机**总攻击任务周期 \( T_{total} \)** 内，敌方目标雷达因感知到威胁而被迫处于 **静默/关机状态的时间总和 \( T_{off} \)** 所占的百分比。

**数学公式：**
$$
C_{off} = \frac{\sum T_{off}}{T_{total}} \times 100\%
$$

**关键点：**
- **\( T_{total} \) 通常定义为：** 从第一架无人机进入敌方雷达探测边界，到最后一架无人机命中目标或自毁为止的**整段压制窗口**。
- **关机状态判定依据：** 敌方雷达停止射频辐射输出，导致其防空导弹无法获取制导数据、预警网出现盲区。

### 二、 案例背景设定

为了让计算过程直观，我们设定一个典型的 **“两机协同压制连级防空雷达”** 场景。

| 参数项 | 数值与设定 | 备注 |
| :--- | :--- | :--- |
| **攻击目标** | 1部 **中程防空导弹制导雷达** | 若该雷达关机，该连队将失去对20km外目标的射击能力。 |
| **我方兵力** | 2架反辐射无人机 (UAV1, UAV2) | 发射时间间隔 5 分钟。 |
| **敌方战术条令** | **“发现来袭反辐射导弹/无人机信号即关机，20秒后若未命中则重启搜索”** | 这是常见的对抗反辐射武器的战术规范。 |
| **单架无人机压制时长** | 从进入敌方雷达**截获距离**到**命中/飞越**的时间 = **120秒**。 | 假设无人机巡航速度 0.7 Ma，雷达截获距离 30km。 |
| **敌方雷达重启耗时** | **15秒** | 从关机状态恢复到正常搜索/跟踪状态的技术时间。 |

### 三、 作战时序推演与关机时长计算

这是计算覆盖率的核心步骤，我们需要画出时序图并计算累计关机时间 \( \sum T_{off} \)。

#### 1. 时序流程分解

| 时间点 | 事件描述 | 雷达状态变化 | 关机持续时间计算 |
| :--- | :--- | :--- | :--- |
| **T+0s** | **UAV1** 被敌方雷达截获，敌方雷达告警响起。 | **开机 \(\rightarrow\) 关机** | 开始计时 |
| **T+20s** | 敌方雷达操作员根据条令尝试重启雷达，搜索空情。 | **关机 \(\rightarrow\) 开机** | 首次关机 **20s** |
| **T+25s** | 雷达重启后仅5秒，再次截获仍在逼近的 **UAV1** 信号。 | **开机 \(\rightarrow\) 关机** | 开始计时 |
| **T+120s** | **UAV1** 命中雷达阵地附近区域或飞越目标。 | 雷达持续关机直到确认撞击声响/信号消失。 | **此段关机 95s** |
| **T+125s** | **UAV1** 攻击结束，敌方尝试重启雷达。 | **关机 \(\rightarrow\) 开机** | |
| **T+130s** | 雷达刚刚正常工作5秒，**UAV2** 抵达截获线（发射间隔5分钟，UAV2晚到300s，但此处我们看UAV1结束后的UAV2到来时间）。*注：此处需精确计算UAV2到达时间。* | | |
| *...中间等待...* | | | |
| **T+300s** | **UAV2** 进入截获距离，雷达告警再次响起。 | **开机 \(\rightarrow\) 关机** | 开始计时 |
| **T+320s** | 敌方尝试重启（20秒原则）。 | **关机 \(\rightarrow\) 开机** | 关机 **20s** |
| **T+325s** | 再次发现 **UAV2** 信号，立即关机。 | **开机 \(\rightarrow\) 关机** | 开始计时 |
| **T+420s** | **UAV2** 命中/飞越目标。 | 关机直到确认撞击。 | 此段关机 **95s** |
| **T+425s** | 攻击结束，雷达恢复工作。 | **关机 \(\rightarrow\) 开机** | |

#### 2. 累计关机时长汇总 (\( \sum T_{off} \))

- **UAV1 造成的关机：** 20s (首次规避) + 95s (持续压制) = **115秒**
- **UAV2 造成的关机：** 20s (首次规避) + 95s (持续压制) = **115秒**
- **总关机时长 \( \sum T_{off} \)：** 115 + 115 = **230秒**

#### 3. 总任务时长确定 (\( T_{total} \))

- **任务开始：** T+0s (UAV1 进入截获线)
- **任务结束：** T+425s (UAV2 命中后雷达恢复)
- **\( T_{total} \)：** 425 秒

### 四、 计算结果与深度解读

**1. 计算结果**
\[
C_{off} = \frac{230}{425} \times 100\% \approx \mathbf{54.1\%}
\]

**2. 效能解读**

- **物理意义：** 在这场持续约7分钟的突袭行动中，该敌方雷达有 **超过一半的时间（54.1%）** 处于致盲状态。
- **联合作战价值：** 这230秒的 **“静默走廊”** 极其宝贵。
    - 在这230秒内，哪怕我方没有隐身战机，普通的非隐身攻击机编队也可以利用这个雷达盲区安全地穿越该连的防空火力杀伤区。
    - 敌方指挥员在这54.1%的时间里完全不知道空域真实情况，导致其 **OODA 循环**（观察-判断-决策-行动）彻底断裂。

### 五、 覆盖率优化的战术启示

通过调整上述模型中的变量，我们可以得出提升 **雷达关机覆盖率** 的几种关键战术手段：

| 变量调整 | 对覆盖率 \( C_{off} \) 的影响 | 实战建议 |
| :--- | :--- | :--- |
| **缩短发射间隔** | **显著提高** | 若 UAV1 和 UAV2 间隔缩短为 1 分钟，敌方雷达刚重启就被下一架压下去，覆盖率可从 54% 提升至 **80%** 以上。 |
| **降低无人机速度** | **提高** | 若无人机飞行速度慢（如采用巡飞弹模式），单架次的压制时间 \( T_{off} \) 延长，覆盖率随之线性增长。 |
| **增加集群规模** | **非线性提高** | 3架无人机采用时间间隔精确协同，甚至可以达到 **接近 100% 的持续压制**（即该雷达在整个战斗过程中全程不敢开机）。 |

**总结：**
雷达关机覆盖率本质上反映的是 **“时间维度上的控制权”**。上述案例中 **54.1%** 的数值看似不高，但对于一个仅需5分钟执行投弹任务的攻击编队来说，这已经是**决定性**的掩护效果。
