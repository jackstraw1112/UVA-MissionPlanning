# 总体流程
1. 设置战斗机打击任务（关联实体装备）
2. 添加反辐射无人机协同任务（2个目标），并关联协同任务
3. 任务分配时，对每个目标分配2架无人机
4. 针对无人机进行部署
5. 设计航路
# 雷达关机覆盖率
1. 仿真
	1. 统计无人机单次任务攻击窗口时长：开始-结束
	2. 设计雷达开关机逻辑任务（探测到反辐射无人机后进行关机-重启）
	3. 统计雷达主动关机时间（默认全时段开机、行为模型设置关机）
2. 保存数据
	1. 单次任务：攻击窗口开始时间-结束时间、目标雷达关机时长（记录多段）

# 区域防空压制覆盖率
1. 仿真
	1. 统计战斗机进入攻击窗口累计时长：开始-结束
	2. 统计战斗机进入攻击窗口安全时长：未被雷达探测时长
2. 保存数据
	1. 单次任务：攻击窗口开始时间-结束时间、未被雷达探测时长（记录多段）

# 核心计算
1. 攻击窗口时长
	1. 开始时间：计算每一架无人机第一次进入攻击窗口时间，统计最短的时间为开始时间
	2. 结束时间：计算每一架无人机存活后的最终时间，或离开攻击区域的最后一次时间
	3. 目标雷达关机时长：统计雷达自主关机时间段，根据攻击窗口时间段进行筛选出最终关机时长
	4. 未被雷达探测时长：记录雷达开机时长以及参数，记录战斗机位置点（能够被探测的位置点）
