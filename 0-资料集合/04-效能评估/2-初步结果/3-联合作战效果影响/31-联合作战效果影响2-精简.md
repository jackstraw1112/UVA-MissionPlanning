好的，我们对联合作战效果影响评估大项进行最大程度精简，保留 **2项核心指标 + 1项可选指标**。精简原则是：**抓住联合作战最本质的两个问题——友军安全了没有？敌方弱化了没有？**

---

## 一、 精简后的评估指标体系

| 指标代号 | 指标名称 | 类别 | 回答的核心问题 |
| :--- | :--- | :--- | :--- |
| **J1** | **友军突防成功率提升幅度** | **最终效果指标（必选）** | 反辐射无人机压制后，友军完成任务的比例实际提高了多少？ |
| **J2** | **敌方防空拦截效能下降率** | **对抗效果指标（必选）** | 敌方的防御能力被削弱了多少？ |
| **J3** | **安全走廊连续有效时长** | **过程保障指标（可选）** | 为友军创造的无威胁窗口有多长？用于诊断J1不达标的原因。 |

**精简逻辑**：
- **J1（友军突防成功率提升）** 是 **最终答案**。无论压制过程多复杂，指挥员最关心的只有一点：友军是不是更容易打进去了？
- **J2（敌方拦截效能下降）** 是 **J1的原因变量**。它直接解释为什么友军成功率提升：因为敌方打不准了。
- **J3（安全走廊时长）** 是 **诊断指标**。如果J1不理想，通过J3可以判断是“走廊开得不够久”还是“走廊有了但友军没用上”。

## 二、 各指标详细定义与计算模型

### J1. 友军突防成功率提升幅度（必选）

**定义**：有反辐射无人机压制时，友军突防编队成功完成任务的平台比例，相对于无压制基准的 **绝对提升量**。

**计算公式**：
\[
J1 = P_{success\_with} - P_{success\_base}
\]
其中：
- \( P_{success\_with} = \frac{N_{\text{成功突防平台（有压制）}}}{N_{\text{参战平台总数}}} \)
- \( P_{success\_base} \)：无压制时历史数据或平行仿真得出的基准成功率。

**单位**：百分点（或小数）。

**物理意义**：
- J1 = +30% 表示压制行动让友军成功率从50%提升到了80%。
- J1 ≤ 0 表示压制未产生正面效果。

---

### J2. 敌方防空拦截效能下降率（必选）

**定义**：有压制时，敌方每次拦截尝试的命中概率，相对于无压制基准的 **相对下降比例**。

**计算公式**：
\[
J2 = \frac{P_{intercept\_base} - P_{intercept\_with}}{P_{intercept\_base}} \times 100\%
\]
其中：
- \( P_{intercept\_with} = \frac{N_{\text{友军被命中次数（有压制）}}}{N_{\text{敌方发射拦截弹总数（有压制）}}} \)
- \( P_{intercept\_base} \)：无压制时敌方拦截成功率。

**特殊处理**：
- 若 \( P_{intercept\_base} = 0 \)，则定义 J2 = 0%（无下降空间）。
- 若 \( P_{intercept\_with} = 0 \)，则 J2 = 100%（完全压制）。

**物理意义**：
- J2 = 60% 表示敌方的拦截效率被削弱了六成，原本10枚导弹能打中3架，现在只能打中1.2架。

---

### J3. 安全走廊连续有效时长（可选）

**定义**：在友军突防窗口内，预定义突防航线上 **连续无雷达覆盖** 的最长时间段的长度。

**计算公式**：
\[
J3 = \max(\text{连续安全时段长度})
\]
单位：秒。

**诊断逻辑**：
- 若 **J3 ≥ 友军通过所需最小时间**（例如编队通过走廊需120秒），但 **J1仍不达标**，说明问题不在压制，而在友军自身（如被其他未压制雷达发现、遭遇战斗机拦截等）。
- 若 **J3 < 最小通过时间**，说明压制窗口太短，需增加无人机数量或优化时序。

## 三、 精简版仿真数据采集清单

为计算上述2+1项指标，仿真系统仅需记录以下 **最小数据集**：

| 数据项 | 数据格式 | 用途 |
| :--- | :--- | :--- |
| 友军平台总数 | 整数 \( N_{friend} \) | 计算J1 |
| 友军平台最终状态 | 标称型（成功突防/被击落/中途返航） | 计算J1 |
| 敌方拦截弹发射总数 | 整数 | 计算J2 |
| 敌方拦截弹命中次数 | 整数 | 计算J2 |
| 突防航线雷达覆盖状态时序 | 时间序列（每秒，是/否被雷达覆盖） | 计算J3 |
| 无压制基准数据 | \( P_{success\_base} \)、\( P_{intercept\_base} \) | 作为基准线 |

*注：无需记录OODA周期、频谱控制详情等，极大简化数据采集负担。*

## 四、 仿真案例演示（精简版）

### 4.1 背景设定

- **任务**：红方4架F-16突防蓝方防空阵地，执行对地攻击。
- **压制方案**：6架反辐射无人机提前压制蓝方2部关键制导雷达。
- **对比基线**（无压制时）：
  - 友军突防成功率：\( P_{success\_base} = 55\% \)（4架中约2.2架成功）。
  - 敌方拦截成功率：\( P_{intercept\_base} = 40\% \)（每10枚导弹命中4架）。
- **突防窗口**：\( T_{escort} = 240 \)秒。

### 4.2 仿真结果记录（有压制时）

| 统计项 | 数值 |
| :--- | :--- |
| 友军参战平台总数 | 4 架 |
| 友军成功突防数量 | 3 架 |
| 友军被击落数量 | 1 架 |
| 敌方发射拦截弹总数 | 8 枚 |
| 敌方命中友军次数 | 1 次 |
| 突防航线连续无雷达覆盖的最长时段 | 165 秒 |

### 4.3 指标计算

#### J1. 友军突防成功率提升幅度

\[
P_{success\_with} = \frac{3}{4} = 75\%
\]
\[
J1 = 75\% - 55\% = \mathbf{+20\%}
\]

**解读**：压制行动使友军成功率提高了20个百分点。

#### J2. 敌方防空拦截效能下降率

\[
P_{intercept\_with} = \frac{1}{8} = 12.5\%
\]
\[
J2 = \frac{40\% - 12.5\%}{40\%} = \frac{27.5\%}{40\%} = \mathbf{68.75\%}
\]

**解读**：敌方拦截效率被削弱了近七成。

#### J3. 安全走廊连续有效时长（可选）

\[
J3 = 165 \text{秒}
\]
**解读**：走廊最长连续安全窗口为165秒。F-16编队通过该区域约需120秒，因此窗口时长 **足够**。

### 4.4 综合评估结论

| 指标 | 实测值 | 参考阈值 | 评价 |
| :--- | :--- | :--- | :--- |
| **J1 友军突防成功率提升** | **+20%** | ≥15% 为有效 | ✅ **有效** |
| **J2 敌方拦截效能下降** | **68.8%** | ≥50% 为显著 | ✅ **显著** |
| **J3 安全走廊连续时长** | **165秒** | ≥120秒（友军需时） | ✅ **充足** |

**最终结论**：
> **压制行动对联合作战贡献显著。**
>
> 6架反辐射无人机成功将敌方防空拦截效率削弱68.8%，直接促成友军突防成功率从55%跃升至75%。安全走廊连续时长165秒，为友军提供了充裕的突防窗口。该方案可作为联合作战的标准压制预案。

## 五、 精简前后对比

| 对比项 | 原方案（6项指标） | 精简方案（2+1项） |
| :--- | :--- | :--- |
| **指标数量** | 6项 | 2项核心 + 1项可选诊断 |
| **数据采集** | 需记录OODA周期、频谱状态、走廊宽度等 | **仅需友军状态、拦截事件、航线覆盖时序** |
| **计算复杂度** | 需加权综合或多项达标判定 | **Excel 三行公式完成** |
| **结论清晰度** | 需专家解读多维度结果 | **成功率+拦截下降率，一目了然** |
| **适用场景** | 深度评估、装备论证 | **快速推演、方案比选、指挥员简报** |

此精简方案可与之前讨论的 **攻击毁伤能力（A1/A2/A3）** 和 **辐射源压制效益（P1/P2/P3）** 并列，构成反辐射无人机作战效能评估的三大核心模块，形成完整的轻量化评估框架。
