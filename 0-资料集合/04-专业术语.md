# 决断高度
在反辐射无人机的语境下，“决断高度”通常指的是飞机在着陆前的一个关键决策点。但考虑到反辐射无人机是用于攻击敌方雷达的“自杀式”武器，它本身并不需要降落。因此，你提到的这个词，更可能是一种形象化的比喻，来描述其攻击逻辑中的某个关键阶段。

虽然没有直接资料定义“反辐射无人机”的“决断高度”，但我们可以结合通用航空术语的定义和反辐射无人机的工作流程来理解这个比喻：

### 1. 通用航空中的“决断高度” 
在民航和通用航空领域，**决断高度（Decision Height, DH）** 指的是飞机在最后进近着陆阶段的一个特定高度。当飞机下降到这个高度时，飞行员必须做出一个关键决定：
*   如果飞行员能够清楚地看到跑道，具备着陆条件，就继续降落。
*   如果不能建立必需的目视参考，就必须立刻执行复飞，终止降落。

这是一个确保飞行安全的“**生死抉择点**”。

### 2. “决断高度”在反辐射无人机语境下的可能含义
将这个概念应用到反辐射无人机的作战流程中，它很可能比喻无人机攻击逻辑中的**不可逆点**或**末制导启动点**。反辐射无人机的工作流程通常如下：
1.  发射升空，飞向目标区域。
2.  到达目标区后，在目标上空巡逻待机，被动搜索敌方雷达信号。
3.  **关键节点**：当无人机截获并锁定雷达信号后，会转入俯冲攻击。
4.  **“决断高度”的比喻义**：这个术语很可能就是指无人机在俯冲攻击过程中，到达某个预设的高度。此时，机载计算机必须“决断”：
    *   如果导引头持续稳定地跟踪着雷达信号，则确认攻击，继续俯冲直至摧毁目标。
    *   如果在这个高度，信号突然丢失（例如敌方雷达紧急关机），无人机可能需要“决断”是立即引爆（靠惯性继续攻击），还是拉起机头、重新爬升到巡逻高度，等待敌方雷达再次开机。

因此，这里的“决断高度”可以被理解为：**无人机在俯冲攻击的最后阶段，用于判断是“继续攻击”还是“终止攻击并复飞（重新搜索）”的高度临界点。**
