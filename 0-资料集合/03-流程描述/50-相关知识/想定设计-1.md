# 反辐射无人机JWS01打击目标雷达战术想定

## 「利刃穿网」— 复杂电磁环境下反辐射无人机突防与雷达对抗推演

---

## 第一章 想定总则

### 1.1 想定目的
- 推演JWS01反辐射无人机在复杂电磁环境下的突防效能
- 检验目标雷达防空体系对抗反辐射无人机的多层战术手段
- 探索"无人机集群饱和打击"与"雷达组网协同闪避"之间的攻防博弈极限
- 为防空雷达作战条令修订提供参考依据

### 1.2 对抗双方

| 要素 | 红方（攻击方） | 蓝方（防御方） |
|:----:|---------------|---------------|
| **代号** | "利刃"特遣队 | "天盾"防空群 |
| **核心装备** | JWS01反辐射无人机 ×12 | AN/FPS-117、HR-2000S、AN/MPQ-65等 |
| **作战目标** | 压制/摧毁蓝方核心预警与火控雷达，为后续突击编队打开通道 | 保持防空雷达网完整性，拦截全部反辐射无人机 |
| **指挥理念** | 饱和突防、协同诱骗、记忆攻击 | 闪避静默、诱饵欺骗、硬杀伤拦截 |

### 1.3 战场环境设定

## 第二章 装备技术参数详表

### 2.1 红方主战装备：JWS01反辐射无人机

| 参数类别 | 详细规格 | 战术意义 |

| **导引头灵敏度** | -70 dBm（典型值） | 对AN/FPS-117类大功率雷达截获距离约50-60km；对HR-2000S约35-45km |
| **测角精度** | 方位≤1.5°（RMS） | 终端精度足以直接撞击天线阵面 |
| **抗关机能力** | GPS/INS组合导航 + 记忆航迹点 + 被动重新搜索 | 雷达关机后按最后方位飞往记忆点，到达后盘旋搜索，搜索窗口半径约5km，搜索时间≥20分钟 |
| **重捕逻辑** | 雷达重新开机且信号强度超过门限→2秒内重新锁定 | 对蓝方"闪避照射"战术构成核心威胁 |
| **单机成本** | 约30-50万美元（公开估算） | 远低于爱国者导弹（约400万美元/枚），具备成本不对称优势 |
| **数据链** | UHF双向数据链，作用距离200km（对空/对海） | 支持地面站远程更改航路点、目标优先级、自毁指令 |

## 第三章 作战时间线推演

### 总时间基准
- **T-60min**：红方发射前准备阶段
- **T0**：JWS01集群发射
- **T+45min**：集群抵达目标区域外围
- **T+45min ~ T+150min**：攻防博弈核心阶段（巡飞搜索与对抗）
- **T+150min**：集群燃油耗尽或全部消耗

---

### 阶段一：发射与突防（T-60min ~ T+45min）

#### 红方行动

**T-60min**
- 2辆JWS01发射车从前沿隐蔽阵地展开（位置：大陆海岸线后方5km密林中）
- 地面控制站完成12架JWS01任务装订：
  - **第1波（4架）**：目标——大汉山FPS-117，编号R01~R04
  - **第2波（4架）**：目标——乐山FPS-117，编号R05~R08
  - **第3波（4架）**：目标——大岗山FPS-117，编号R09~R12
- 任务策略：**饱和攻击 + 时差到达**
  - 第1波设定高度2000m，直飞航路
  - 第2波设定高度500m（超低空，利用海面杂波遮蔽），延迟8分钟发射
  - 第3波设定高度3500m（高空），延迟15分钟发射
- 电子战飞机（远距200km外）开始对蓝方预警机数据链实施压制干扰

**T-45min**
- 第1波R01~R04发射
- 发射后爬升至2000m巡航高度，转入GPS/INS自主导航
- 导引头此时**处于静默状态**（不开启被动接收），避免过早暴露
- 巡航速度200km/h，向海峡中线飞行

**T-37min**
- 第2波R05~R08发射（500m超低空）
- 超低空飞行增加被蓝方常规对空监视雷达发现的概率极低，但海浪可能影响GPS精度

**T-30min**
- 第3波R09~R12发射（3500m高空）

**T-15min**
- 第1波抵达海峡中线附近
- R01~R04导引头**开机**（被动接收模式），开始搜索FPS-117的L波段信号
- 此时距离大汉山约90km，处于JWS01截获距离（55km）之外，暂时未截获

#### 蓝方反应

**T-50min**
- 蓝方电子侦察船在海峡中线探测到红方电子战飞机加电信号
- COC判定威胁等级提升，转入**第二级战备（紧张升级期）**
- 下令：全部FPS-117切入 **M2（LPI低截获模式）**——降低发射功率至30%，开启频率捷变
- 预警机E-2K升空，开启 **M5（UHF反隐身探测）** 和 **M6（ESM被动协同）**

**T-30min**
- E-2K的ALR-73被动侦测系统**未发现**JWS01导引头信号（因导引头尚未开机）
- 但电子侦察船报告：在海峡中线上空探测到微弱UHF数据链上行信号（JWS01地面站遥控指令）
- COC判断：可能有小微型无人机/巡飞弹集群正在接近
- 下令：HR-2000S（清泉岗、嘉义）切入 **M7（LPI隐蔽开机）**，低功率监视低空

**T-15min**
- 清泉岗HR-2000S在低功率模式下的探测距离大幅缩短（从240km降至约80km）
- **仍未发现**500m高度飞行的第2波R05~R08
- 对3500m高度飞行的第3波R09~R12，理论上可探测到，但因其RCS极小（0.01-0.05m²），且HR-2000S当前为低功率模式，实际发现概率<20%
- COC继续保持静默策略，等待更多情报

> **阶段一评估**：红方利用导引头延迟开机、多高度层突防、超低空海面遮蔽，成功实现战术突防。蓝方依靠电子侦察获取了模糊预警，但未能获得精确跟踪。

### 阶段二：捕获与初始攻击（T+45min ~ T+70min）

#### 红方行动

**T+45min**
- 第1波R01~R04抵达距大汉山约55km处
- JWS01导引头成功截获大汉山FPS-117的L波段信号
  - 但此时FPS-117处于M2模式（功率降至30%+频率捷变），截获距离从理论55km缩短至约**40km**
  - R01在42km处率先截获，R02在38km截获，R03/R04在35km截获
- 4架JWS01同时转入**主动寻的**，向信号源俯冲
- 速度增至250km/h（俯冲增速），预计7-8分钟后抵达目标

**T+47min**
- 蓝方大汉山FPS-117的RWR（雷达告警接收机）**告警**：探测到4个被动辐射源正在逼近，方位270°-285°，距离约40km，判断为反辐射无人机
- FPS-117立即执行**紧急关机**
- JWS01导引头信号丢失

**T+47min ~ T+52min（关键博弈窗口）**

**红方JWS01抗关机逻辑启动：**
1. 导引头信号丢失瞬间，记录最后方位角（275°）、信号强度、距离估算（38km）
2. 切换至GPS/INS模式，按最后方位角继续飞行
3. 飞抵记忆点（大汉山阵地坐标预装订，误差≤100m），到达后**进入盘旋搜索模式**
4. 盘旋半径3km，搜索时间可持续20分钟以上

**蓝方对抗决策（大汉山阵地指挥官面临选择）：**

```
决策树：
├── 选项A：持续关机等待JWS01燃油耗尽（~2小时后）
│   ├── 优点：JWS01找不到信号，无法攻击
│   ├── 缺点：大汉山雷达退出防空网2小时，北部预警盲区巨大
│   └── 风险：红方可能派出第2批无人机轮换
│
├── 选项B：启动有源诱饵TLQ-32，同时本站关机
│   ├── 优点：诱饵模拟本站信号，可能引走部分JWS01
│   ├── 缺点：JWS01导引头可能识别诱饵与真雷达的细微差异
│   └── 风险：如果诱饵被识破，JWS01仍在盘旋，且本站仍无法开机
│
├── 选项C：间歇闪避照射（每10秒开机2秒）
│   ├── 优点：保持有限空情能力，同时测试JWS01反应速度
│   ├── 缺点：2秒窗口可能被JWS01捕捉（其重捕时间仅2秒）
│   └── 风险：极高——相当于给JWS01"喂目标"
│
└── 选项D：协同闪避——大汉山关机，命令乐山FPS-117调整波束覆盖北部
    ├── 优点：保持北部基本预警能力
    ├── 缺点：乐山覆盖北部时存在盲区，且乐山自身也暴露
    └── 风险：如果红方第2波同时攻击乐山，则两个节点同时丢失
```

**蓝方选择：选项B + 选项D组合**
1. 大汉山FPS-117关机
2. 启动AN/TLQ-32有源诱饵，设置在大汉山阵地西侧2km处，模拟FPS-117的L波段信号特征，但功率故意调高20%（试图以更强信号吸引JWS01）
3. 同时命令乐山FPS-117将波束北偏，部分覆盖大汉山原责任区

**T+50min**
- R01（最先到达记忆点）进入盘旋搜索
- **诱饵TLQ-32开机发射**
- R01导引头截获诱饵信号！但JWS01内置的**辐射源指纹识别算法**开始工作：
  - 分析脉冲重复间隔（PRI）稳定性：真FPS-117的PRI抖动特征与TLQ-32模拟的略有不同
  - 分析脉冲内调制特征（脉内指纹）：TLQ-32的发射管与FPS-117不同，脉冲包络存在微秒级差异
  - 分析天线扫描调制：真雷达有360°旋转的天线方向图调制（AM），而TLQ-32是全向辐射，无此调制
- **判定时间**：约8-12秒
- R01判定为诱饵概率70%，**不攻击诱饵**，继续在记忆点盘旋等待真雷达信号
- R02、R03被诱饵强信号吸引（较旧版本软件），转向诱饵方向
- **R02撞击TLQ-32诱饵**——诱饵被摧毁，战斗部爆炸，但主阵地无损
- R03在接近诱饵时真雷达信号特征库匹配失败，中途放弃，返回盘旋
- R04在盘旋中持续搜索

**T+55min**
- 第2波R05~R08抵达大汉山附近（超低空突防成功）
- 此时大汉山FPS-117仍处于关机状态
- R05~R08进入盘旋搜索模式，与大汉山上空形成**8架JWS01盘旋网**
- 蓝方大汉山阵地指挥官面临更大压力：关机时间越长，盘旋的无人机越多

> **阶段二评估**：红方第1波4架中1架被诱饵消耗，1架被诱骗（未命中主目标），2架仍在盘旋搜索。蓝方诱饵战术取得部分成功，但未能清除威胁，且丧失了大汉山预警能力。

### 阶段三：协同闪避与硬杀伤拦截（T+70min ~ T+120min）

#### 蓝方启动体系化对抗

**T+70min — 蓝方COC做出重大决策：启动"轮替闪避"协议**

核心思路：不让任何一部雷达在同一位置持续开机超过15秒，利用JWS01的反应延迟（截获→锁定→俯冲需约10-15秒）形成"时间差生存"。

**具体执行——大汉山区域：**
1. 大汉山FPS-117保持关机
2. 部署在大汉山附近（3km外备用阵地）的**机动HR-2000S**执行"闪照"任务：
   - 开机15秒（快速360°一扫，获取一轮空情数据）→ 关机
   - 等待45秒
   - 再次开机15秒 → 关机
   - 周期60秒
3. 开机时使用 **M7（LPI模式，功率20%）**，缩短JWS01截获距离至约25km
4. 关机期间通过数据链将上一轮空情数据发送给COC和其他节点

**T+72min**
- 机动HR-2000S第一次闪照开机（15秒）
- 此时盘旋中的R01距离该机动雷达约8km（处于截获距离25km内）
- R01**截获信号**
- 但HR-2000S在JWS01完成锁定+俯冲机动之前（需约12秒）**已关机**
- R01导引头信号丢失，转为记忆飞行，飞向机动雷达记忆点
- 60秒后HR-2000S第二次闪照，但位置已不同（阵地微调），R01的记忆点偏移

**T+75min — 蓝方启动硬杀伤拦截**

COC下令：清水阵地PAC-3对JWS01实施拦截

### 4.3 战术评估

**红方优势：**
1. 成本不对称极度有利
2. 抗关机巡飞能力对传统"关机规避"形成降维打击
3. 多高度层突防+导引头延迟开机实现了有效突防
4. "数据链协同攻击"突破了闪避照射的时间窗口

**红方劣势：**
1. 对诱饵的识别率不够高（尤其面对高保真升空诱饵时）
2. 对移动目标（E-2K）无能为力
3. 单机战斗部威力有限（20kg），对有掩体保护的雷达效果差
4. 数据链易被干扰（若蓝方对UHF数据链实施有效压制，协同攻击即失效）

**蓝方优势：**
1. 有源诱饵+升空诱饵组合有效消耗了25%的攻击架次
2. "轮替闪避"在红方切换协同战术前效果显著
3. PAC-3硬杀伤拦截成功率高达100%（锁定即击落）
4. 核心固定站雷达（FPS-117）凭借掩体保护，即使被近失弹也未丧失功能

**蓝方劣势：**
1. 硬杀伤拦截的经济成本不可持续
2. 全面静默意味着防空网自我瘫痪
3. 闪避照射的15秒窗口在协同攻击面前不足
4. 缺乏对JWS01导引头数据链的有效干扰手段
