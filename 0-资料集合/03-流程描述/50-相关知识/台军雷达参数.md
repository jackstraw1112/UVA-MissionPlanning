基于台湾地区31部雷达的频段特性、探测距离和战术定位，我将其按**13个核心型号**进行归类。同一型号在不同阵地的硬件一致，但根据具体面向的威胁方向（如面向大陆、面向太平洋、面向巴士海峡），其10组工作参数的侧重点会有所不同。

以下参数设计基于雷达物理原理（频段-PRF-脉宽的物理约束）和现代防空战术标准推演。

## 第二部分：联合防空雷达运行规则流程（SOP）

基于"战备等级"和"威胁态势"建立以下四级运行流程：

### 第一级：日常和平时期（DEFCON 5 / 战备等级4）
**核心原则：建立基线，保持训练，设备保养**
1. **开机比例**：固定站（FPS-117）保持2/3开机（如大汉山开、乐山热备、大岗山检修轮换）；机动雷达封存或低频训练。
2. **参数配置**：全部使用 M1（常规广域搜索），固定载频，不开启频率捷变和极化切换（减少发射机损耗，降低对民用航空的干扰）。
3. **数据流**：各站独立生成空情图，上报联合作战中心（COC），不进行跨战区组网融合。
4. **预警机**：E-2K保持每周2-3次例行巡逻，使用M1/M2模式，不开启隐身探测（M5）等高耗能模式。

### 第二级：紧张升级期（DEFCON 4-3 / 战备等级3）
**核心原则：扩大探测，隐真示假，组网融合**
1. **开机比例**：所有固定站100%开机；机动雷达（TPS-117、HR-2000S）推至预设阵地展开；天弓雷达（MPN-45）由搜索状态转入跟踪待命。
2. **参数配置**：
   - FPS-117 切换至 **M2 (LPI模式)** 或 **M9 (分区时序)**，降低被电子侦察卫星/飞机截获的概率。
   - 机动站TPS-117开启 **M8 (战术欺骗辐射)**，在非真实阵地发射模拟信号，制造虚假雷达网拓扑。
   - E-2K 全天候巡逻，开启 **M5 (隐身目标探测)** 和 **M6 (ESM被动协同)**。
3. **网络规则**：Link-16数据链全通。建立"一主二备"规则：某站点若信号丢失，相邻站点自动在3秒内切换至接替模式（如花莲TPS-117切入M2）。
4. **超视距雷达**：FPS-120由试验状态转入 **M1 (标准探测)**，24小时向COC推送大陆沿海纵深的兵力调动数据。

### 第三级：高威胁/冲突爆发（DEFCON 2 / 战备等级2）
**核心原则：抗干扰生存，多源融合，火力交接**
1. **参数配置（全面转入ECCM模式）**：
   - 所有L/S波段雷达强制开启 **M4/M5 (抗干扰/ARM闪避)**，开启超宽带频率捷变和自适应极化。
   - 相控阵雷达（HR-2000S、MPN-45、SPS-48E）启用 **M5 (自适应零陷)**，自动将波束零点对准已知的干扰机方向。
   - 爱国者雷达（MPQ-53/65）切入 **M4 (ARM协同闪避)**，阵地周边的AN/TLQ-32诱饵天线同步开启。
2. **隐身目标拦截流程（核心战术）**：
   - **发现**：FPS-120(超视距) 或 解放军侧泄露信号 → 提示E-2K。
   - **确认**：E-2K 切入 **M5 (UHF反隐身)**，利用长波长锁定模糊方位。
   - **精测**：COC指令最近的HR-2000S切入 **M6 (反隐身加强)**，在指定扇区增加能量积聚。
   - **拦截**：交接给MPQ-65（爱国者）切入 **M1 (搜索截获)**，最终锁定发射PAC-3。
3. **静默交替规则（防ARM饱和打击）**：
   - 同一战区的两部雷达（如大汉山FPS-117与三芝MPN-45）不同时处于全功率状态。
   - A站全功率扫描10秒 → A站转入静默 → B站全功率接管 → 循环。让反辐射导弹失去连续跟踪基准。

### 第四级：全面交战与战损管理（DEFCON 1 / 战备等级1）
**核心原则：极限生存，降级作战，网络化协同交战(CEC)**
1. **硬杀伤生存**：若卫星/无人机确认某固定阵地（如乐山）即将被打击，该站立即执行 **M5 (ARM闪避)** 后物理断电，人员撤入山洞。
2. **机动接替流程**：
   - 机动分队在接收到"阵地丧失"指令后5分钟内，TPS-117切入 **M1 (快速展开)**，在30分钟内重新建立战区覆盖。
3. **网络化协同交战（CEC - 最极端模式）**：
   - 爱国者MPQ-65全面切入 **M10 (全主动模式)**，本机雷达**完全关机不发射**。
   - 前沿的E-2K或生存下来的陆基雷达通过高速数据链，将高精度的目标轨迹直接装订到PAC-3导弹中。
   - PAC-3发射后冷启动，在飞行中段靠数据链修正，直到末端Ka波段导引头开机自行锁定。**实现"看别人的雷达，打自己的导弹"，彻底消除火力阵地电磁暴露。**
4. **极限降级运行**：若所有S/C波段雷达被摧毁或压制，退守最后底线——利用FPS-120（天波超视距）提供上千公里外的模糊方位警告，配合残存的UHF预警机，引导战斗机进行视距内目视拦截。
