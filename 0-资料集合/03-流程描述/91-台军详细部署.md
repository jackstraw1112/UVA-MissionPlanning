### 二、台军雷达预警体系部署详情

台军雷达网是反辐射作战的**核心目标**，其架构分为固定式远程预警雷达与机动式野战雷达两类：

#### （一）固定式远程预警雷达（高价值/高难度目标）

1. **乐山远程预警雷达**
   - **部署位置**：新竹乐山（海拔2620米）
   - **系统型号**：美制AN/FPS-115“铺路爪”系统
   - **核心参数**：探测距离超5000公里，可追踪弹道导弹，是台军“导弹防御体系的眼睛”
   - **防护特点**：部署于加固掩体内，周边配属爱国者防空阵地及近程防御系统
   - **JSW-01攻击评估**：**需多波次/多方向饱和攻击**，单一无人机难以突破

2. **小雪山雷达站**
   - **部署位置**：台中雪山西稜（海拔3020米，全台最高）
   - **隶属单位**：海军监侦指挥部
   - **任务功能**：俯瞰台湾海峡近200公里范围，监控海峡船舶及空中动态
   - **防护特点**：高山部署，机动支援困难，但雷达信号特征明显

3. **长白相控阵雷达**（天弓系统配套）
   - **部署数量**：至少7座固定雷达站
   - **核心参数**：S波段（2-4GHz），对1㎡目标探测距离约400公里，覆盖120°扇区
   - **部署规律**：每座天弓阵地均配置一部，多位于西海岸平原及丘陵地带
   - **战术弱点**：**固定部署、天线阵面朝向固定**，为JSW-01理想打击目标

#### （二）机动式雷达（高机动/反辐射对抗目标）

1. **新式机动雷达**（规划采购中）
   - 台军已明确规划采购新型机动雷达，旨在“强化侦搜能力及战场存活率”
   - **战术意义**：未来台军将更依赖“打了就跑”的机动雷达战术

2. **CS/MPG-25照射雷达**（天弓一/二型配套）
   - 部署位置随天弓阵地机动部署
   - 负责导弹末端照射，**开机即暴露、关机即失去拦截能力**

### 三、天弓系列防空阵地详细部署分析

天弓系统是台军“台湾之盾”的**核心支柱**，也是JSW-01反辐射无人机的主要打击目标。

#### （一）天弓三型（TK-Ⅲ）阵地

| **项目**     | **详细信息**                              |
| :--------- | :------------------------------------ |
| **已知阵地数量** | 至少2个（截至2024年）                         |
| **推测部署位置** | 北部（大台北防御圈）、中部（台中将军事基地周边）              |
| **配套雷达**   | 常山相控阵雷达（4-8GHz，J波段），具备电子反对抗能力         |
| **发射装置**   | 4联装储运发射箱，垂直发射，机动部署                    |
| **导弹参数**   | 弹长5.5米，弹径0.4米，发射重880kg，速度7马赫，射程200公里  |
| **制导方式**   | 中继惯导+雷达数据链上行修正，末端Ku波段（12-18GHz）主动雷达导引 |
| **对抗要点**   | 雷达采用**主动相控阵体制**，抗干扰能力强，需电子战飞机协同压制     |

#### （二）天弓一/二型阵地

| **项目** | **详细信息** |
| :--- | :--- |
| **阵地总数** | 6个（台湾本岛4个，澎湖1个，马祖1个） |
| **部署位置推测** | 本岛：北部（台北港/淡水）、中部（台中港/清泉岗）、南部（高雄/屏东）、东部（花莲）；外岛：澎湖、马祖 |
| **配套雷达** | 长白相控阵雷达（固定式）+ CS/MPG-25照射雷达 |
| **发射方式** | 地下井固定发射 + 拖车式机动发射混合部署 |
| **天弓二型参数** | 射程150公里，速度4.5马赫，弹道末端主动雷达导引（28-32GHz，Ka波段） |
| **对抗要点** | 固定雷达站**坐标明确**，是反辐射作战的**优先打击目标**；需注意部分采用地下化部署 |

#### （三）天弓四型（强弓）系统

- **状态**：中科院自主研发，预量产阶段，属“台湾之盾”中高层反导核心
- **核心参数**：拦截高度70公里（可拦截大气层外目标），探测距离超500公里
- **雷达特性**：主动式相列雷达（AESA），抗干扰能力强，波束灵活
- **发射方式**：360°垂直发射，4枚箱组，机动部署
- **威胁评估**：对JSW-01等无人机目标具备高空拦截能力，但**雷达信号特征显著**，可纳入高价值目标清单

### 五、JSW-01反辐射无人机作战运用建议

针对上述目标特征，建议在训练场景中设定以下战术科目：

1. **针对固定雷达站的“坐标打击”模式**
   - 长白雷达等固定目标坐标已知，JSW-01可采用预编程惯性导航+末端被动导引
   - 重点演练：**雷达关机后的再搜索与攻击**（模拟台军“快开快关”战术）

2. **针对机动雷达的“广域游猎”模式**
   - 模拟JSW-01在目标区域上空巡逻滞空（可达4小时以上），一旦探测到雷达辐射信号即俯冲攻击
   - 重点演练：与**歼-16D电子干扰机**协同，诱使敌方雷达开机

3. **针对高山雷达站的“地形规避突防”模式**
   - 小雪山、乐山雷达站位于高海拔山区，需规划低空地形跟随航线
   - 重点演练：利用山体遮挡规避台军低空雷达探测

4. **饱和攻击协同模式**
   - 乐山雷达站防护严密，需**多波次、多方向同时攻击**
   - 重点演练：JSW-01与**攻击-11隐形无人机**、**歼-20**的协同突防

### 六、训练评估要点

| **评估指标** | **目标值** | **说明** |
| :--- | :--- | :--- |
| 杀伤链闭合时间 | ≤10秒 | 从探测到雷达信号到命中，针对台军“快开快关”战术 |
| 多目标同时打击能力 | ≥4个/批次 | 同时压制多个雷达站 |
| 抗关机能力 | 命中率≥70% | 雷达关机后仍能依靠记忆坐标或游猎模式命中 |
| 与电子战飞机协同度 | 压制成功率≥85% | 歼-16D掩护下突破台军防空 |
