# 「碎盾」联合反辐射作战想定

## JWS01反辐射无人机与多空中平台协同攻防体系推演

---

## 第一章 想定概述

### 1.1 作战背景
红方决定对蓝方实施联合火力打击，首要任务是**瘫痪蓝方北部及中部防空预警体系**，为后续轰炸机编队和两栖登陆舰队开辟安全走廊。蓝方在台北-新竹-南投一线部署了由AN/FPS-117（远程预警）、HR-2000S（三维引导）、AN/MPQ-65（爱国者火控）构成的严密防空雷达网。

### 1.2 核心作战概念：SEAD/DEAD联合体
传统反辐射导弹（如Kh-31P、YJ-91）为"一发入魂"但耗资巨大且载机风险高。本想定采用**"分布式协同SEAD"**理念：以JWS01反辐射无人机群为"持久压制力量"，辅以电子战飞机、隐身战斗机、预警机、光学巡飞弹，形成**"侦-控-扰-打-评"**闭环。

### 1.3 红方参战空中平台编成

| 编队代号 | 平台类型 | 数量 | 任务定位 | 与JWS01的协同关系 |
|:-------:|---------|:----:|---------|-----------------|
| **A1** | 空警-500预警机 | 1架 | 战场统帅，态势感知，空域管制 | 为JWS01提供目标坐标和航路规划 |
| **E1** | 歼-16D电子战飞机 | 4架 | 远距支援干扰，致盲诱骗 | 迫使蓝方雷达暴露或致盲，掩护JWS01突防 |
| **S1** | 歼-20隐身战斗机 | 4架 | 定点清除，制空护航 | 猎杀蓝方拦截机，保护JWS01不被战斗机击落 |
| **U1** | JWS01反辐射无人机 | 24架 | 持久雷达压制，记忆攻击 | 核心打击力量 |

## 第二章 空间-时间协同战术设计（四大核心协同模式）

### 协同模式一：「闪屏诱猎」—— 歼-16D与JWS01的电磁配合

**战术意图**：利用蓝方雷达"抗干扰时必须提高功率"的物理规律，人为制造电磁暴露窗口。

```
时间轴：T+00:00 ~ T+02:30

蓝方FPS-117(大汉山)                 红方作战序列
     │
     │ (处于LPI低截获模式，功率30%)
     │
     ├─────[侦] 空警-500被动截获微弱信号，锁定方位
     │
     ▼
[干扰开始] 歼-16D开启大功率杂波/欺骗干扰(瞄准FPS-117频段)
     │
     ├──▶ 蓝方雷达屏幕被雪花/假目标淹没
     │
     ├──▶ 蓝方操作员判断：必须提高功率+窄带滤波对抗干扰
     │      (这是物理规律决定的，不提高功率就看不见真目标)
     │
     ├──▶ FPS-117功率从30%拉升至100%，关闭频率捷变(锁定频点)
     │      ★ 暴露窗口开启 ★
     │
     ├──[传] 空警-500侦测到功率跃升，立即通过数据链通知JWS01集群
     │      "目标Alpha，方位275°，功率满格，建议立即寻的"
     │
     ├──[打] JWS01集群(8架)导引头同时截获强信号
     │
     ├──[断] 歼-16D在JWS01截获后第3秒 瞬间停止干扰 ★关键配合★
     │      (消除干扰对JWS01导引头的杂波影响)
     │
     └──[终] JWS01在纯净电磁环境下锁定真实天线位置，俯冲命中
```

**关键参数约束**：
- 歼-16D干扰持续时间：持续至空警-500确认JWS01截获信号为止
- 歼-16D关停延迟：≤3秒（防止JWS01导引头被旁瓣干扰误导）
- 蓝方反应时间：操作员判断并拉高功率约需15-30秒，JWS01从截获到命中约8分钟，时间充裕

### 协同模式二：「盲弓引矢」—— 空警-500与JWS01的跨域交接

**战术意图**：解决蓝方"轮替闪避照射"（开15秒关45秒）对单一JWS01的克制。

```
蓝方FPS-117执行闪避：[开15s] ─── [关45s] ─── [开15s] ─── [关45s]
                    窗口A                窗口B                窗口C

单一JWS01反应：
  截获(第3s) → 锁定 → 俯冲 → 抵达(第8min) → 雷达早已关机 → 盲目盘旋
  ❌ 失败

空警-500 + JWS01 协同反应：

[窗口A] FPS-117开机15秒
  │
  ├─ JWS01-A 导引头截获(第3秒)
  ├─ 空警-500光电/红外传感器(EO/IR)同时凝视该阵地
  │   ★ 即使FPS-117关机，天线阵面的热特征/几何特征仍在 ★
  │
  ├─ 空警-500生成精确坐标：经度121.5214° 纬度24.8823° (精度<5m)
  │
  ├─ 通过保密数据链将坐标广播给整个JWS01集群
  │
  └─ FPS-117关机

[关机期 45秒内]
  │
  ├─ JWS01-A 切换至 GPS/INS + 数据链坐标修正模式
  │   (不再依赖自身导引头，直接飞向空警-500给出的坐标)
  │
  ├─ JWS01-B/C/D 同时接收坐标，从不同方位(间隔120°)切入
  │   (形成三角包围，即使坐标有5m误差，总有一架能命中天线阵面)
  │
  └─ 无需等待下一个开机窗口，在关机期间完成打击
```

**技术实现要求**：
- 空警-500需配备高分辨率合成孔径雷达（SAR）或红外搜索跟踪（IRST）设备
- 数据链需具备低延迟（<100ms）和抗干扰能力
- JWS01飞控软件需支持"外部坐标注入+INS融合"模式

### 协同模式三：「暗卫猎鹰」—— 歼-20与JWS01的护航协同

**战术意图**：蓝方必定出动F-16V携带AIM-120拦截缓慢的JWS01，歼-20负责清除这些拦截机。

```
高度层剖面图：

35,000 ft (10km)  ──── 歼-20编队 (隐身巡航，不开启雷达)
                          │
                          │ 通过空警-500数据链获取态势
                          │
                          │ 发现F-16V起飞拦截JWS01
                          │
6,500 ft  (2km)   ──── JWS01集群 (缓慢平飞/盘旋)
                          │
                          │ 完全没有空战能力，依赖歼-20保护
                          │
0 ft      (海面)  ──── 台湾海峡

交战逻辑：
1. 蓝方COC下令F-16V从清泉岗紧急起飞拦截JWS01
2. F-16V开启APG-83雷达搜索 → 立即被空警-500或歼-20ESM截获
3. 歼-20不发射雷达，利用数据链获取F-16V精确方位
4. F-16V进入距JWS01 30km范围准备发射AIM-120
5. 歼-20在F-16V侧后方40km处发射PL-15(超视距空空导弹)
6. F-16V被击落，JWS01安全通过
7. ★ 关键原则：歼-20绝不开雷达，保持无线电静默 ★
   (如果歼-20开雷达暴露自己，可能引来更多F-16V，反而危及JWS01)
```

### 协同模式四：「双模补刀」—— JWS01与HS-010光学巡飞弹的接力

**战术意图**：针对蓝方最极端的对抗手段——"全面关机+物理伪装"。

```
场景：大汉山FPS-117全面关机，天线降入掩体，阵地覆盖伪装网

JWS01集群反应：
  导引头无信号 → 盘旋搜索 → 燃油逐渐耗尽 → ❌ 无法完成任务

HS-010光学巡飞弹接力：
  HS-010特点：不带雷达导引头，携带高分辨率红外/可见光相机+AI目标识别
             续航2小时，可在目标区域上空低速游弋

协同流程：
  1. JWS01-A盘旋在大汉山上空，导引头搜索无果
  2. 空警-500指令2架HS-010从不同方向进入大汉山阵地上空
  3. HS-010开启光电载荷，对阵地进行厘米级光学扫描
  4. AI算法识别特征：
     - 虽然天线降入掩体，但掩体开启的通风散热口存在热信号异常
     - 伪装网下方存在非自然几何轮廓（矩形天线基座）
     - 阵地周边电缆沟、人员活动痕迹
  5. HS-010锁定"掩体通风口"或"天线升降机构"位置
  6. HS-010俯冲攻击，战斗部穿入掩体内部引爆
  7. ★ 即使未能彻底摧毁，破坏升降机构后，FPS-117也无法重新升起天线 ★
     → 实现"功能性摧毁"，效果等同于摧毁雷达
```

## 第六章 总结

本想定展示的核心结论是：**JWS01反辐射无人机不应被视为单一的"飞行的反辐射导弹"，而应定位为"防空网压迫系统"的一个末端节点。**

单独使用JWS01，面对成熟的"关机+闪避+诱饵"战术，其效能极其有限（本想定中单打独斗命中率<5%）。但将其嵌入以**空警-500（天眼）、歼-16D（软刀）、歼-20（硬盾）、HS-010（补刀）**构成的多维协同网络后，每一种蓝方对抗手段都会被另一种红方平台所克制，形成无法逃避的**体系绞杀**。

现代防空作战的胜负，已不再是"一部雷达对抗一枚导弹"的单挑，而是**两个传感器-射手网络的完整性对抗**。谁的协作链条更短、更快、更抗干扰，谁就能获得决定性优势。
