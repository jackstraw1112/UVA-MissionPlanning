# 雷达系统核心参数深度研究报告

## 一、 引言

雷达参数是描述雷达系统性能、定义其作战能力的核心量化指标。从远程预警的千里眼到火控照射的精准锁，不同用途的雷达，其参数设置存在着天壤之别。理解这些参数的物理意义，以及它们在不同应用场景下的差异与约束，是掌握雷达技术本质的关键。本报告将系统性地拆解雷达的核心参数体系，分析不同用户雷达的参数差异，并深入探讨参数设计过程中的工程与物理限制。

## 二、 雷达核心参数体系与深度解读

雷达的参数体系可以分为**射频参数、信号参数、天线参数以及性能参数**四个维度，每一个维度都直接决定了雷达的 “性格”。

### 2\.1 工作频率与波段选择

工作频率是雷达最根本的参数，它几乎决定了雷达的所有其他特性。根据波长的不同，雷达波段被划分为不同的类别：

|波段|频率范围|波长范围|典型特性与应用|
|---|---|---|---|
|**UHF**|300MHz\-1GHz|1m\-0\.3m|波长较长，绕射能力强，反隐身效果好，穿透 foliage 能力强。多用于远程预警和反隐身雷达。|
|**L 波段**|1\-2GHz|30\-15cm|兼顾探测距离与精度，大气衰减小。多用于远程警戒雷达、空管一次雷达。|
|**S 波段**|2\-4GHz|15\-7\.5cm|平衡了探测距离、角分辨率和天气影响。是预警机、舰载多功能雷达的首选波段。|
|**C 波段**|4\-8GHz|7\.5\-3\.75cm|分辨率优于 S 波段，受天气影响中等。多用于中程防空导弹制导雷达、气象雷达。|
|**X 波段**|8\-12GHz|37\.5\-25mm|波长短，天线可以做的很小，角分辨率极高。多用于战斗机火控雷达、反舰导弹末制导雷达。|
|**Ku/Ka 波段**|12\-40GHz|25\-7\.5mm|极高的分辨率，但大气衰减严重，雨衰明显。多用于战术导弹制导、无人机侦察雷达。|

### 2\.2 脉冲重复频率 \(PRF\) 与脉冲重复周期 \(PRI\)

脉冲重复频率（PRF）是雷达每秒发射的脉冲个数，而 \\*\\* 脉冲重复周期（PRI）\\*\\* 则是它的倒数，即两个相邻发射脉冲之间的时间间隔。这对参数共同解决了雷达 “测距” 与 “测速” 的核心矛盾。

#### 2\.2\.1 核心原理：不模糊边界

PRI 直接决定了雷达的**最大不模糊距离**，其公式为：
$R_{max} = \frac{c \times PRI}{2}$
其中 c 为光速。简单来说，雷达必须等上一个脉冲的回波回来，才能发射下一个脉冲，否则就会把 “上一个脉冲的远距回波” 误判为 “当前脉冲的近距回波”，这就是距离模糊。
同理，PRF 决定了最大不模糊速度。

#### 2\.2\.2 不同重复周期的使用场景

根据 PRI 的长短，雷达的设计有着完全不同的应用取向：

1. **长重复周期 \(Long PRI,\&gt;1ms\)**

    - **参数特征**：PRI \&gt; 1ms，对应 PRF \&lt;1kHz。最大不模糊距离极远（\&gt;150km），完全无距离模糊，但存在严重的速度模糊。

    - **使用场景**：**远程战略预警雷达**。例如 “铺路爪” 雷达，其 PRI 长达 10ms，对应的不模糊距离达到 1500km，足以覆盖其数千公里的探测范围。这类雷达只需要发现目标在哪里，不在乎瞬时速度，因此牺牲测速换测距。

2. **中重复周期 \(Medium PRI, 100us\-1ms\)**

    - **参数特征**：PRI 在 100us\-1ms 之间，对应 PRF 1\-10kHz。距离和速度都有轻微模糊，但通过多重 PRI 交错很容易解模糊。

    - **使用场景**：**机载预警雷达、舰载多功能雷达**。例如空警 \- 500 的雷达，PRI 通常在 200us\-500us。这个区间既能保证下视时的地物杂波抑制能力，又能兼顾对几百公里外目标的测距，是兼顾搜索、跟踪和抗杂波的最优折中。

3. **短重复周期 \(Short PRI, \&lt;100us\)**

    - **参数特征**：PRI \&lt;100us，对应 PRF\&gt; 10kHz。最大不模糊速度极高，完全无速度模糊，但存在严重的距离模糊。

    - **使用场景**：**机载火控雷达**。例如 F\-35 的 AN/APG\-81 雷达，PRI 可以短到 10us。这意味着它可以准确测量高达 3 马赫以上的目标速度，从而在迎头状态下，从强大的地物杂波中把高速目标滤出来。虽然这会导致距离模糊，但火控雷达的探测距离只有 200km，通过多重 PRI 很容易解模糊。

### 2\.3 发射功率与功率孔径积

发射功率是雷达的 “嗓门”，功率越大，声音传得越远。对于有源相控阵雷达，我们更关注**功率孔径积 \(PAP\)**，即发射功率乘以天线面积。这是衡量雷达探测距离的黄金指标。

- 传统的砷化镓 \(GaAs\) 器件，功率密度约为 1W/mm²。

- 新一代的氮化镓 \(GaN\) 器件，功率密度提升到了 3\-5W/mm²，这直接让雷达的探测距离提升了 40% 以上。

### 2\.4 天线增益与旁瓣电平

- **天线增益**：天线把能量集中起来的能力。增益越高，波束越窄，探测距离越远，角分辨率越高。

- **旁瓣电平**：主波束之外的多余辐射。超低旁瓣（\&lt;\-40dB）是预警雷达的关键，它能极大地抑制地物杂波，让雷达能从极强的背景中看到微弱的目标。

### 2\.5 分辨率与刷新率

- **距离分辨率**：雷达区分两个距离很近目标的能力，取决于信号带宽。带宽越大，分辨率越高。

- **角分辨率**：区分两个角度很近目标的能力，取决于天线孔径和波长。天线越大，波长越短，分辨率越高。

- **数据刷新率**：雷达更新一次航迹的时间。对于高速机动目标，刷新率必须足够高（\&lt;1s），否则目标就会 “飞丢”。

## 三、 不同应用场景的雷达参数差异

不同的用户需求，决定了雷达参数的完全不同的设计取向。

### 3\.1 远程预警用户：“看的远” 是唯一追求

对于战略预警用户来说，他们需要在几千公里外发现来袭的导弹或轰炸机。因此：

- **频率**：选择最低的 P 或 UHF 波段，利用大气衰减小、波长长得特点，实现超远距离传播。

- **天线**：做的极大，动辄几十米的天线阵面，为了提升增益和角分辨率。

- **PRF**：极低的重复频率，为了保证不出现距离模糊，能看清几千公里外的目标。

- **功率**：极高的发射功率，兆瓦级的峰值功率，为了把信号打到几千公里外。

### 3\.2 机载火控用户：“看的清、反应快”

对于战斗机飞行员来说，他们需要的是锁定几十公里外的敌机，并且引导导弹攻击。因此：

- **频率**：选择最高的 X 波段，为了在小小的机头雷达罩里，做出足够高的角分辨率。

- **天线**：受机头尺寸限制，天线直径通常只有 0\.6\-1 米，非常小。

- **PRF**：极高的重复频率，为了在迎头状态下，从强大的地物杂波中把高速目标滤出来。

- **功率**：虽然单个组件功率不大，但由于 T/R 组件多，功率密度极高。

### 3\.3 舰载多功能用户：“既要又要还要”

对于驱逐舰来说，一部雷达要同时负责远程预警、防空反导、反舰探测。因此：

- **频率**：选择中庸的 S 波段，兼顾探测距离和天气适应性，不会因为下雨就瞎了。

- **天线**：采用数字阵列，通过软件划分成多个虚拟波束，同时执行搜索、跟踪、制导任务。

- **PRF**：采用多重 PRF 交错，同时解决距离和速度模糊问题。

- **散热**：由于要在舰岛上长时间满功率运行，对散热要求极高。

### 3\.4 民用用户：“合规、低成本”

对于空管或气象雷达这类民用用户，参数受到严格的管制：

- **频率**：必须使用国际电信联盟（ITU）划分的民用频段，不能占用军用频段。

- **功率**：严格限制发射功率，避免对周边的通信设备造成干扰。

- **功能**：功能单一，只负责探测，不需要复杂的抗干扰或电子战功能，成本优先。

## 五、 雷达参数设置的约束与限制

雷达参数的设计并不是随心所欲的，它受到物理定律、工程技术以及法规的多重限制。

### 5\.1 电磁频谱管制的硬约束

这是民用雷达最首要的限制。国际电信联盟（ITU）对全球的无线电频谱进行了严格的划分。

- 例如，X 波段的火控雷达不能随便用，因为该频段部分被卫星通信占用。

- 军用雷达虽然有专用频段，但也不能随意超频，否则会干扰己方的通信或友军的雷达。

- 频率的拥挤导致了现在雷达都在追求 “频谱共享” 和 “认知频谱”，即在不干扰别人的前提下，动态占用空闲频段。

### 5\.2 物理与工程的极限约束

- **器件功率极限**：虽然 GaN 很厉害，但它也有极限。功率做的太高，器件就会被烧毁。这限制了雷达的最大发射功率。

- **平台尺寸约束**：机载雷达的天线不能比机头还大，舰载雷达的天线不能比舰岛还高。天线尺寸的物理限制，直接限制了雷达的增益和分辨率。

- **散热瓶颈**：功率越大，发热越多。在密闭的雷达罩里，如何把热量散出去，是一个巨大的难题。很多时候，不是做不出大功率器件，而是散不掉热。

### 5\.3 探测模糊的固有矛盾

这是雷达原理上的固有矛盾，无法彻底消除，只能折中。

- **距离模糊**：如果 PRF 太高，雷达发出第二个脉冲的时候，第一个脉冲的回波还没回来，雷达就会误以为这个回波是第二个脉冲的，从而算错了距离。

- **速度模糊**：如果 PRF 太低，雷达就无法准确测量目标的多普勒频移，从而算错了目标的速度。

- 工程师们只能通过**多重 PRF** 技术，用几个不同的频率交替发射，来解模糊，但这会牺牲数据率。

### 5\.4 低截获概率 \(LPI\) 的约束

现代战场，雷达开着就等于在黑夜里打手电，会暴露自己。因此，先进雷达的参数设置必须考虑低截获概率。

- **功率管理**：不能一直满功率发射，要根据目标距离自动调整功率，远处用大功率，近处用小功率。

- **波形捷变**：频率和相位要随机跳变，让敌方的电子支援设备（ESM）无法截获和识别你的雷达信号。

- 这就限制了雷达不能随便用最高功率，也不能用固定的波形，必须在探测性能和隐蔽性之间做平衡。

### 5\.5 环境适应性的约束

雷达的参数必须适应环境。

- **海杂波**：在海上，海浪的杂波很强，雷达必须调整 PRF 和多普勒滤波参数，才能把目标从海浪杂波里滤出来。

- **雨衰**：在大雨天，高频段（如 Ka 波段）的信号衰减非常严重，雷达必须自动降低频率或者增大功率来补偿。

- **地杂波**：在山区，地形杂波复杂，雷达必须调整旁瓣抑制参数，避免虚警。

## 六、 结论

雷达参数的设计，本质上是一个在约束条件下的多目标优化过程。没有最好的参数，只有最适合的参数。

- 对于预警，我们牺牲分辨率换距离；

- 对于火控，我们牺牲距离换精度；

- 对于民用，我们牺牲性能换合规。
