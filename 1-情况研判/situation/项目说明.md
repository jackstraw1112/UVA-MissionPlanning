# 反辐射无人机仿真系统 - 态势分析模块

## 项目概述

本项目是一个基于Qt的态势分析系统，用于展示辐射源列表、频谱分析、威胁评估和防控火力等信息。

## 文件结构

```
situation/
├── CMakeLists.txt          # CMake构建配置
├── SituationAnalysis.h     # 头文件
├── SituationAnalysis.cpp   # 源文件
├── SituationAnalysis.ui    # UI文件（参考用）
├── SituationAnalysis.pro   # qmake项目文件
├── main.cpp                # 程序入口
├── resources.qrc           # 资源文件
├── 打开项目.bat           # 快速打开项目脚本
└── build/                  # 构建目录（自动生成）
```

## 快速开始

### 方式一：使用批处理脚本（推荐）

1. 双击运行 `打开项目.bat`
2. 脚本会自动配置CMake并打开Visual Studio

### 方式二：手动配置

1. 创建构建目录：
   ```bash
   mkdir build
   cd build
   ```

2. 生成Visual Studio项目：
   ```bash
   cmake .. -G "Visual Studio 17 2022" -A x64
   ```

3. 打开解决方案：
   ```bash
   start SituationAnalysis.sln
   ```

## Visual Studio操作指南

### 编译项目

1. 在Visual Studio中打开解决方案
2. 选择配置：`Debug` 或 `Release`
3. 选择平台：`x64`
4. 点击菜单：**生成** → **生成解决方案**（或按 `Ctrl+Shift+B`）

### 运行程序

1. 确保 `SituationAnalysis` 设为启动项目
2. 按 `F5` 键开始调试
3. 或按 `Ctrl+F5` 直接运行

## 项目配置

### Qt路径配置

如果Qt安装在非默认路径，请修改 `CMakeLists.txt` 中的：

```cmake
set(CMAKE_PREFIX_PATH "您的Qt路径")
```

或者设置环境变量 `Qt5_DIR`。

### 依赖项

- Qt 5.12.12 或更高版本
- Visual Studio 2019 或更高版本
- CMake 3.16 或更高版本

## 功能模块

| 模块 | 位置 | 说明 |
|------|------|------|
| 辐射源列表 | 左上 | 显示雷达、电台、通信对抗、雷达对抗等目标 |
| 态势控制 | 左下 | 控制各模块的显示开关 |
| 频谱分析 | 右上 | 展示频率分布和信号强度 |
| 防控火力 | 右中 | 显示防空系统信息 |
| 威胁评估 | 右下 | 按优先级显示威胁目标 |

## 常见问题

### Q: CMake找不到Qt怎么办？

A: 检查以下几点：
1. 确认Qt已正确安装
2. 设置 `Qt5_DIR` 环境变量
3. 或修改 `CMakeLists.txt` 中的路径

### Q: 编译错误怎么办？

A: 
1. 清理 `build` 目录
2. 重新运行CMake配置
3. 确保使用x64平台

### Q: 运行时缺少DLL？

A: 
1. 将Qt的bin目录添加到PATH
2. 或使用 `windeployqt` 工具部署依赖
