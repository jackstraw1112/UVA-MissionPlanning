# 图标资源

## 图标文件说明

本目录包含应用程序所需的图标资源文件。由于实际图标文件需要图形设计，这里提供图标资源的说明和占位符。

## 所需图标列表

### 1. 文件夹图标
- **folder.png**: 文件夹图标，用于树形结构中的编组节点
- **subfolder.png**: 子文件夹图标，用于树形结构中的装备类型节点

### 2. 文档图标
- **document.png**: 文档图标，用于任务列表项

### 3. 设置图标
- **settings.png**: 设置图标，用于设置按钮

### 4. 工具栏图标
- **new.png**: 新建图标
- **open.png**: 打开图标
- **save.png**: 保存图标
- **edit.png**: 编辑图标
- **delete.png**: 删除图标
- **calculate.png**: 计算图标
- **allocate.png**: 分配图标
- **path.png**: 路径图标
- **target.png**: 目标图标

## 图标规格

### 尺寸
- **工具栏图标**: 32x32 像素
- **树形图标**: 16x16 像素
- **按钮图标**: 24x24 像素

### 格式
- **推荐格式**: PNG (支持透明度)
- **备用格式**: SVG (矢量图形)

### 颜色
- **主色调**: 青色 (#00cfff)
- **悬停色**: 亮青色 (#00e5ff)
- **禁用色**: 灰色 (#808080)

## 图标设计建议

### 1. 风格
- 现代化扁平设计
- 简洁明了
- 一致的外观

### 2. 视觉效果
- 适当的阴影和发光效果
- 清晰的轮廓
- 良好的可识别性

### 3. 交互反馈
- 悬停状态变化
- 点击状态变化
- 禁用状态变化

## 占位符使用

在图标文件未提供的情况下，应用程序使用以下占位符：

1. **文本图标**: 使用Unicode字符作为临时图标
2. **颜色标识**: 使用颜色块表示不同类型
3. **简单图形**: 使用基本几何图形

## 图标替换步骤

1. 设计符合规格的图标文件
2. 将图标文件放入本目录
3. 更新 `resources.qrc` 文件中的图标引用
4. 重新构建应用程序

## 注意事项

1. 保持图标风格一致
2. 确保图标在不同尺寸下清晰
3. 考虑不同主题下的图标显示
4. 提供高对比度版本以支持可访问性
